Introduction to Computer Security

What is Computer Security?

Computer security (cybersecurity) is the protection of computer systems and networks from theft, damage, or disruption of services.

Core principles (CIA Triad):

  • Confidentiality – only authorized access
  • Integrity – data not modified by unauthorized parties
  • Availability – systems accessible when needed
Threat Landscape

Threats include: malware, hacking, DoS attacks, cyberstalking, fraud, and insider threats.

Networks, Internet & DoS Attacks

Networks and Internet Security

Network security involves protecting the underlying infrastructure from unauthorized access, misuse, or attacks. Common threats: packet sniffing, IP spoofing, DoS.

Denial of Service (DoS) Attacks

DoS attacks overwhelm a system with traffic, making it unavailable to legitimate users. DDoS uses multiple compromised systems.

  • SYN flood – exploits TCP handshake
  • Ping of Death – oversized packets
  • HTTP flood – massive GET/POST requests
  • Amplification attacks (NTP, DNS)

Scanning

Attackers scan networks to find open ports and vulnerabilities. Tools: Nmap, Nessus.

DoS Tools

Common DoS tools: LOIC (Low Orbit Ion Cannon), HOIC, Slowloris, Hping3. These are often used in hacktivism.

Tracert · Visual Route · Wireshark

Tracert (Traceroute)

Tracert (Windows) / traceroute (Linux) maps the path packets take from source to destination, showing each hop (router).

tracert example.com

Used for network diagnostics and identifying routing paths.

Visual Route

Visual Route is a graphical traceroute tool that displays the path on a world map, showing geographic locations of routers.

Helps visualize network latency and routing issues.

Wireshark

Wireshark is a network protocol analyzer that captures and inspects packets in real-time. Used for:

  • Troubleshooting networks
  • Detecting malicious traffic
  • Analyzing DoS attacks
  • Educational purposes (see TCP handshake, etc.)
# Capture traffic on interface eth0
wireshark -i eth0

Cyber Stalking, Fraud & Abuse

Cyber Stalking

Cyber stalking is the use of the internet to harass, threaten, or intimidate someone. It includes:

  • Monitoring online activities
  • Sending threatening messages
  • Identity theft / impersonation
  • Posting false information

Cyber Fraud

Online scams to steal money/data: phishing, advance-fee scams, online auction fraud, romance scams.

Online Abuse

Cyberbullying, trolling, hate speech, revenge porn, doxxing (publishing private info).

Legal Aspects

Many countries have laws against cyber stalking (e.g., IT Act in India). Victims should preserve evidence and report to cyber crime cells.

Malware (Malicious Software)

What is Malware?

Malware is any software intentionally designed to cause damage to a computer, server, or network.

TypeDescriptionExample
VirusAttaches to clean files and spreadsCIH, Melissa
WormSelf-replicates without user actionMorris, Blaster
TrojanDisguised as legitimate softwareZeus, Emotet
RansomwareEncrypts data and demands paymentWannaCry, Petya
SpywareSecretly monitors user activityKeyloggers, FinFisher
AdwareForces unwanted adsFireball, Appearch
RootkitHides malicious activity from systemSony rootkit
Advanced Malware

Fileless malware – runs in memory without leaving traces. Polymorphic malware – changes code to avoid detection.

Techniques Used by Hackers

Hacker Methodology

Hackers follow steps: Reconnaissance → Scanning → Gaining Access → Maintaining Access → Covering Tracks.

Phishing

Fake emails/websites to steal credentials.

Social Engineering

Manipulating people to reveal info.

Password Cracking

Brute force, dictionary, rainbow tables.

Man-in-the-Middle

Intercepting communication.

Exploiting Vulnerabilities

Buffer overflows, SQL injection.

Botnets

Networks of compromised machines for DDoS.
SQL Injection Example
-- Unsafe query:
"SELECT * FROM users WHERE username = '" + userInput + "';"
-- Attacker inputs: ' OR '1'='1
-- Query becomes: SELECT * FROM users WHERE username = '' OR '1'='1';
Ethical Hacking

Ethical hackers use same techniques but with permission to find vulnerabilities and fix them.
