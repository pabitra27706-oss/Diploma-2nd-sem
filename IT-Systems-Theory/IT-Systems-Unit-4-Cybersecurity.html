Introduction to Computer Security

What is Computer Security?

Computer Security (also called Cyber Security or IT Security) is the practice of protecting computer systems, networks, programs, and data from unauthorized access, attacks, damage, or theft.

Real-world analogy: Just as you lock your house doors and windows to protect your belongings from thieves, computer security involves setting up "digital locks" (passwords, firewalls, antivirus) to protect your digital belongings (files, accounts, money). 🔒

❓ Why Do We Need Computer Security?

Need for Computer Security
  1. Protect personal data: Photos, documents, bank details, passwords
  2. Prevent financial loss: Online banking fraud, credit card theft
  3. Protect business information: Company secrets, customer data
  4. Maintain system availability: Prevent attacks that crash systems
  5. Legal compliance: Laws require organizations to protect user data (IT Act 2000 in India)
  6. Prevent identity theft: Criminals can impersonate you using stolen information
  7. National security: Government systems, defense networks must be protected
  8. Growing internet usage: More people online = more targets for criminals

🔥 The CIA Triad — Three Goals of Security (Very Important!)

The CIA Triad is the most fundamental concept in computer security. Every security measure aims to protect one or more of these three goals:

🔒 C — Confidentiality

Definition: Only authorized (allowed) people should be able to access information.

Example: Your bank password should only be known to you — not to anyone else.

Threats: Data theft, hacking, eavesdropping

Protection: Encryption, passwords, access control

✏️ I — Integrity

Definition: Data should remain accurate, complete, and unchanged by unauthorized people.

Example: Your exam marks in the system should not be modified by anyone except authorized faculty.

Threats: Data tampering, man-in-the-middle attack

Protection: Checksums, digital signatures, hashing

🟢 A — Availability

Definition: Data and systems should be available to authorized users whenever needed.

Example: An online exam website should be accessible when students need to take the exam — it should not crash.

Threats: DoS attacks, hardware failure, natural disasters

Protection: Backups, redundancy, firewalls

🛡️ Basic Security Measures

How to Protect Computer Systems
  1. Passwords: Use strong, unique passwords (mix of letters, numbers, symbols)
  2. Antivirus Software: Install and regularly update antivirus programs
  3. Firewall: A barrier between your computer and the internet that blocks unauthorized access
  4. Encryption: Converting data into coded form so only authorized users can read it
  5. Regular Backups: Keep copies of important data in a safe place
  6. Software Updates: Keep operating system and software updated (patches fix security holes)
  7. Two-Factor Authentication (2FA): Using two methods to verify identity (password + OTP)
  8. User Education: Training people to recognize and avoid threats
Key Terms
  • Authentication: Verifying the identity of a user (Are you who you claim to be?)
  • Authorization: Determining what an authenticated user is allowed to do
  • Encryption: Converting readable data (plaintext) into unreadable coded form (ciphertext)
  • Decryption: Converting ciphertext back into readable plaintext
  • Firewall: Hardware or software that filters network traffic to block unauthorized access

🔧 Network Utilities (Tools)

Network utilities are software tools used to analyze, test, troubleshoot, and monitor computer networks. Security professionals use these tools to find vulnerabilities, and attackers may misuse them.

📡 Tracert (Traceroute) 🔥

What is Tracert?

Tracert (short for Trace Route) is a command-line network tool used to trace the path (route) that data packets take from your computer to a destination server (like google.com). It shows every router (hop) the data passes through along the way.

Analogy: Imagine you're sending a letter from Kolkata to Delhi by train. Tracert would show you every station the train stops at: Kolkata → Dhanbad → Gaya → Allahabad → Agra → Delhi. 🚂

How to Use Tracert

C:\> tracert www.google.com

Tracing route to www.google.com [142.250.182.164]
over a maximum of 30 hops:

  1     1 ms     1 ms     1 ms   192.168.1.1        (Your Router)
  2     5 ms     4 ms     5 ms   10.0.0.1           (ISP Gateway)
  3    12 ms    11 ms    12 ms   172.16.45.1         (ISP Router)
  4    25 ms    24 ms    25 ms   72.14.209.82        (Google Router)
  5    28 ms    27 ms    28 ms   142.250.182.164     (Google Server)

Trace complete.

Each line = one "hop" (one router along the path)
The ms values = time taken (in milliseconds) for round-trip to that hop
Tracert Output — What Each Column Means
ColumnMeaning
Hop number (1, 2, 3...)Sequential number of each router in the path
Round-trip times (ms)Three timing tests — time for packet to go and come back
IP address / hostnameAddress of that router or server
* * *Request timed out — that router didn't respond
Uses of Tracert
  1. Troubleshooting: Find where in the network a problem (slow speed or failure) is occurring
  2. Network mapping: See the route packets take to reach a destination
  3. Identifying bottlenecks: Find which hop is causing high delay (latency)
  4. Security analysis: Understand network topology for security testing
  5. Verifying routing: Check if traffic is going through expected paths
Command Variations
  • Windows: tracert www.google.com
  • Linux/Mac: traceroute www.google.com
  • Ping (related tool): ping www.google.com — checks if a host is reachable and measures response time

Visual Route is a graphical (GUI-based) traceroute tool that shows the path of data packets on a world map. Unlike the text-based Tracert command, Visual Route provides a visual, easy-to-understand representation of the network path.

🦈 WireShark — Network Packet Analyzer 🔥

What is WireShark?

WireShark is a free, open-source network protocol analyzer (also called a packet sniffer). It captures and analyzes data packets traveling through a network in real-time. It lets you see exactly what data is being sent and received on your network.

Analogy: Imagine you could open every letter (packet) passing through a post office (network) and read its contents — that's what WireShark does for computer networks. 📬

Key Features of WireShark
  • Captures live network traffic in real-time
  • Supports hundreds of network protocols (HTTP, TCP, UDP, DNS, etc.)
  • Shows packet details — source IP, destination IP, protocol, data content
  • Can filter traffic by protocol, IP address, port number
  • Can save captured data for later analysis
  • Free and open-source
  • Available for Windows, macOS, and Linux
Uses of WireShark
  1. Network troubleshooting: Find why network is slow or not working
  2. Security analysis: Detect suspicious traffic or unauthorized connections
  3. Protocol analysis: Study how network protocols work
  4. Detecting attacks: Identify DoS attacks, port scanning, etc.
  5. Network forensics: Investigate security incidents after they happen
  6. Learning tool: Students use it to understand networking concepts
Legal Warning

Using WireShark to capture packets on a network without permission is illegal. Only use it on networks you own or have explicit permission to analyze. Unauthorized packet sniffing can lead to criminal charges under the IT Act 2000.

Cyber Stalking, Fraud & Abuse

What is Cyber Crime?

Cyber crime (or computer crime) refers to any criminal activity that involves a computer, network, or the internet. It includes using technology to commit fraud, steal data, harass people, spread malware, or disrupt services.

📋 Types of Cyber Crimes

CategoryExamples
Against IndividualsCyber stalking, identity theft, phishing, harassment, defamation
Against PropertyHacking, virus attacks, software piracy, intellectual property theft
Against OrganizationsData breach, DoS attacks, ransomware, corporate espionage
Against Society/GovernmentCyber terrorism, spreading fake news, attacking government websites

👁️ Cyber Stalking 🔥

What is Cyber Stalking?

Cyber stalking is the use of the internet, email, social media, or other digital technology to harass, threaten, intimidate, or monitor a person without their consent. It is the online version of physical stalking.

Examples of Cyber Stalking
  • Sending repeated unwanted emails or messages
  • Constantly monitoring someone's social media profile
  • Creating fake profiles to follow or contact the victim
  • Posting threatening or embarrassing content about someone online
  • Using GPS or spyware to track someone's location
  • Spreading personal photos or information without consent
  • Hacking into someone's email or social media accounts
Legal Consequences

Under India's IT Act 2000 (Section 66A) and Indian Penal Code (IPC Section 354D), cyber stalking is a punishable offence with imprisonment up to 3 years and fine.

💰 Cyber Fraud 🔥

What is Cyber Fraud?

Cyber fraud refers to any dishonest or illegal activity conducted through the internet with the intention of stealing money, data, or identity.

Common Types of Cyber Fraud

🎣

Phishing

Sending fake emails/websites that look like real ones (bank, Gmail) to steal passwords and credit card details.

🆔

Identity Theft

Stealing someone's personal information (Aadhaar, PAN, bank details) to impersonate them and commit fraud.

💳

Credit Card Fraud

Using stolen credit/debit card information to make unauthorized purchases online.

🏪

Online Shopping Fraud

Fake e-commerce websites that take payment but never deliver the product.

🎰

Lottery/Prize Scam

"You have won Rs. 10 Lakhs! Send Rs. 5000 to claim your prize." — Always fake!

💼

Job Fraud

Fake job offers asking for registration fees or personal documents.

⚠️ Cyber Abuse

What is Cyber Abuse?

Cyber abuse (also called cyber bullying) is using digital technology to harass, humiliate, threaten, or abuse another person. It can happen through social media, messaging apps, email, or gaming platforms.

Forms of Cyber Abuse
  1. Cyber bullying: Repeatedly sending mean, threatening messages
  2. Trolling: Posting provocative comments to upset people
  3. Doxing: Publishing someone's private information (address, phone) online without consent
  4. Revenge posting: Sharing private photos/videos to humiliate someone
  5. Hate speech: Spreading hatred against a person, group, or community online
  6. Impersonation: Creating fake profiles in someone else's name
The IT Act 2000 (India)

India's Information Technology Act, 2000 (amended in 2008) is the primary law dealing with cyber crimes. Key sections include:

  • Section 43: Penalty for damage to computer systems
  • Section 66: Computer-related offences (hacking)
  • Section 66C: Identity theft
  • Section 66D: Cheating by personation (impersonation using computers)
  • Section 66E: Violation of privacy
  • Section 67: Publishing obscene material online

Denial of Service (DoS) Attacks

What is a DoS Attack? 🔥

A Denial of Service (DoS) attack is a type of cyber attack where the attacker floods a target server, website, or network with so much fake traffic that it becomes overwhelmed and unable to serve legitimate (real) users.

Analogy: Imagine a small shop (server) can serve 10 customers at a time. If 10,000 fake customers (attack traffic) crowd the shop, the real customers cannot enter or be served. That's a DoS attack! 🏪🚫

🎯 Goal of DoS Attack

The primary goal is to disrupt availability — make the service unavailable to legitimate users. This attacks the "A" (Availability) in the CIA Triad.

📋 Types of DoS Attacks 🔥

TypeHow It WorksAnalogy
Flood AttackSends massive amount of traffic to overwhelm the serverOpening all taps to flood a house
Ping of DeathSends oversized ping packets that crash the target systemSending a package too big for the mailbox
SYN FloodSends thousands of connection requests but never completes the handshake, exhausting server resourcesCalling a phone and hanging up repeatedly so no one else can call

HTTP FloodSends massive number of HTTP requests (webpage requests) to a web serverMillions of people refreshing a website simultaneously
Teardrop AttackSends fragmented packets that cannot be reassembled, crashing the targetSending puzzle pieces that don't fit together

💣 DDoS — Distributed Denial of Service 🔥

What is DDoS?

A DDoS (Distributed Denial of Service) attack is a more powerful version of DoS where the attack comes from multiple computers (distributed sources) simultaneously. The attacker uses a network of compromised computers (called a botnet) to launch the attack.

What is a Botnet?

A botnet is a network of compromised (infected) computers that are controlled remotely by a hacker without the owners' knowledge. Each infected computer is called a "bot" or "zombie". The hacker uses this army of bots to launch DDoS attacks.

🔧 Scanning with WireShark (in context of DoS)

How WireShark Helps Detect DoS Attacks
  • Capture packets: WireShark can capture all incoming network traffic
  • Analyze volume: Abnormally high volume of packets from one or many sources indicates DoS
  • Identify attack type: By examining packet types (SYN packets for SYN flood, ICMP for ping flood)
  • Filter suspicious traffic: Use filters like tcp.flags.syn == 1 to find SYN flood
  • Find source IPs: Identify which IP addresses are sending the attack traffic
  • Evidence collection: Save captured data as evidence for investigation

🛡️ Prevention of DoS/DDoS Attacks

How to Prevent DoS Attacks
  1. Firewalls: Configure firewalls to filter and block suspicious traffic
  2. Rate limiting: Limit the number of requests from a single IP address
  3. Load balancing: Distribute traffic across multiple servers
  4. Content Delivery Network (CDN): Use services like Cloudflare to absorb attack traffic
  5. Intrusion Detection System (IDS): Monitor traffic patterns for anomalies
  6. Over-provisioning: Have more bandwidth/capacity than normally needed
  7. IP blacklisting: Block known malicious IP addresses
  8. Keep systems updated: Patch vulnerabilities that attackers exploit
Real-World DoS Attack Examples
  • 2016: DDoS attack on Dyn DNS — took down Twitter, Netflix, Reddit, and many other major websites
  • 2020: AWS (Amazon Web Services) faced the largest DDoS attack in history — 2.3 Tbps
  • India: Government websites are frequently targeted by DoS attacks

Malware — Types & Working

What is Malware? 🔥

Malware (short for Malicious Software) is any software intentionally designed to cause damage to a computer, server, network, or user. It includes viruses, worms, trojans, ransomware, and many other harmful programs.

Analogy: Just as biological viruses make your body sick 🤒, computer malware makes your computer "sick" — slowing it down, stealing data, or damaging files.

🔥 Types of Malware (Very Important — Must Know All)

🦠

1. Virus

A malicious program that attaches itself to a legitimate file or program and spreads when the infected file is opened or shared. It needs human action to spread (e.g., opening an email attachment).

🐛

2. Worm

A self-replicating malware that spreads automatically through networks without any human action. It copies itself from computer to computer using network connections.

🐴

3. Trojan Horse

Malware disguised as legitimate software. Looks useful (game, tool) but performs harmful actions once installed. Named after the Greek Trojan Horse story.

🔐

4. Ransomware

Encrypts (locks) your files and demands a ransom (money) to unlock them. If you don't pay, your files may be permanently lost. Very dangerous!

👀

5. Spyware

Secretly monitors your computer activity — keystrokes, browsing history, passwords. Sends this data to the attacker without your knowledge.

📢

6. Adware

Automatically displays unwanted advertisements (pop-ups, banners) on your screen. May slow down your computer. Sometimes tracks your browsing habits.

🌿

7. Rootkit

Hides deep inside the operating system to give the attacker administrator-level (root) access. Very hard to detect and remove.

⌨️

8. Keylogger

Records every keystroke (key press) you make — capturing passwords, messages, credit card numbers. Can be software or hardware.

🔄 How Malware Spreads

Common Ways Malware Enters Your System
  1. Email attachments: Opening infected files attached to emails
  2. Malicious websites: Visiting fake or compromised websites
  3. Pirated software: Downloading cracked/pirated apps or games
  4. USB drives: Inserting infected USB pen drives
  5. Fake downloads: "Click here to download free movie" type links
  6. Software vulnerabilities: Unpatched (not updated) operating systems or software
  7. Social engineering: Tricking users into installing malware
  8. File sharing networks: P2P (peer-to-peer) networks like torrents

🛡️ Protection Against Malware

  • Install and update antivirus software (e.g., Windows Defender, Kaspersky, Norton)
  • Keep your operating system and software updated
  • Never open email attachments from unknown senders
  • Don't download software from untrusted websites
  • Avoid using pirated software
  • Scan USB drives before opening files
  • Enable firewall on your computer
  • Take regular backups of important files
  • Be careful with pop-up advertisements — don't click randomly
  • Use strong, unique passwords for all accounts

Techniques Used by Hackers

What is Hacking?

Hacking refers to gaining unauthorized access to a computer system, network, or data. A hacker is a person who uses technical skills to break into computer systems.

🎭 Types of Hackers 🔥

🤍

White Hat Hacker

Ethical hacker. Uses hacking skills for good — finds security holes in systems and reports them to the owner. Works legally. Also called Security Researcher.

Example: A company hires a white hat to test their website security.

🖤

Black Hat Hacker

Criminal hacker. Breaks into systems illegally with malicious intent — to steal data, money, or cause damage. Also called Cracker.

Example: Someone who hacks bank accounts to steal money.

🩶

Grey Hat Hacker

Mix of both. Breaks into systems without permission but usually doesn't cause harm. May report vulnerabilities after hacking — but does so without authorization.

Example: Finding a website bug and publicly disclosing it without asking.

🔧 Common Hacking Techniques 🔥

1. Phishing 🎣

Phishing

Phishing is a technique where attackers send fake emails or create fake websites that look exactly like real ones (bank, Gmail, Facebook) to trick users into entering their passwords, credit card numbers, or personal information.

Example: You receive an email: "Your SBI account has been locked! Click here to verify." The link takes you to a fake SBI website that looks real, but when you enter your password, the hacker captures it.

Types of Phishing:

  • Email Phishing: Fake emails (most common)
  • Spear Phishing: Targeted at a specific person or organization
  • Smishing: Phishing via SMS messages
  • Vishing: Phishing via voice calls (phone calls)

2. Social Engineering 🎭

Social Engineering

Social engineering is the art of manipulating people psychologically to trick them into giving away confidential information or performing actions that compromise security. It exploits human trust and emotions rather than technical vulnerabilities.

Example: A caller pretending to be from "IT Support": "This is the IT department. We need your password to fix a problem with your account." — They are not really from IT!

Techniques:

  • Pretexting: Creating a fake scenario to gain trust
  • Baiting: Leaving an infected USB drive in a public place hoping someone picks it up
  • Tailgating: Following an authorized person through a secure door
  • Quid pro quo: Offering something (free software) in exchange for information

3. Brute Force Attack 🔨

Brute Force

A brute force attack involves trying every possible combination of passwords until the correct one is found. It uses automated software to try thousands or millions of password combinations.

Analogy: If you forgot the combination of a 3-digit lock, you'd try 000, 001, 002... up to 999. That's brute force! 🔓

Prevention: Use long, complex passwords. Enable account lockout after multiple failed attempts. Use CAPTCHA.

4. Dictionary Attack 📖

Dictionary Attack

Similar to brute force, but instead of trying all combinations, it tries common words and phrases from a dictionary/wordlist. Faster than brute force because most people use simple passwords like "password123", "admin", or their name.

Prevention: Don't use common words as passwords. Use a mix of letters, numbers, symbols.

5. Keylogging ⌨️

Keylogger

A keylogger is a program (or hardware device) that records every keystroke (key press) on a computer. It captures usernames, passwords, credit card numbers, messages — everything typed on the keyboard.

Types:

  • Software keylogger: A program installed on the computer (usually hidden)
  • Hardware keylogger: A small physical device plugged between keyboard and computer

Prevention: Use antivirus. Check for unknown devices on keyboard cable. Use on-screen keyboard for sensitive data.

6. Password Cracking 🔑

Password Cracking

Password cracking is the process of recovering or guessing passwords from stored or transmitted data. Hackers use various methods:

  • Brute Force: Try all combinations
  • Dictionary Attack: Try common words
  • Rainbow Table: Pre-computed tables of password hashes
  • Shoulder Surfing: Looking over someone's shoulder while they type password
  • Social Engineering: Tricking the person into revealing the password

7. Man-in-the-Middle (MITM) Attack 🕵️

MITM Attack

The attacker secretly positions themselves between two communicating parties (e.g., you and a website). They can intercept, read, and modify the data being exchanged without either party knowing.

Analogy: You're passing a note to your friend, but someone in between reads it, changes it, and passes it along. Neither you nor your friend knows the note was tampered with. ✉️

Prevention: Use HTTPS websites. Avoid public Wi-Fi for sensitive activities. Use VPN.

8. SQL Injection 💉

SQL Injection

SQL injection is an attack where the hacker inserts malicious SQL (database) commands into input fields (like login forms) to access, modify, or delete data from the website's database.

Example: In a login form, instead of entering a username, the hacker types: ' OR '1'='1 — which tricks the database into giving access without a correct password.

Prevention: Input validation, parameterized queries, web application firewalls.

Prevention & Cyber Safety Best Practices

Complete Cyber Safety Checklist
  • Use strong passwords — at least 8 characters with uppercase, lowercase, numbers, and symbols
  • Enable Two-Factor Authentication (2FA) on all important accounts
  • Install antivirus software and keep it updated
  • Keep operating system and software updated (security patches)

  • Don't download pirated software or cracked apps
  • Take regular backups of important data

  • Log out of accounts when using shared/public computers

  • Mix: UPPERCASE + lowercase + numbers + symbols (@, #, $, !)

  • Consider using a password manager (LastPass, Bitwarden)

Important Definitions

Computer Security

The practice of protecting computer systems, networks, programs, and data from unauthorized access, attacks, damage, or theft.

CIA Triad

The three fundamental goals of information security: Confidentiality (only authorized access), Integrity (data accuracy and completeness), and Availability (accessible when needed).

Cyber Crime

Any criminal activity that involves a computer, network, or the internet, including hacking, fraud, stalking, and data theft.

Cyber Stalking

The use of the internet, email, or social media to repeatedly harass, threaten, intimidate, or monitor a person without their consent.

Phishing

A social engineering technique where attackers send fake emails or create fake websites to trick users into revealing passwords, credit card numbers, or personal information.

Denial of Service (DoS) Attack

A cyber attack where the attacker floods a target server with excessive traffic, making it unable to respond to legitimate users' requests.

DDoS Attack

Distributed Denial of Service — a more powerful DoS attack launched from multiple compromised computers (botnet) simultaneously.

Botnet

A network of compromised (infected) computers controlled remotely by a hacker, used to launch DDoS attacks or send spam.

Malware

Short for Malicious Software — any software intentionally designed to cause damage to computers, networks, or users. Includes viruses, worms, trojans, ransomware, spyware.

Computer Virus

A malicious program that attaches itself to a legitimate file and spreads when the infected file is opened or shared. Requires human action to spread.

Worm

A self-replicating malware that spreads automatically through networks without human action. Does not need a host file.

Trojan Horse

Malware disguised as legitimate or useful software. Once installed, it performs harmful actions like creating backdoors for hackers.

Ransomware

Malware that encrypts (locks) the victim's files and demands a ransom payment to restore access. Very dangerous and increasingly common.

Social Engineering

The art of manipulating people psychologically to trick them into giving away confidential information or performing actions that compromise security.

Firewall

A hardware or software security system that monitors and controls incoming and outgoing network traffic based on security rules, acting as a barrier between a trusted internal network and untrusted external network.

Encryption

The process of converting readable data (plaintext) into unreadable coded form (ciphertext) using an algorithm and a key, so that only authorized users with the decryption key can read it.

Tracert (Traceroute)

A command-line network tool used to trace the path (route) that data packets take from the user's computer to a destination server, showing every router (hop) along the way.

WireShark

A free, open-source network protocol analyzer (packet sniffer) that captures and analyzes data packets traveling through a network in real-time.

Question & Answer Bank

📝 Short Answer Questions (2 Marks)

Q1. Define Computer Security.

Computer Security is the practice of protecting computer systems, networks, programs, and data from unauthorized access, attacks, damage, or theft. It ensures confidentiality, integrity, and availability of information.

Q2. What is the CIA Triad? Name its three components.

The CIA Triad is the fundamental model of information security with three goals:

  1. Confidentiality: Only authorized users can access information.
  2. Integrity: Data remains accurate, complete, and unaltered.
  3. Availability: Data and systems are accessible when needed by authorized users.

Q3. What is Phishing? Give one example.

Phishing is a social engineering attack where fake emails or websites are created to trick users into revealing passwords, credit card details, or personal information.

Example: A fake email from "SBI Bank" asking you to click a link and enter your password to "verify your account." The link leads to a fake website that captures your credentials.

Q4. What is the difference between a virus and a worm?

  • Virus: Needs a host file to attach to and requires human action (opening the file) to spread.
  • Worm: Is standalone (no host needed) and spreads automatically through networks without any human action.

Q5. What is a Firewall?

A Firewall is a hardware or software security system that monitors and controls incoming and outgoing network traffic based on predefined security rules. It acts as a barrier between a trusted internal network and an untrusted external network (like the internet) to block unauthorized access.

Q6. What is Cyber Stalking?

Cyber Stalking is the use of the internet, email, social media, or other digital technology to repeatedly harass, threaten, intimidate, or monitor a person without their consent. It is a criminal offence under India's IT Act 2000 and IPC Section 354D.

Q7. What is Tracert? What is its purpose?

Tracert (Trace Route) is a command-line network tool used to trace the path that data packets take from the user's computer to a destination server. It shows every router (hop) along the way with response times.

Purpose: Network troubleshooting, finding bottlenecks, mapping network routes.

Usage: tracert www.google.com

Q8. What is a Botnet?

A Botnet is a network of compromised (infected) computers controlled remotely by a hacker. Each infected computer is called a "bot" or "zombie." Hackers use botnets to launch DDoS attacks, send spam emails, or spread malware.

Q9. What is Social Engineering in the context of hacking?

Social Engineering is the art of manipulating people psychologically to trick them into revealing confidential information or performing actions that compromise security. It exploits human trust, fear, or curiosity rather than technical vulnerabilities.

Examples: Pretexting, baiting, tailgating, phishing calls.

Q10. Differentiate between Authentication and Authorization.

  • Authentication: Verifying "Who are you?" — confirming the identity of a user using passwords, OTP, biometrics. Happens first.
  • Authorization: Determining "What can you do?" — deciding what resources/actions the authenticated user is allowed to access. Happens after authentication.

📄 Long Answer Questions (5–7 Marks)

Q1. Explain the CIA Triad in detail. Why is it important in computer security? Give examples for each component. [7 marks] 🔥

The CIA Triad is the most fundamental model of information security. It defines three core goals that every security system aims to achieve:

1. Confidentiality:

  • Only authorized (allowed) people should access information.
  • Example: Your bank password should only be known to you.
  • Protection: Encryption, passwords, access control, 2FA.
  • Threats: Data theft, hacking, eavesdropping.

2. Integrity:

  • Data should remain accurate, complete, and unchanged by unauthorized users.
  • Example: Your exam marks should not be modified by anyone except authorized faculty.
  • Protection: Checksums, digital signatures, hashing.
  • Threats: Data tampering, man-in-the-middle attack.

3. Availability:

  • Data and systems should be accessible when needed by authorized users.
  • Example: Online exam portal should be available during exam time.
  • Protection: Backups, redundancy, load balancing, firewalls.
  • Threats: DoS/DDoS attacks, hardware failure, natural disasters.

Importance: Every security decision, tool, and policy is designed to protect one or more of these three goals. A breach in any one of them compromises the overall security of the system.

Q2. What is a Denial of Service (DoS) attack? Explain its types. How is DDoS different from DoS? How can DoS attacks be prevented? [7 marks] 🔥

DoS Attack: A Denial of Service attack is a cyber attack where the attacker floods a target server with excessive fake traffic, making it unable to respond to legitimate users' requests.

Types:

  1. Flood Attack: Sends massive traffic to overwhelm the server.
  2. SYN Flood: Sends connection requests without completing the handshake, exhausting server resources.
  3. Ping of Death: Sends oversized ping packets that crash the system.
  4. Smurf Attack: Sends ping requests with victim's IP, making others flood the victim with replies.
  5. HTTP Flood: Sends millions of HTTP requests to a web server.

DoS vs DDoS:

  • DoS comes from a single computer; DDoS comes from multiple computers (botnet).
  • DDoS is more powerful and harder to block.
  • DDoS is very difficult to trace back to the real attacker.

Prevention: Firewalls, rate limiting, load balancing, CDN (like Cloudflare), Intrusion Detection Systems, IP blacklisting, keeping systems updated.

Q3. What is Malware? Explain any five types of malware with examples. [7 marks] 🔥

Malware (Malicious Software) is any software intentionally designed to cause damage to computers, servers, networks, or users.

Five Types:

  1. Virus: Attaches to legitimate files. Needs human action to spread. Corrupts/deletes files. Example: ILOVEYOU virus.
  2. Worm: Self-replicating. Spreads automatically through networks without human action. Consumes bandwidth. Example: Code Red.
  3. Trojan Horse: Disguised as useful software. Once installed, gives attacker backdoor access. Example: Zeus Trojan.
  4. Ransomware: Encrypts files and demands ransom money to unlock them. Very dangerous. Example: WannaCry.
  5. Spyware: Secretly monitors user activity — keystrokes, browsing history, passwords. Sends data to attacker. Example: CoolWebSearch.

Prevention: Install antivirus, keep software updated, don't open suspicious emails, avoid pirated software, use firewall, take regular backups.

Q4. Explain the different techniques used by hackers. What are the types of hackers? [7 marks] 🔥

Types of Hackers:

  1. White Hat (Ethical): Authorized, legal, finds and fixes vulnerabilities.
  2. Black Hat (Criminal): Unauthorized, illegal, steals data or causes damage.
  3. Grey Hat (Mixed): Unauthorized but may not cause harm, sometimes reports vulnerabilities.

Hacking Techniques:

  1. Phishing: Fake emails/websites to steal credentials.
  2. Social Engineering: Manipulating people to reveal confidential info.
  3. Brute Force: Trying all possible password combinations.
  4. Keylogging: Recording keystrokes to capture passwords.
  5. Man-in-the-Middle: Intercepting communication between two parties.
  6. SQL Injection: Inserting malicious database commands via input fields.

Q5. What is Cyber Stalking and Cyber Fraud? Give examples and mention relevant Indian laws. [5 marks]

Cyber Stalking: Using internet, email, or social media to repeatedly harass, threaten, or monitor a person without consent.

Examples: Sending repeated threatening messages, creating fake profiles to follow someone, posting embarrassing content online.

Cyber Fraud: Dishonest or illegal activities conducted through the internet to steal money, data, or identity.

Examples: Phishing (fake bank emails), identity theft, credit card fraud, lottery scams, fake online shopping sites.

Indian Laws:

  • IT Act 2000, Section 66 — Computer-related offences
  • IT Act 2000, Section 66C — Identity theft
  • IT Act 2000, Section 66D — Cheating by personation
  • IPC Section 354D — Stalking

Q6. Explain Tracert and WireShark. How are they used in network security? [5 marks]

Tracert (Trace Route):

  • Command-line tool that traces the path data packets take to a destination.
  • Shows every router (hop) along the way with response times.
  • Usage: tracert www.google.com
  • Used for: Troubleshooting, finding bottlenecks, mapping routes.

WireShark:

  • Free, open-source network packet analyzer (packet sniffer).
  • Captures and analyzes data packets in real-time.
  • Supports hundreds of protocols (HTTP, TCP, UDP, DNS).
  • Used for: Detecting DoS attacks, analyzing suspicious traffic, network forensics, learning networking.

Security usage: Both tools help security professionals understand network behavior, detect anomalies, trace attack paths, and identify security threats.

Q7. What is a Brute Force attack? How is it different from a Dictionary attack? How can we prevent password cracking? [5 marks]

Brute Force Attack: Tries every possible password combination systematically until the correct one is found. Time-consuming but guaranteed to work eventually.

Dictionary Attack: Tries common words and phrases from a pre-compiled wordlist. Faster than brute force but won't work if the password is not a common word.

Difference:

  • Brute force tries ALL combinations; Dictionary attack tries only common words.
  • Brute force is slower but more thorough; Dictionary attack is faster but limited.

Prevention:

  • Use long, complex passwords (mix of letters, numbers, symbols).
  • Don't use common dictionary words as passwords.
  • Enable account lockout after multiple failed attempts.
  • Use CAPTCHA to prevent automated attacks.
  • Enable Two-Factor Authentication (2FA).

Q8. List and explain at least five measures to protect a computer from cyber threats. [5 marks]

Five Cyber Protection Measures:

  1. Antivirus Software: Install and regularly update antivirus to detect and remove malware (viruses, trojans, spyware).
  2. Strong Passwords: Use passwords with 8+ characters, mixing uppercase, lowercase, numbers, and symbols. Never reuse passwords.
  3. Firewall: Enable firewall to monitor and block unauthorized network traffic. Acts as a barrier between your system and the internet.
  4. Software Updates: Keep OS and applications updated with latest security patches to fix known vulnerabilities.
  5. Regular Backups: Take regular backups of important data on external drives or cloud storage. Protects against ransomware and data loss.

Additional measures: Use 2FA, avoid suspicious links, don't use pirated software, be cautious on public Wi-Fi, educate yourself about threats.
