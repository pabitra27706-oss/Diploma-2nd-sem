HTML Introduction · Elements · Tags

HTML - Introduction

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of web pages using markup.

HTML consists of a series of elements which tell the browser how to display content.

HTML Elements

An HTML element is defined by a start tag, content, and an end tag: <tagname>content</tagname>

Example: <p>This is a paragraph.</p>

HTML Tags

Tags are keywords surrounded by angle brackets like <html>. Most tags come in pairs (opening & closing).

Self-closing tags: <br>, <img>, <hr>

Basic HTML structure
<!DOCTYPE html>
<html>
<head>
  <title>Page Title</title>
</head>
<body>
  <h1>My First Heading</h1>
  <p>My first paragraph.</p>
</body>
</html>

HTML Links · Images · Image Links

HTML Text Links

Links are created with the <a> tag. The href attribute specifies the destination.

<a href="https://example.com">Visit Example</a>
Sample Link (hover)

HTML Images

<img src="url" alt="description">

<img src="image.jpg" alt="Description" width="200">

HTML Image Links

To make an image clickable, wrap it in an <a> tag.

<a href="page.html"><img src="icon.png" alt="Home"></a>

HTML Tables · Bgcolor · Color Codes

HTML Tables

Tables are defined with <table>, rows with <tr>, headers with <th>, and data with <td>.

<table border="1">
  <tr><th>Name</th><th>Age</th></tr>
  <tr><td>John</td><td>25</td></tr>
</table>

Special Tags · Body · Meta · Div · Layouts

HTML Body, Meta, Style
  • <body> - contains visible page content.
  • <meta> - provides metadata (charset, description, keywords).
  • <style> - embeds CSS styles.
<head>
  <meta charset="UTF-8">
  <meta name="description" content="Free tutorials">
  <style> body { font-family: Arial; } </style>
</head>

HTML Div

The <div> tag is a block-level container used for grouping and layout.

<div style="background:#f0f0f0; padding:10px;">Content</div>

HTML Layouts

Modern layouts use <header>, <nav>, <section>, <article>, <aside>, <footer> with CSS.

HTML Frames

Frames (<frameset>) are obsolete in HTML5. Use <iframe> instead.

<iframe src="page.html" width="300" height="200"></iframe>
