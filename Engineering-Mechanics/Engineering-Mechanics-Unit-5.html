Basic Concepts: Load, Effort, MA, VR, Efficiency

Lifting Machine

A lifting machine is a device used to lift heavy loads by applying a smaller effort. It enables us to overcome a large resistance (load) with a small applied force (effort).

Applications & Advantages: Cranes, elevators, screw jacks, pulley systems – they provide mechanical advantage, making work easier.

Load (W) & Effort (P)

Load (W): resistance to be overcome (N).
Effort (P): force applied to lift the load (N).

Mechanical Advantage (MA)

MA = Load / Effort = W / P

Velocity Ratio (VR)

VR = (Distance moved by effort) / (Distance moved by load) = dP / dW

Efficiency (η)

η = (Output work / Input work) = (W × dW) / (P × dP) = MA / VR

Relationships

η = MA / VR    MA = η × VR    VR = MA / η

Law of Machine

Law of Machine

The relationship between effort (P) and load (W) for a machine is given by:

P = mW + C

where m = constant (related to friction), C = constant (effort to overcome friction when load is zero).

From this, MA = W/P = W/(mW + C). As W increases, MA approaches 1/m.

Example: Law of Machine

A machine has law P = 0.2W + 10. Find effort for a load of 500N and MA at that load.

P = 0.2×500 + 10 = 100 + 10 = 110 N
MA = W/P = 500/110 = 4.55

Ideal Machine · Friction · Reversibility

Ideal Machine

No friction, efficiency η = 100% (MA = VR). In practice, η < 100%.

Friction in Machine

Friction causes loss of energy. It is responsible for the constant C in the law of machine.

Maximum MA & Efficiency

As load increases, MA increases and approaches a maximum value 1/m. Efficiency also increases with load.

Reversible & Non-reversible Machines

Reversible (self-locking?) Actually: A machine is reversible if the load can move back when effort is removed (e.g., simple pulley).

Non-reversible (self-locking) if load remains stationary when effort removed (e.g., screw jack).

Condition for reversibility: η > 50% (or MA/VR > 0.5). If efficiency less than 50%, machine is non-reversible (self-locking).

Reversibility Check

A machine has MA = 4 and VR = 6. Is it reversible?

η = MA/VR = 4/6 = 0.667 = 66.7% (>50%) → Reversible

🔄

Simple Pulley Block

VR = number of ropes supporting load (e.g., for n movable pulleys, VR = 2n)
