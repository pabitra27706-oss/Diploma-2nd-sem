2.1.1 Introduction to Analog Circuits

Now a days electronic circuits play a vital role in all engineering field applications. Two main types of electronic circuits are analog electronic circuits and digital electronic circuits.

Analog Electronic Circuits

Rectifier, amplifier and oscillator are most commonly used analog electronic circuits. These circuits can process continuous signals i.e. linear or analog signals. Therefore analog circuits are also called as linear electronics circuits.

  • Rectifier circuit: Used to convert A.C. signals into D.C. signals. Diodes are used for rectifier circuits.
  • Amplifier: Used to increase amplitude of input signals. Amplifier can be constructed using discrete components such as BJT or FET.
Drawbacks of Discrete Component Circuits

Analog circuits constructed using discrete components show many drawbacks such as:

  • Larger size
  • More power consumption
  • Less reliability

To overcome these drawbacks, Integrated Circuits (ICs) are used.

Operational Amplifier (Op Amp)

Operational amplifiers known commonly as Op Amps are very popular building blocks in electronic circuits. Op-amps are used for a variety of applications such as:

  • AC and DC signal amplification
  • Filters
  • Oscillators
  • Voltage regulators
  • Comparators
  • Most consumer and industrial applications

2.1.2.2 Op Amp Parameters

The operational amplifier is simply a high gain, direct coupled amplifier. It is usually designed to amplify signals extending over a wide frequency range and is normally used with external feedback networks. There are various parameters of Op Amp, which are necessary for faithful amplification.

Fig. 2.4: Input bias currents IB1 and IB2; and offset voltage Vio

🔍 Click to view on Google Images
Input Offset Current (Iio)

The input offset current is the difference between the separate currents entering the input terminals of a balanced amplifier.

\[ I_{io} = I_{B1} - I_{B2}, \quad \text{when } V_o = 0 \] ...(2.1)
Input Bias Current (IB)

The input bias current IB is the average value of the current flowing into input terminals with the output at zero volt.

\[ I_B = \frac{I_{B1} + I_{B2}}{2}, \quad \text{when } V_o = 0 \] ...(2.2)
Input Offset Current Drift

It is defined as the ratio of the change of input offset current to the change of temperature.

\[ \text{Input offset current drift} = \frac{\Delta I_{io}}{\Delta T} \] ...(2.3)
Input Offset Voltage (Vio)

It is that voltage required to apply between the input terminals to get output voltage zero, with no input signal.

Input Offset Voltage Drift

Defined as the ratio of the change of input offset voltage to the change in temperature.

\[ \text{Input offset voltage drift} = \frac{\Delta V_{io}}{\Delta T} \] ...(2.4)
Output Offset Voltage

It is the output voltage of the Op Amp when the input terminals are grounded. For ideal Op Amp this parameter is 0V. For practical Op Amp this parameter value must have very low value. To minimize it, offset null pins are used.

Common Mode Range

It is the maximum range of input voltage that can be simultaneously applied to both inputs without causing cut-off or saturation of amplifier stages.

Input Differential Range

It is the maximum difference signal that can be applied safely to the Op Amp input terminals.

Output Voltage Range

It is the maximum output swing that can be obtained without significant distortion.

Full Power Bandwidth

It is the maximum frequency range over that the full output voltage swing can be obtained.

Power Supply Rejection Ratio (PSRR)

The PSRR is the ratio of the change in input offset voltage to the corresponding change in one power supply voltage, with other power supply voltages held constant.

Slew Rate (Sr)

This is the maximum rate of change of output voltage. It is also defined as the rate of change of the closed loop amplifier's output voltage. It is expressed in V/μS unit. Maximum operating frequency of Op Amp depends on slew rate.

\[ F_{max} = \frac{\text{Slew Rate}}{2\pi V_P} \] ...(2.5)
Unity Gain Bandwidth

This is the frequency range from direct current i.e. 0 Hz to that frequency at which the open-loop gain crosses unity.

Input Impedance (Zi)

It is defined as the ratio of input voltage Vi to the input current Ii.

\[ Z_i = \frac{V_i}{I_i} \] ...(2.6)
Output Voltage Swing

The AC output is the maximum unclipped peak to peak output voltage that an Op Amp can produce. Since the quiescent output is ideally zero, the AC output voltage can swing positive or negative. This also indicates the values of positive and negative saturation voltages of the OPAMP. The output voltage never exceeds these limits for a given supply voltages +VCC and −VEE. For 741 IC, it is ±13 V.

Common Mode Rejection Ratio (CMRR)

The ability of differential amplifier to reject a common mode signal is expressed by its CMRR. It is the ratio of differential mode gain Ad to the common mode gain Acm. Usually expressed in decibel (dB).

\[ CMRR = \frac{A_d}{A_{cm}} \] ...(2.7)
\[ CMRR \text{ in dB} = 20 \log_{10}\left(\frac{A_d}{A_c}\right) \] ...(2.8)
Differential Input Resistance (Ri)

It is equivalent resistance measured at either input terminal with other terminal grounded.

2.1.3 Ideal Op Amp

Properties of Ideal Op Amp
  1. Its input impedance is infinite (Zi = ∞).
  2. Its output impedance is zero (Zo = 0).
  3. It has infinite voltage gain (AV = ∞).
  4. It has infinite bandwidth i.e. its open-loop gain tends to infinity — it provides a constant gain for all frequency ranges.
  5. Common Mode Rejection Ratio (CMRR) is infinite.
  6. It produces zero output voltage when V1 = V2.
  7. Characteristics do not drift (swing) with temperature.

The ideal operational amplifier is shown in Fig. 2.5. A signal appearing at the negative terminal (V2) is inverted at the output, a signal appearing at the positive terminal (V1) appears at the output with no change in sign. Hence, the negative terminal is called the "inverting" terminal and the positive terminal the "non-inverting" terminal.

In general, the output voltage is directly proportional to Vd = V1 − V2. The constant of proportionality (AV) is called the voltage gain of the amplifier.

2.1.4 Op Amp Configurations

Op Amp can be operated in any one of two configurations:

Open Loop Configuration

Op Amp without feedback

Close Loop Configuration

Op Amp with feedback

2.1.4.1 Op Amp Open Loop Configuration

The Op Amp without feedback is known as open loop configuration of Op Amp.

Open-Loop Gain (AOL)

If 'Vd' is the differential input voltage of an Op Amp (it is very small) and Vo is the output voltage, then open-loop gain is defined as the ratio of the output voltage Vo to the differential input voltage Vd. It is termed as open-loop gain because possible feedback connection between output and input terminals are absent.

\[ A_{OL} = \frac{V_o}{V_d} \] ...(2.9)

In open loop as feedback is absent, so gain is infinite and non-controllable. To control gain and operate Op Amp as per requirement, feedback is essential. Feeding some part or complete output signal to input is called as feedback.

2.1.4.2 Op Amp Close Loop Configuration

The Op Amp without feedback is not a very useful device, since the extremely small voltage at the input will cause it to go into saturation at the output. Hence, it is necessary to apply feedback to get finite voltage gain. When feedback is applied, the characteristics of the Op Amp are determined largely by the feedback network. Op Amp with feedback is called close loop configuration.

Closed-Loop Gain (ACL)

The gain of the amplifier is called closed-loop gain because the feedback resistor closes a loop from the Op Amp output terminal to the inverting input terminal i.e. negative terminal.

\[ A_{CL} = \frac{V_o}{V_i} \] ...(2.10)

Fig. 2.6: Op-Amp close-loop configuration — showing R₁, Rf, input V₁, inverting terminal with Zi, output V₀, and supply ±VCC

🔍 Click to view on Google Images

The closed-loop gain for the inverting amplifier is equal to −(Rf/R1).

Output Impedance (Zo) — Closed Loop

The closed-loop output impedance Zo(CL) is defined as the ratio of the open-loop output impedance Zo(OL) to the loop gain.

\[ Z_{o(CL)} = \frac{Z_{o(OL)}}{\text{Loop gain}} \] ...(2.11)

where, Loop gain = Open-loop gain − Closed-loop gain.

The Virtual Ground

To illustrate the features of an operational amplifier, consider the feedback circuit Fig. 2.7, in which negative voltage feedback is produced by the resistor Rf connected between the input and output. The feedback is negative because of phase inversion in the amplifier.

Fig. 2.7: Block diagram of Op Amp with feedback — showing input Vi, R₁, branch point S, V₁, gain A, Rf, output Vo

🔍 Click to view on Google Images

Derivation of Virtual Ground Concept

Applying KCL at branch point S, since the amplifier input impedance is large, the current in R equals the current in Rf:

\[ \frac{V_i - V_1}{R} = \frac{V_1 - V_o}{R_f} \]

Introducing \( V_1 = \frac{-V_o}{A} \) and rearranging:

\[ V_o\left(1 + \frac{1}{A} + \frac{R_f}{A}\right) = -\frac{R_f}{R} \cdot V_i \]

Since the gain A is very large:

\[ V_o = -\frac{R_f}{R} \cdot V_i \]

This means the output voltage is just the input voltage multiplied by the constant factor −(Rf/R).

Impedance at Virtual Ground

The impedance of branch point S to ground is given by:

\[ Z_S = \frac{V_1}{I_i} = \frac{V_1 R_f}{V_1 - V_o} = \frac{R_f}{1 + A} \] ...(2.13)

The impedance is very low if the gain is large. Typical values: Rf = 105 Ω and A = 104, giving impedance of 10 Ω. The low impedance results from the negative feedback voltage, which cancels the input signal at 'S' and tends to keep the branch point at ground potential. For this reason, the point 'S' is called virtual ground.

Note: Although 'S' is kept at ground potential by feedback action, no current to ground exists at this point.

2.1.5.1 Inverting Mode Amplifier

Inverting Amplifier

The inverting operational amplifier configuration is one of the simplest and most commonly used op amp operating mode. In inverting mode amplifier, the output is exactly 180° out of phase with respect to input (i.e. if a positive voltage is applied, output will be negative). Output is an inverted (in terms of phase) amplified version of input. It uses negative feedback, meaning the feedback signal opposes the input signal.

\[ V_O = -\frac{R_f}{R_1} \times V_i \] ...(2.14)

Working Principle

The positive input terminal of the amplifier is grounded and the input signal e1 is applied to the negative input terminal through resistor R1. The feedback applied through Rf from the output to the input terminal is negative.

Considering that the Op Amp is ideal (infinite gain), with infinite voltage gain, the potential difference between the input terminals must be zero. As the input impedance of the amplifier is infinite, the input current to the amplifier is zero.

For ideal op-amp, two conditions must be satisfied:

  1. The potential difference between the amplifier terminals is zero.
  2. The current into each input terminal is zero.

Derivation of Inverting Amplifier Gain

Since the terminal (1) has the same potential as terminal (2), and terminal (2) is grounded, terminal (1) is also virtually grounded.

Therefore: i1 = if

\[ \frac{e_I - e_s}{R_I} = \frac{e_s - e_o}{R_f} \]

As es = 0 (virtual ground):

\[ \frac{e_I}{R_I} = \frac{-e_o}{R_f} \]
\[ \frac{e_o}{e_I} = -\frac{R_f}{R_I} \] ...(2.16)
Key Points of Inverting Amplifier
  1. It gives phase difference of 180° between input and output.
  2. Input impedance is lower.
  3. Smaller bandwidth as compared to non-inverting amplifier.
  4. The gain depends on the ratio of Rf/R1 — can be set to any value, even less than 1.

2.1.5.2 Non-Inverting Mode Amplifier

Non-Inverting Amplifier

The non-inverting amplifier configuration is one of the most popular and widely used forms. The op amp non-inverting amplifier circuit provides a high input impedance along with all the advantages gained from using an operational amplifier.

Non-inverting amplifier is one in which the output is in phase with respect to input (i.e. if input is positive voltage, output will be positive). Output is a non-inverted (in terms of phase) amplified version of input.

\[ V_o = \left(1 + \frac{R_f}{R_1}\right) \times V_i \] ...(2.17)

Fig. 2.10: Non-Inverting Amplifier — Input applied to non-inverting (+) terminal, R₁ to ground, Rf from output to inverting (−) terminal

🔍 Click to view on Google Images
Circuit Operation

The input signal is applied directly to the non-inverting (positive) input terminal. The feedback resistor Rf is connected between the output terminal and the negative input terminal. R1 is connected between the inverting terminal and ground.

In this case, es = 0 but e1 ≠ 0, meaning that the non-inverting circuit has no virtual ground at either one of its input terminals.

Derivation of Non-Inverting Amplifier Gain

Since e1 = e2 and e1 = i1:

\[ e_1 = e_o \cdot \frac{R_1}{R_1 + R_f} = e_2 \]

Thus:

\[ \frac{e_0}{e_2} = \frac{R_f + R_1}{R_1} = \frac{R_f}{R_1} + \frac{R_1}{R_1} = \frac{R_f}{R_1} + 1 \]
\[ A_{CL} = \frac{e_0}{e_2} = 1 + \frac{R_f}{R_1} \] ...(2.18)
Voltage Follower (Buffer)

If Rf = 0 and R1 = ∞, then the gain is exactly equal to one and the amplifier acts as a voltage follower — the output voltage follows the input voltage exactly. The advantage is impedance buffering, providing isolation of signal source and load, preventing undesired interactions or loading effects.

Advantages of Non-Inverting Amplifier
  1. There is no phase shift between input and output.
  2. The input impedance is higher than inverting configuration.
  3. Larger bandwidth as compared to inverting amplifier.
  4. Circuit uses negative feedback.

2.2.1 Op Amp as an Adder (Summing Amplifier)

Summing Amplifier

Adder and subtractor circuits using Op Amp are used to perform arithmetic operations like addition, subtraction etc. Op Amp adder is also called summing amplifier. In inverting configuration of an Op Amp, if more than one input is given to inverting terminal, the resultant circuit works as a summing amplifier or adder.

This circuit provides an output voltage proportional to or equal to the algebraic sum of two or more input voltages each multiplied by a constant gain factor. It can be used to add AC or DC signals.

Fig. 2.11: Op Amp as an Adder circuit — Two inputs VA and VB through R₁ and R₂, feedback Rf, point P (virtual ground), output VO

🔍 Click to view on Google Images

Derivation — Summing Amplifier Output

Applying KCL at inverting input node P:

\[ i_1 + i_2 = i_f + I_{B2} \]

As Op Amp is ideal, IB2 = 0, so i1 + i2 = if

\[ \frac{V_A - V_P}{R_1} + \frac{V_B - V_P}{R_2} = \frac{V_P - V_o}{R_f} \]

Due to virtual ground, VP = 0:

\[ \frac{V_A}{R_1} + \frac{V_B}{R_2} = \frac{-V_o}{R_f} \]
\[ V_0 = -\left(\frac{R_f}{R_1}V_A + \frac{R_f}{R_2}V_B\right) \] ...(2.19)
Special Cases

Case 1: If R₁ = R₂ = R

\[ V_0 = -\frac{R_f}{R}(V_A + V_B) \quad \text{...Summing amplifier} \] ...(2.20)

Case 2: If R₁ = R₂ = Rf = R

\[ V_o = -(V_A + V_B) \]

Case 3: If R₁ = R₂ = 2Rf (Averaging Circuit)

\[ V_0 = -\frac{V_1 + V_2}{2} \] ...(2.21)

2.2.2 Op Amp as a Differentiator

Differentiator Circuit

The circuit in which the output voltage waveform is the derivative of the input waveform is called a differentiator. In differentiator circuit, the reactance XC (capacitor) is connected to the input terminal of the inverting amplifier while the resistor Rf forms the negative feedback element.

The differentiator produces an output voltage which is directly proportional to the input voltage's rate-of-change with respect to time. The faster or larger the change to the input voltage signal, the greater the input current, and the greater will be the output voltage change — becoming more of a "spike" in shape.

How It Works

The input signal is applied to the capacitor. The capacitor blocks DC content — no current flows to the amplifier summing point, resulting in zero output voltage for DC. The capacitor only allows AC type input voltage changes to pass through and whose frequency is dependent on the rate of change of the input signal.

The differentiator may be constructed from a basic inverting amplifier if the input resistor R₁ is replaced by a capacitor Cin.

Differentiator Output Equation
\[ V_O = -R_F \cdot C_1 \cdot \frac{dV_{in}}{dt} \] ...(2.22)

The (−) sign indicates a 180° phase shift of the output waveform VO with respect to the input signal.

2.2.3 Op Amp as an Integrator

Integrator Circuit

The circuit in which the output voltage waveform is the integral of the input voltage waveform is called as integrator or integration amplifier. Integrator circuit is obtained by using a basic inverting amplifier. If the feedback resistor RF is replaced by a capacitor CF, then the circuit acts as integrator.

In integrator circuit, the position of the capacitor and resistor have been reversed as that of differentiator circuit.

Key Concepts
  • An integrator circuit which consists of active devices is called an Active integrator.
  • An active integrator provides a much lower output resistance and higher output voltage than is possible with a simple RC circuit.
  • Integrator circuits are usually designed to produce a triangular wave output from a square wave input.
  • Integration is a process of continuous additions.
  • The most popular application is to produce a ramp of output voltage — linearly increasing or decreasing voltage.

Fig. 2.14: Op Amp Integrator circuit — R₁ at input, Cf in feedback path, +VCC/−VEE, output VOUT

🔍 Click to view on Google Images
Integrator Output Equation
\[ V_o = \frac{1}{R_1 C_F}\int_{0}^{t}V_{in}\,dt + C \] ...(2.23)

The output voltage is directly proportional to the negative integral of the input voltage and inversely proportional to the time constant R₁CF.

Fig. 2.15: Ideal output waveforms — (a) Square wave input → triangular wave output with slope = 1/Vs, (b) Sine wave input → cosine wave output

🔍 Click to view on Google Images

When Vin = 0, the integrator works as an open-loop amplifier as CF acts as an open circuit (CF = ∞). The input offset voltage Vio and the input current charging capacitor CF produce error voltage at the output. Therefore, to reduce the error voltage, a resistor RF is connected across the feedback capacitor CF.

Cutoff Frequency of Integrator (Low Pass Filter)
\[ F_1 = \frac{1}{2\pi R_1 C_f} \] ...(2.24)

The integrator acts as a low pass filter. The cut-off frequency is inversely proportional to feedback components Rf and Cf.

Applications of Integrator
  • Analog computers
  • Analog to digital converter (ADC)
  • Signal wave shaping circuits
  • Ramp or sweep generators
  • Filters

Op Amp Applications Overview

The Op Amp was originally developed for the requirement of analog computers. As Op Amp is a high gain direct coupled amplifier with the feature of externally voltage gain controlled, it finds many applications in signal processing and conditioning applications. Due to its low cost, high performance and versatile nature, it is used in many analog electronics circuits.

Op Amp is always used in close loop mode with negative feedback and the voltage gain is controlled by external components Ri and Rf. When the power supply is connected, there is output even when the two inputs are grounded — this is called offset. It can be made zero by connecting a 10 KΩ POT between pin 1 and 5 and connecting the wiper to Pin 4.

Example 2.1.2

For the inverting amplifier mode of Op Amp, Rf = 10 KΩ and Ri = 2 KΩ. Calculate close loop voltage gain ACL.

Solution
ACL = −Rf / R1 = −(10 KΩ / 2 KΩ)
ACL = −5
Example 2.1.3

For the non-inverting amplifier mode of Op Amp, Rf = 10 KΩ and Ri = 1 KΩ. Calculate close loop voltage gain ACL and feedback factor.

Solution
ACL = 1 + (Rf / R1) = 1 + (10 KΩ / 1 KΩ)
ACL = 11
β = R1 / (R1 + Rf) = 1 KΩ / (1 KΩ + 10 KΩ)
β = 0.09
Example 2.1.4

Calculate the CMRR of Op Amp that has a differential gain of 300000 and common mode gain is 12.66.

Solution
CMRR = Ad / AC = 300000 / 12.66
CMRR = 23696.68
CMRR in dB = 20 log10(Ad/AC) = 20 log10(23696.68)
CMRR = 87.49 dB

Calculate the feedback resistor value if in Non-inverting Op Amp amplifier input resistor is of 4 KΩ and required gain for specific application is 13.

Solution

Given: Rin = 4 KΩ, Gain = 13, Non-inverting mode

Gain = 1 + (Rf/Rin)
13 = 1 + (Rf/4 KΩ)
12 = Rf/4 KΩ
Rf = 48 KΩ

For an Op Amp integrator, if cut off frequency is 159 Hz, Rin = 1 KΩ and Rf = 100 KΩ, Calculate the value of feedback capacitance.

Solution

Given: Rin = 1 KΩ, Rf = 100 KΩ, Fc = 159 Hz

F₁ = 1/(2πRfCf)
159 = 1/(2 × 3.14 × 100 × 10³ × Cf)
Cf = 1/(2 × 3.14 × 100 × 159 × 10³)
Cf = 1.0 × 10⁻⁸ = 0.01 μF

Solution

Given: Rf = 10 KΩ, Cin = 0.01 μF

f₁ = 1/(2πRfCin) = 1/(2 × 3.14 × 10 × 10³ × 0.01 × 10⁻⁶)
f₁ = 1.59 KHz
Example 2.2.3

Calculate output of two input inverting summing amplifier with feedback resistor of 4 KΩ and two inputs are V₁ = 3V and V₂ = 4V. Two input resistors are R₁ = 4 KΩ and R₂ = 8 KΩ.

Solution

Given: Rf = 4 KΩ, R₁ = 4 KΩ, R₂ = 8 KΩ, V₁ = 3V, V₂ = 4V

\[ V_o = -\left(\frac{R_f}{R_1}V_A + \frac{R_f}{R_2}V_B\right) \] \[ = -\left[\frac{4}{4}\times 3 + \frac{4}{8}\times 4\right] = -[3 + 2] \]
Vo = −5 V

Q 2.1

The feedback path of Op Amp integrator consists of:

  • a. Resistor
  • b. Capacitor
  • c. Inductor
  • d. Diode
Answer: b. Capacitor

In an integrator, the feedback resistor RF is replaced by a capacitor CF.

Q 2.2

If input signal is fed to inverting input through capacitor and feedback path consists of resistor, then that Op Amp circuit is called as:

  • a. Adder circuit
  • b. Integrator circuit
  • c. Differentiator circuit
  • d. Non-inverting amplifier
Answer: c. Differentiator circuit

In differentiator, capacitor is at input and resistor is in feedback path.

Q 2.3

Two input terminals of Op Amp are known as:

  • a. High and low
  • b. Inverting and Non-inverting
  • c. Phase and neutral
  • d. Integrator and differentiator
Answer: b. Inverting and Non-inverting

The (−) terminal is inverting and the (+) terminal is non-inverting.

Q 2.4

For biasing, Op Amp IC 741 requires:

  • a. Single power supply
  • b. Two power supplies
  • c. Four power supplies
  • d. No power supply
Answer: b. Two power supplies

IC 741 requires dual supply: +VCC (typically +15V) and −VEE (typically −15V).

Q 2.5

For the differentiator circuit to operate as high pass filter, time constant of circuit must be:

  • a. High
  • b. Very high in comparison to the time period of input signal
  • c. Small
  • d. Very small in comparison to the time period of input signal
Answer: d. Very small in comparison to the time period of input signal

For high pass filter operation, the time constant (RfCin) must be much smaller than the input signal's time period.

Q 2.6

For inputs averaging circuit:

  • a. Rin = Rf / 4
  • b. Rin = Rf + 4
  • c. Rin = Rf
  • d. Rin = Rf × 16
Answer: c. Rin = Rf

For a 2-input averaging circuit, R1 = R2 = 2Rf. For a 4-input averaging circuit, Rin = 4Rf. The general condition for averaging is Rin = nRf where n is number of inputs. However, considering the options and typical exam context, Rin = Rf produces the averaging function when gain = 1.

B. Subjective Questions

Q1. List six characteristics of an ideal Op Amp.

  1. Infinite input impedance (Zi = ∞)
  2. Zero output impedance (Zo = 0)
  3. Infinite voltage gain (AV = ∞)
  4. Infinite bandwidth
  5. Infinite CMRR
  6. Zero output voltage when V₁ = V₂
  7. Characteristics do not drift with temperature

Q2. State the various operational amplifier parameters.

The various Op Amp parameters are: Input offset current (Iio), Input bias current (IB), Input offset current drift, Input offset voltage (Vio), Input offset voltage drift, Output offset voltage, Common mode range, Input differential range, Output voltage range, Full power bandwidth, PSRR, Slew rate (Sr), Unity gain bandwidth, Input impedance (Zi), Output voltage swing, CMRR, and Differential input resistance (Ri).

Q3. Explain the non-inverting operational amplifier with neat circuit diagram.

In non-inverting amplifier, the input signal is applied to the non-inverting (+) terminal. Rf connects output to inverting (−) terminal. R₁ connects inverting terminal to ground. Output is in phase with input. Gain = 1 + (Rf/R₁). Advantages: high input impedance, no phase shift, larger bandwidth.

Q4. Explain Op Amp as a buffer circuit.

A voltage follower (buffer) is a non-inverting amplifier with Rf = 0 and R₁ = ∞. Gain = 1. Output follows input exactly. Provides impedance buffering — very high input impedance and very low output impedance. Used to isolate signal source from load, preventing loading effects.

Q5. Explain the significance of slew rate parameter in communication circuits.

Slew rate is the maximum rate of change of output voltage (V/μS). It determines the maximum operating frequency: Fmax = Slew Rate / (2πVP). In communication circuits, signals change rapidly; if slew rate is insufficient, the output cannot follow fast input changes, causing distortion. Higher slew rate enables processing of higher frequency signals faithfully.

Q7. Calculate output of three input inverting summing amplifier with feedback resistor of 2 KΩ and three inputs are V₁ = 2V, V₂ = 4V and V₃ = 6V. Three input resistors are R₁ = 2 KΩ, R₂ = 4 KΩ and R₃ = 6 KΩ.

Vo = −[(Rf/R₁)V₁ + (Rf/R₂)V₂ + (Rf/R₃)V₃]

= −[(2/2)×2 + (2/4)×4 + (2/6)×6]

= −[2 + 2 + 2] = −6 V

Q8. State effect of variable resistor connected in feedback path of inverting configuration of Op Amp.

When a variable resistor (potentiometer) is connected in the feedback path of an inverting Op Amp, the gain ACL = −Rf/R₁ can be continuously varied. Increasing Rf increases the gain magnitude, and decreasing Rf decreases it. This provides a variable gain amplifier useful for volume control, signal conditioning, and calibration applications.

Q9. Explain construction of Op Amp based subtractor using adder circuit.

An Op Amp subtractor (difference amplifier) can be constructed by applying one input to the inverting terminal through R₁ and another input to the non-inverting terminal through R₂. A resistor Rf is placed in feedback from output to inverting terminal, and another resistor from non-inverting terminal to ground. With equal resistors (R₁ = R₂ = Rf = R), the output is: Vo = V₂ − V₁, which gives the difference of two inputs.

Q10. A differentiator has feedback resistor of 20 KΩ and cut off frequency is 1.5 KHz. Calculate input capacitor value.

Given: Rf = 20 KΩ, f₁ = 1.5 KHz

f₁ = 1/(2πRfCin)

Cin = 1/(2π × 20 × 10³ × 1.5 × 10³)

Cin = 1/(2 × 3.14 × 20 × 10³ × 1.5 × 10³)

Cin = 1/(188400) = 5.31 × 10⁻⁶

Cin ≈ 5.3 nF (0.0053 μF)

Unit Summary

Key Takeaways
  • The operational amplifier, more commonly known as Op Amp, is an analog circuit.
  • Op Amps perform many arithmetic functions, linear and nonlinear operations in the analog or continuous domain.
  • Op Amps are also used in several kinds of analog amplifiers and active filters.
  • Op Amp with close configuration having negative feedback can be used in inverting and non-inverting amplifier mode.
  • In inverting and non-inverting amplifier mode, close loop gain depends on feedback resistor and input resistor.
  • Non-inverting amplifier with unity gain is used as an analog buffer.
