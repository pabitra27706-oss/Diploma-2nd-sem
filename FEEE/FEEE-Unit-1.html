Passive Components: Resistor · Capacitor · Inductor

Concept of Passive Components

Passive components cannot introduce net energy into the circuit. They only consume or store energy.

Resistor

Opposes current flow. Unit: Ohm (Ω).
V = I·R (Ohm's law). Energy dissipated as heat.

R = ρ·L/A

Capacitor

Stores energy in electric field. Unit: Farad (F).
I = C·dV/dt. Blocks DC, passes AC.

C = ε·A/d

Inductor

Stores energy in magnetic field. Unit: Henry (H).
V = L·dI/dt. Passes DC, blocks AC.

L = N²·μ·A/l
Series/Parallel combinations

Resistors series: Req = R₁ + R₂ + …
Parallel: 1/Req = 1/R₁ + 1/R₂ + …

Capacitors series: 1/Ceq = 1/C₁ + 1/C₂ + …
Parallel: Ceq = C₁ + C₂ + …

Inductors series: Leq = L₁ + L₂ + …
Parallel: 1/Leq = 1/L₁ + 1/L₂ + …

Signal Waveforms: DC/AC · Voltage/Current · Periodic

DC vs AC

DC (Direct Current)

Constant polarity. Example: battery

AC (Alternating Current)

Sinusoidal, changes polarity. Example: mains supply

Voltage vs Current

Voltage is potential difference; current is flow of charge. They can be DC or AC.

Periodic vs Non-periodic

Periodic: repeats after T (sine, square). Non-periodic: random, transient.
Common Periodic Waveforms
  • Sinusoidal: v(t) = Vm sin(ωt + φ)
  • Square wave: alternates between two levels
  • Triangular / Sawtooth

Voltage/Current Sources: Ideal · Non-ideal · Independent/Dependent

Ideal vs Non-ideal

Ideal voltage source: maintains constant voltage regardless of current (zero internal resistance).

Ideal current source: maintains constant current regardless of voltage (infinite internal resistance).

Non-ideal (practical) sources have internal resistance: voltage source with series Rs, current source with parallel Rs.

Independent Source

Value fixed or time-varying but not controlled by other circuit variables.

Dependent Source

Value depends on voltage/current elsewhere in circuit (VCVS, CCVS, VCCS, CCCS).
Symbols

Independent sources: circle. Dependent sources: diamond shape.

Source Transformation · Simple Problems

Source Transformation Principle

A voltage source V in series with resistance R can be transformed into a current source I = V/R in parallel with same resistance R, and vice‑versa.

V
─── R ──
⇄
I = V/R
─── R ──
Example 1: Voltage → Current

Convert a 12V voltage source with series 4Ω to a current source.

I = V / R = 12V / 4Ω = 3A
3A current source in parallel with 4Ω resistor.
Example 2: Current → Voltage

Convert a 5A current source with parallel 10Ω to a voltage source.

V = I × R = 5A × 10Ω = 50V
50V voltage source in series with 10Ω resistor.

Problem 1: Transform the 20V voltage source with series 5Ω resistor into an equivalent current source.

Solution:
I = V / R = 20V / 5Ω = 4A
Equivalent: 4A current source in parallel with 5Ω resistor.

Problem 2: Transform the 2A current source with parallel 8Ω resistor into an equivalent voltage source.

Solution:
V = I × R = 2A × 8Ω = 16V
Equivalent: 16V voltage source in series with 8Ω resistor.

Problem 3 (mixed): Use source transformation to find the current through a 4Ω load connected across terminals A-B: circuit has a 10V source in series with 2Ω, transformed?

Hint: Transform 10V + 2Ω → 5A current source in parallel with 2Ω. Then combine parallel resistances if needed. Current through 4Ω load can be found by current divider.
Simple transformation steps
  1. Identify source and its series/parallel resistance.
  2. Apply V = I·R or I = V/R.
  3. Draw new source with same resistance in opposite configuration.
  4. Polarity/direction: current source points toward positive terminal of voltage source.
