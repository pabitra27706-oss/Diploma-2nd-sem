Basic AC Concepts

Key Definitions

Angular speed (ω)

ω = 2πf rad/s

Cycle

One complete set of positive and negative values.

Frequency (f)

Cycles per second (Hz)

Periodic time (T)

T = 1/f seconds

Amplitude

Maximum value (Vm or Im)

RMS value

Vrms = Vm/√2 (for sine wave)

Average value

Vavg = 2Vm/π (for half cycle)

Form Factor

= RMS / Avg = 1.11 for sine

Peak Factor

= Peak / RMS = √2 for sine

Impedance (Z)

Total opposition Z = √(R² + X²)

Phase difference

Angle between voltage and current

Problem: A sinusoidal voltage has peak value 100V. Find RMS, average, form factor, and peak factor.

Vrms = 100/√2 = 70.71 V
Vavg (half cycle) = 2×100/π = 63.66 V
Form factor = 70.71/63.66 = 1.11
Peak factor = 100/70.71 = 1.414

Representation: Exponential · Complex · Polar

Forms

A sinusoidal quantity v = Vm sin(ωt + φ) can be represented as:

  • Exponential form: Vm ej(ωt+φ) (using Euler's identity)
  • Complex (rectangular) form: V = Vrms (cosφ + j sinφ) = a + jb
  • Polar form: V = Vrms ∠ φ
Example

v = 100 sin(314t + 30°). Write in polar and complex form (RMS).

Vrms = 100/√2 = 70.71
Polar: 70.71 ∠ 30°
Complex: 70.71(cos30 + j sin30) = 61.24 + j35.36

Pure R, L, C Circuits (with phasors)

Pure Resistance

v = Vm sinωt, i = Im sinωt (in phase)

Phasor: V and I aligned.

Pure Inductance

v = Vm sinωt, i = Im sin(ωt - 90°) (current lags)

Phasor: I lags V by 90°.

Pure Capacitance

v = Vm sinωt, i = Im sin(ωt + 90°) (current leads)

Phasor: I leads V by 90°.

R: V & I →
L: I ↓ V →
C: I ↑ V →

Series R-L, R-C, R-L-C Circuits

R-L Series

Z = √(R² + XL²), XL = ωL

Phase angle φ = tan⁻¹(XL/R) (current lags)

R-C Series

Z = √(R² + XC²), XC = 1/ωC

Phase angle φ = tan⁻¹(-XC/R) (current leads)

R-L-C Series

Z = √(R² + (XL - XC)²)

φ = tan⁻¹((XL-XC)/R)

Problem: A series R-L circuit has R=10Ω, L=0.1H, f=50Hz, V=100V. Find current and phase angle.

XL = 2π×50×0.1 = 31.42Ω
Z = √(10² + 31.42²) = √(100 + 987) = √1087 = 32.97Ω
I = V/Z = 100/32.97 = 3.03 A
φ = tan⁻¹(31.42/10) = tan⁻¹(3.142) = 72.3° (current lags)

Parallel R-L, R-C, R-L-C Circuits

Parallel AC circuits

Use admittance Y = 1/Z = G + jB. For R-L: Y = 1/R - j(1/XL). For R-C: Y = 1/R + j(1/XC).

R-L parallel example

R=20Ω, XL=15Ω, V=100∠0°. Find total current.

IR = 100/20 = 5∠0°, IL = 100/15 = 6.67∠-90°
Itotal = 5 - j6.67 = √(5²+6.67²) ∠ tan⁻¹(-6.67/5) = 8.33 ∠ -53.1° A

Impedance Triangle · Power Triangle

Impedance Triangle

Z² = R² + X², φ = tan⁻¹(X/R)

Power Factor

cosφ = R/Z = P/S

Active Power (P)

P = VI cosφ (Watts)

Reactive Power (Q)

Q = VI sinφ (VAR)

Apparent Power (S)

S = VI (VA)

Power Triangle

S² = P² + Q²

Problem: A load draws 5A at 230V with power factor 0.8 lagging. Find P, Q, S.

S = V·I = 230×5 = 1150 VA
P = S·cosφ = 1150×0.8 = 920 W
Q = S·sinφ = 1150×0.6 = 690 VAR (since sin⁻¹0.8=36.87°, sin=0.6)

Star and Delta Connections

Star (Y)

R Y B

VL = √3 Vph, IL = Iph

Delta (Δ)

R─Y─B─R

VL = Vph, IL = √3 Iph

Problem: A balanced star-connected load has phase voltage 230V. Find line voltage. If same load is delta-connected, find phase current if line current is 10A.

Star: VL = √3 × 230 = 398.4 V
Delta: Iph = IL / √3 = 10 / 1.732 = 5.77 A
