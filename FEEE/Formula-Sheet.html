UNIT 2: Electric & Magnetic Circuits

⚡ Basic Electrical
$$V = W/Q$$
$$I = Q/t$$
$$P = VI = I^2R = V^2/R$$
$$E = P × t = VIt$$
🧲 Magnetic Circuits
MMF: $$F = NI$$ (At)
H: $$H = NI/l$$ (A/m)
Flux: $$\phi = B×A$$ (Wb)
B: $$B = \phi/A = \mu H$$ (T)
μ: $$\mu = \mu_0 \mu_r$$
Reluctance: $$S = l/(\mu A) = F/\phi$$
📉 Losses
Hysteresis: $$P_h = K_h f B_m^n$$
Eddy: $$P_e = K_e f^2 B_m^2$$
⚡ EMI
Faraday: $$e = -N d\phi/dt$$
L: $$L = N\phi/I$$
M: $$M = k\sqrt{L_1L_2}$$
Energy: $$E = \frac12 LI^2$$
Fleming's Left (Motor): Thumb = Force, Index = Field, Middle = Current
Fleming's Right (Generator): Thumb = Motion, Index = Field, Middle = Current

UNIT 3: AC Circuits

📊 AC Fundamentals
$$\omega = 2\pi f$$
$$T = 1/f$$
$$V_{avg} = 0.637 V_m$$
$$V_{rms} = 0.707 V_m$$
Form Factor = 1.11
Peak Factor = 1.414
🔮 Phasor Forms
Exponential: $$V = V_m e^{j\theta}$$
Polar: $$V = V_m \angle \theta$$
💡 Pure Elements
R: $$V=IR$$, φ=0°
L: $$V=IX_L$$, φ=+90°
C: $$V=IX_C$$, φ=-90°
🔧 Reactance
$$X_L = 2\pi f L = \omega L$$
$$X_C = 1/(2\pi f C) = 1/(\omega C)$$
📐 Impedance (Z)
R-L: $$Z = \sqrt{R^2 + X_L^2}$$
R-C: $$Z = \sqrt{R^2 + X_C^2}$$
R-L-C: $$Z = \sqrt{R^2 + (X_L - X_C)^2}$$
⚡ Power
Active: $$P = VI\cos\phi$$ (W)
Reactive: $$Q = VI\sin\phi$$ (VAR)
Apparent: $$S = VI$$ (VA)
PF = cos φ = P/S
🔺 Star-Delta
Star: $$V_L = \sqrt{3}V_{ph},\ I_L = I_{ph}$$
Delta: $$V_L = V_{ph},\ I_L = \sqrt{3}I_{ph}$$

UNIT 6: Op-Amp

🔺 Configurations
Inverting: $$A_v = -R_f/R_{in}$$
Non-inverting: $$A_v = 1 + R_f/R_{in}$$
Follower: A_v = 1
Adder: $$V_{out} = -(R_f/R_1 V_1 + R_f/R_2 V_2)$$
Integrator: $$V_{out} = -\frac{1}{RC}\int V_{in} dt$$
Differentiator: $$V_{out} = -RC\frac{dV_{in}}{dt}$$
