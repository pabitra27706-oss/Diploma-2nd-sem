6.1 Introduction to Op-Amp & Features of Ideal Op-Amp

What is an Operational Amplifier (Op-Amp)? 🔥⭐

Definition: Operational Amplifier (Op-Amp)

An Operational Amplifier (Op-Amp) is a high-gain, direct-coupled, differential amplifier that can amplify the difference between two input voltages. It is a versatile analog IC that can perform various mathematical operations like addition, subtraction, integration, differentiation, and comparison — which is why it is called "Operational" amplifier.

Understanding the Name
  • "Operational" — because it was originally designed to perform mathematical OPERATIONS in analog computers
  • "Amplifier" — because its primary function is to AMPLIFY signals
  • "Differential" — because it amplifies the DIFFERENCE between its two inputs
  • "Direct-coupled" — because it can amplify DC signals (no coupling capacitors between stages internally)

⭐ Key Equation — Op-Amp Output:
Vout = AOL × (V⁺ − V⁻)

Where AOL is the open-loop voltage gain of the Op-Amp. This means the Op-Amp amplifies the difference between the two input voltages.

Features (Characteristics) of an Ideal Op-Amp 🔥⭐

An Ideal Op-Amp Has These Properties

An "ideal" Op-Amp is a theoretical perfect amplifier. Real Op-Amps come close to these values but never achieve them perfectly. These ideal properties make circuit analysis much simpler.

Sl. Parameter Ideal Value Practical Value (IC 741) Explanation
1 Open-Loop Voltage Gain (AOL) Infinite (∞) ~2 × 10⁵ (200,000) Even the tiniest input difference produces huge output
2 Input Impedance (Zin) Infinite (∞) ~2 MΩ Draws zero current from input source — doesn't load the source
3 Output Impedance (Zout) Zero (0) ~75 Ω Can deliver maximum voltage/current to load without drop
4 Bandwidth (BW) Infinite (∞) ~1 MHz (unity gain BW) Can amplify signals of any frequency equally
5 CMRR (Common Mode Rejection Ratio) Infinite (∞) ~90 dB Rejects signals common to both inputs completely (noise rejection)
6 Slew Rate Infinite (∞) ~0.5 V/μs Output can change infinitely fast — no delay
7 Input Offset Voltage Zero (0) ~1–5 mV When both inputs are zero, output is exactly zero
8 Input Bias Current Zero (0) ~80 nA No current flows into input terminals
9 Input Offset Current Zero (0) ~20 nA Difference between bias currents at two inputs is zero
10 Drift (with temperature) Zero (0) Small but non-zero Parameters don't change with temperature
🔥 Exam Must-Know — 5 Key Ideal Properties:
  1. Open-loop gain = ∞
  2. Input impedance = ∞
  3. Output impedance = 0
  4. Bandwidth = ∞
  5. CMRR = ∞

These 5 points are asked in almost every WBSCTE exam. Remember: "Gain, Input Z, BW, CMRR → ∞; Output Z → 0"

Simple Analogy

Think of an ideal Op-Amp like a perfect loudspeaker:

  • Infinite gain = Even a whisper gets amplified to any volume you want
  • Infinite input impedance = The microphone doesn't drain your voice energy
  • Zero output impedance = Can drive any speaker without losing volume
  • Infinite bandwidth = Works perfectly for bass, treble, and all frequencies

Two Golden Rules of Ideal Op-Amp 🔥⭐

The Two Golden Rules (with Negative Feedback)

When an ideal Op-Amp is used with negative feedback (output connected back to the inverting input through a resistor), these two rules apply:

Rule 1: No Current Flows Into Input Terminals
I⁺ = I⁻ = 0

Because input impedance is infinite, the Op-Amp draws zero current from the signal source through its input pins.

Rule 2: Both Input Terminals Are at the Same Voltage
V⁺ = V⁻

Because gain is infinite, the difference (V⁺ − V⁻) must be essentially zero for the output to remain finite. So both inputs are at the same potential.

These two rules are the foundation for analyzing ALL Op-Amp circuits. Memorize them!

Important Clarification

These Golden Rules apply ONLY when the Op-Amp is used with negative feedback (which is the normal operating mode for all linear applications). Without feedback, the Op-Amp operates in open-loop and the output just saturates to +VCC or −VEE.

Key Terms Explained

Open-Loop Gain (AOL)

The voltage gain of the Op-Amp without any feedback connection. It is extremely high (ideally infinite). Without feedback, even a tiny input difference causes the output to saturate at +VCC or −VEE.

Closed-Loop Gain (ACL)

The voltage gain when feedback is connected from output to input. This is much smaller than AOL and is controlled by external resistors. This is the useful, practical gain.

Negative Feedback

When a portion of the output signal is fed back to the inverting input (−) through a resistor. This stabilizes the gain, increases bandwidth, reduces distortion, and makes the circuit well-behaved.

CMRR (Common Mode Rejection Ratio)

The ratio of differential gain to common-mode gain. It measures the Op-Amp's ability to reject signals that appear equally on both inputs (like noise). Higher CMRR = better noise rejection. Expressed in dB.

CMRR = Ad (Differential Gain)Acm (Common Mode Gain)
Slew Rate

The maximum rate of change of output voltage per unit time. It indicates how fast the Op-Amp output can change. Measured in V/μs. For IC 741, slew rate is about 0.5 V/μs.

Slew Rate = ΔVoutΔt (V/μs)

IC 741C — The Most Popular Op-Amp 🔥⭐

About IC 741C
  • The μA741 (or simply "741") is the most widely used general-purpose Op-Amp IC
  • Originally developed by Fairchild Semiconductor
  • Comes in an 8-pin DIP (Dual In-line Package)

6.1 Virtual Ground & Offset Null Adjustment

Concept of Virtual Ground 🔥⭐

Definition: Virtual Ground

Virtual Ground is a point in a circuit that is maintained at ground potential (0V) but is NOT physically connected to the ground. In an Op-Amp inverting amplifier, the inverting input (−) acts as a virtual ground because it is forced to 0V by the negative feedback, even though it is not directly connected to the ground terminal.

How Virtual Ground Works — Step by Step

Consider an inverting amplifier circuit:

  • The non-inverting input (+) is connected to ground (0V)
  • By Golden Rule 2: V⁺ = V⁻
  • Since V⁺ = 0V (connected to ground), therefore V⁻ = 0V also
  • So the inverting input is at 0V — same as ground potential
  • BUT it is NOT actually connected to ground — it's connected to the input signal through Rin and to the output through Rf
  • This point at 0V (but not connected to ground) is called Virtual Ground

Fig. 6.4: Virtual Ground concept in an Op-Amp inverting amplifier — showing the inverting input at 0V potential without being connected to actual ground

🔍 Click to view on Google Images
Simple Analogy for Virtual Ground

Imagine two people standing on a see-saw (teeter-totter). The pivot point (fulcrum) stays at the same height even though the two ends go up and down. Similarly, the inverting input acts like a pivot point — it stays at 0V even though the input signal and output signal change. It's "virtually" at ground level without touching the ground.

⭐ Why is Virtual Ground Important?
  • It simplifies circuit analysis — we can assume V⁻ = 0V in inverting configurations
  • It is the basis for deriving gain formulas of inverting amplifier, adder, integrator, differentiator
  • It ensures that the input signal source sees a known impedance (Rin)
Common Exam Mistake

Virtual Ground ≠ Actual Ground!

  • Actual ground — physically connected to 0V reference, can sink current
  • Virtual ground — at 0V potential but cannot sink current (because no current flows into ideal Op-Amp input)

Current flowing through Rin does NOT go into ground — it flows through Rf to the output!

Concept of Offset Null Adjustment 🔥

Definition: Input Offset Voltage

The Input Offset Voltage (Vio) is the small DC voltage that must be applied between the two input terminals to make the output voltage exactly zero. In an ideal Op-Amp, Vio = 0, but in practical Op-Amps like IC 741, it is typically 1 to 5 mV.

Why Offset Voltage Exists
  • Inside the Op-Amp, the input stage consists of a differential amplifier using transistor pairs
  • Ideally, both transistors should be perfectly matched (identical)
  • In practice, there are slight manufacturing differences between the two transistors
  • This mismatch causes a small imbalance — even when both inputs are at 0V, the output shows a small non-zero voltage
  • This unwanted output voltage can be problematic in precision circuits
Definition: Offset Null Adjustment

Offset Null Adjustment is the process of nullifying (cancelling) the input offset voltage to ensure that the output is exactly zero when both inputs are at the same voltage. In IC 741, this is done by connecting a 10 kΩ potentiometer between pins 1 and 5, with its wiper connected to the negative supply (−VEE, pin 4).

How to Perform Offset Null Adjustment
  1. Connect a 10 kΩ potentiometer between Pin 1 and Pin 5 of IC 741
  2. Connect the wiper (center terminal) of the potentiometer to Pin 4 (−VEE)
  3. Short both inputs (Pin 2 and Pin 3) to ground (0V)
  4. Connect a voltmeter at the output (Pin 6)
  5. Adjust the potentiometer until the voltmeter reads exactly 0V
  6. Now the offset has been nullified — the Op-Amp output will be 0V when inputs are equal

Fig. 6.5: Offset Null Adjustment circuit for IC 741 — showing 10kΩ potentiometer connected between Pin 1 and Pin 5, wiper to Pin 4 (−VEE)

🔍 Click to view on Google Images
When is Offset Null Needed?
  • Precision DC amplifiers — where even small errors matter
  • Instrumentation amplifiers — measuring tiny sensor signals
  • Integrator circuits — offset causes output to drift over time
  • For AC amplifier applications, offset null is usually not critical because AC coupling capacitors block DC offset

6.2 Inverting Amplifier Mode

Inverting Amplifier — Circuit & Working 🔥⭐

Definition: Inverting Amplifier

An Inverting Amplifier is an Op-Amp circuit in which the input signal is applied to the inverting input (−) through an input resistor Rin, and the output is fed back to the inverting input through a feedback resistor Rf. The non-inverting input (+) is connected to ground. The output is an amplified and inverted (180° phase-shifted) version of the input.

Circuit Description
  • Rin (Input Resistor): Connected between input signal (Vin) and inverting input (Pin 2)
  • Rf (Feedback Resistor): Connected between output (Pin 6) and inverting input (Pin 2)
  • Non-inverting input (+): Connected directly to ground (0V)
  • Power supply: +VCC and −VEE connected to Pin 7 and Pin 4

Fig. 6.6: Inverting Amplifier circuit — showing Vin connected through Rin to inverting input, Rf feedback resistor, non-inverting input grounded, and output Vout

🔍 Click to view on Google Images

Gain Derivation of Inverting Amplifier 🔥⭐

Step-by-Step Derivation
Apply Golden Rule 2 — Virtual Ground:
Since V⁺ = 0V (connected to ground) and V⁺ = V⁻ (golden rule),
V⁻ = 0V (Virtual Ground)
Apply Golden Rule 1 — No current into Op-Amp:
Since no current flows into the inverting input terminal,
All current flowing through Rin must flow through Rf
Iin = If
Calculate Iin (current through Rin):
Current flows from Vin to virtual ground (0V) through Rin
Iin = Vin − 0Rin = VinRin
Calculate If (current through Rf):
Current flows from virtual ground (0V) to Vout through Rf
If = 0 − VoutRf = −VoutRf
Equate Iin = If:
VinRin = −VoutRf
Solve for Vout/Vin (Voltage Gain):
Vout = − RfRin × Vin
Voltage Gain of Inverting Amplifier
Av = VoutVin = − RfRin

Where:

  • Av = Voltage gain (dimensionless, or in V/V)
  • Rf = Feedback resistor (Ohms, Ω)
  • Rin = Input resistor (Ohms, Ω)
  • Negative sign (−) = Output is inverted (180° phase shift)
🔥 Key Points About Inverting Amplifier:
  1. The negative sign indicates phase inversion — if input is positive, output is negative, and vice versa
  2. Gain depends ONLY on external resistors (Rf and Rin) — NOT on Op-Amp's internal gain
  3. If Rf > Rin → |Gain| > 1 → Amplification
  4. If Rf < Rin → |Gain| < 1 → Attenuation
  5. If Rf = Rin → |Gain| = 1 → Unity gain inverter (output = −input)
  6. Input impedance of inverting amplifier = Rin (not infinite!)
Common Exam Mistake

Students often forget the negative sign in the inverting amplifier gain formula. Always include the "−" sign! It indicates the output is inverted. Without the negative sign, you'll lose marks.

Also remember: The magnitude of gain |Av| = Rf/Rin (without the negative sign) gives just the amplification factor.

Input & Output Waveforms

Signal Behaviour
  • If input is a sine wave, output is also a sine wave but inverted (shifted by 180°)
  • If input Vin = +1V and gain = −10, then Vout = −10V
  • If input Vin = −2V and gain = −5, then Vout = +10V
  • Output cannot exceed ±VCC (saturation) — typically about ±13V for ±15V supply

6.2 Non-Inverting Amplifier Mode

Non-Inverting Amplifier — Circuit & Working 🔥⭐

Definition: Non-Inverting Amplifier

A Non-Inverting Amplifier is an Op-Amp circuit in which the input signal is applied to the non-inverting input (+), and the feedback resistor Rf is connected from the output to the inverting input (−). R1 is connected from the inverting input to ground. The output is an amplified, non-inverted (same phase) version of the input.

Circuit Description
  • Vin (Input Signal): Applied directly to the non-inverting input (+) (Pin 3)
  • Rf (Feedback Resistor): Connected between output (Pin 6) and inverting input (Pin 2)
  • R1 (Ground Resistor): Connected between inverting input (Pin 2) and ground
  • This creates a voltage divider in the feedback path

Fig. 6.8: Non-Inverting Amplifier circuit — showing Vin at non-inverting input (+), Rf feedback, R1 to ground, and output Vout

🔍 Click to view on Google Images

Gain Derivation of Non-Inverting Amplifier 🔥⭐

Step-by-Step Derivation
Apply Golden Rule 2:
V⁺ = V⁻
Since Vin is applied to V⁺:
V⁻ = Vin
Voltage divider at inverting input:
R1 and Rf form a voltage divider between Vout and Ground.
The voltage at the inverting input (V⁻) is the voltage across R1:
V⁻ = Vout × R1R1 + Rf
Equate V⁻ from both equations:
Vin = Vout × R1R1 + Rf
Solve for Vout/Vin:
VoutVin = R1 + RfR1 = 1 + RfR1
Voltage Gain of Non-Inverting Amplifier
Av = VoutVin = 1 + RfR1

Where:

  • Av = Voltage gain (always positive and ≥ 1)
  • Rf = Feedback resistor (Ω)
  • R1 = Ground resistor (Ω)
  • No negative sign — output is in the same phase as input (0° phase shift)
🔥 Key Points About Non-Inverting Amplifier:
  1. No phase inversion — output is in same phase as input (0° phase shift)
  2. Gain is always ≥ 1 (minimum gain = 1, when Rf = 0 or R1 = ∞)
  3. Gain depends ONLY on Rf and R1 — external resistors
  4. Input impedance is very high (ideally infinite) — because input goes directly to the non-inverting terminal
  5. This is a major advantage over the inverting amplifier

Special Case: Voltage Follower (Buffer) 🔥

Definition: Voltage Follower

A Voltage Follower (also called Unity Gain Buffer) is a special case of the non-inverting amplifier where Rf = 0 (short circuit) and R1 = ∞ (open circuit). The output directly follows the input with gain = 1 (unity gain).

Voltage Follower Circuit
  • Input signal applied to non-inverting input (+)
  • Output is directly connected to inverting input (−) — 100% feedback
  • No Rf or R1 needed
  • Gain = 1 + Rf/R1 = 1 + 0 = 1
  • Vout = Vin (output exactly follows input)

Why use it if gain = 1?

  • Impedance matching — converts high impedance source to low impedance output
  • Isolation — prevents loading effect on the source
  • Acts as a buffer between stages
  • Input impedance: very high (∞ ideally) → Output impedance: very low (0 ideally)

6.3 Applications of Op-Amp

Applications Covered in Syllabus

The WBSCTE syllabus specifically requires four applications:

  1. Amplifier (Inverting & Non-Inverting — already covered above)
  2. Adder (Summing Amplifier)
  3. Integrator
  4. Differentiator

All four use the inverting configuration as the base circuit, with modifications to the input and feedback elements.

Application 1: Adder (Summing Amplifier) 🔥⭐

Definition: Summing Amplifier (Adder)

A Summing Amplifier (Adder) is an Op-Amp circuit that produces an output voltage which is proportional to the algebraic sum (addition) of two or more input voltages. It is based on the inverting amplifier with multiple input resistors connected to the inverting input.

Circuit Description
  • Multiple input signals (V1, V2, V3...) are applied through separate input resistors (R1, R2, R3...) to the inverting input
  • A feedback resistor Rf connects output to inverting input
  • Non-inverting input is connected to ground
  • Each input contributes independently to the output (due to virtual ground)

Fig. 6.10: Summing Amplifier (Adder) circuit — showing three input voltages V1, V2, V3 through R1, R2, R3 connected to the inverting input with Rf feedback

🔍 Click to view on Google Images

Derivation of Output Formula

Derivation
Virtual Ground: V⁻ = 0V (non-inverting input is grounded)
Current through each input resistor:
I1 = V1R1,   I2 = V2R2,   I3 = V3R3
Current through feedback resistor:
If = −VoutRf
Apply KCL at inverting node (no current into Op-Amp):
I1 + I2 + I3 = If
V1R1 + V2R2 + V3R3 = −VoutRf
Output of Summing Amplifier (General Formula)
Vout = −Rf × (V1R1 + V2R2 + V3R3)
⭐ Special Case — When R1 = R2 = R3 = Rf = R:
Vout = −(V1 + V2 + V3)

The output is simply the negative sum of all inputs. This is the ideal adder.

Weighted vs Non-Weighted Adder
  • Non-weighted adder: All input resistors are equal (R1 = R2 = R3 = Rf) → simple addition
  • Weighted adder: Input resistors have different values → each input is multiplied by a different weight (Rf/Rn)
  • Averaging circuit: If Rf = R/n (where n = number of inputs and R1 = R2 = ... = R), output gives the average of inputs
Applications of Summing Amplifier
  • Audio mixing: Combining multiple microphone/instrument signals
  • Digital-to-Analog Converter (DAC): Using weighted resistor method
  • Waveform generation: Adding sine waves of different frequencies
  • Voltage averaging: Finding average of multiple sensor readings

Application 2: Integrator Circuit 🔥⭐

Definition: Integrator

An Op-Amp Integrator is a circuit that produces an output voltage which is proportional to the time integral of the input voltage. It is formed by replacing the feedback resistor Rf of the inverting amplifier with a capacitor C.

Circuit Description
  • Input signal Vin applied through resistor R to inverting input
  • Capacitor C connected as feedback element (between output and inverting input) — replaces Rf
  • Non-inverting input connected to ground
  • The capacitor "accumulates" (integrates) the input signal over time

Fig. 6.11: Op-Amp Integrator circuit — showing input resistor R, feedback capacitor C, and output Vout

🔍 Click to view on Google Images

Output Formula Derivation

Derivation
Virtual Ground: V⁻ = 0V
Current through R:
Iin = VinR
Current through capacitor C:
For a capacitor: I = C × (dV/dt)
The voltage across C = 0 − Vout = −Vout
IC = C × d(−Vout)dt
Equate Iin = IC and integrate both sides:
This gives us the output as the time integral of the input.
Output of Integrator
Vout = − 1RC ∫ Vin dt

Where:

  • R = Input resistor (Ω)
  • C = Feedback capacitor (Farads)
  • RC = Time constant (seconds) — called the integration time constant
  • ∫Vindt = Time integral of input voltage
  • Negative sign = Output is inverted

What Does the Integrator Do to Common Signals?

Input Signal Output Signal Explanation
Square wave Triangular wave Integral of constant = ramp (linearly increasing/decreasing)
Sine wave Cosine wave (−cosine) Integral of sin = −cos
Triangular wave Parabolic wave Integral of ramp = parabola
Pulse (impulse) Step (constant) Integral of impulse = step
DC (constant) Ramp (linearly increasing) Integral of constant = linearly increasing voltage

Applications of Integrator
  • Waveform generation: Converting square wave to triangular wave
  • Analog computers: Solving differential equations
  • PID controllers: The "I" (integral) part of PID control
  • Ramp generators: Creating linearly increasing voltage
  • Low-pass filter: Acts as an active low-pass filter (passes low frequencies)
Practical Problem: DC Drift

An ideal integrator has a problem: any small DC offset at the input gets integrated continuously, causing the output to drift towards saturation over time. To solve this, a practical integrator adds a large resistor Rf in parallel with the capacitor C to provide a DC path and limit low-frequency gain.

Application 3: Differentiator Circuit 🔥⭐

Definition: Differentiator

An Op-Amp Differentiator is a circuit that produces an output voltage which is proportional to the rate of change (derivative) of the input voltage with respect to time. It is formed by replacing the input resistor R of the inverting amplifier with a capacitor C.

Circuit Description
  • Input signal Vin applied through a capacitor C to the inverting input
  • Feedback resistor R connected between output and inverting input
  • Non-inverting input connected to ground
  • The capacitor "responds to changes" (differentiates) in the input signal

Notice: Integrator has C in feedback (replacing Rf). Differentiator has C at input (replacing Rin). They are mirror images of each other!

Fig. 6.14: Op-Amp Differentiator circuit — showing input capacitor C, feedback resistor R, and output Vout

Output Formula Derivation

Derivation
Virtual Ground: V⁻ = 0V
Current through capacitor C (input side):
Voltage across C = Vin − 0 = Vin
IC = C × dVindt
Current through R (feedback):
IR = 0 − VoutR = −VoutR
Equate IC = IR:
C × dVindt = −VoutR
Output of Differentiator
Vout = −RC × dVindt

Where:

  • R = Feedback resistor (Ω)
  • C = Input capacitor (Farads)
  • RC = Differentiation time constant (seconds)
  • dVin/dt = Rate of change of input voltage with time
  • Negative sign = Output is inverted

What Does the Differentiator Do to Common Signals?

Input Signal Output Signal Explanation
Triangular wave Square wave Derivative of ramp = constant (positive or negative)
Sine wave Cosine wave Derivative of sin = cos

Example 6.1 — Inverting Amplifier

Design an inverting amplifier with a gain of −10 using an Op-Amp. If Rin = 10 kΩ, find Rf. Also find Vout when Vin = 0.5 V.

Solution
Given: Gain Av = −10, Rin = 10 kΩ
Find: Rf = ?, Vout when Vin = 0.5 V
Using gain formula for inverting amplifier:
Av = − RfRin
−10 = − Rf10 kΩ
Solving for Rf:
Rf = 10 × 10 kΩ = 100 kΩ
Finding Vout:
Vout = Av × Vin = (−10) × 0.5
Rf = 100 kΩ, Vout = −5 V

The negative sign means the output is inverted with respect to the input.

Example 6.2 — Non-Inverting Amplifier

A non-inverting amplifier has Rf = 47 kΩ and R1 = 10 kΩ. Calculate the voltage gain. If Vin = 0.2 V, find Vout.

Solution
Given: Rf = 47 kΩ, R1 = 10 kΩ, Vin = 0.2 V
Using gain formula for non-inverting amplifier:
Av = 1 + RfR1 = 1 + 4710 = 1 + 4.7 = 5.7
Finding Vout:
Vout = Av × Vin = 5.7 × 0.2 = 1.14 V
Av = 5.7, Vout = +1.14 V

Output is positive (same phase as input) — no inversion.

Example 6.3 — Summing Amplifier

A summing amplifier has three inputs: V1 = 1V, V2 = 2V, V3 = 3V with R1 = R2 = R3 = 10 kΩ and Rf = 10 kΩ. Find Vout.

Solution
Given: V1 = 1V, V2 = 2V, V3 = 3V
R1 = R2 = R3 = Rf = 10 kΩ
Using summing amplifier formula:
Vout = −Rf × (V1R1 + V2R2 + V3R3)
Since all resistors are equal (Rf = R1 = R2 = R3 = R):
Vout = −(V1 + V2 + V3) = −(1 + 2 + 3)
Vout = −6 V
Example 6.4 — Weighted Summing Amplifier

A summing amplifier has V1 = 2V, V2 = 3V with R1 = 10 kΩ, R2 = 20 kΩ, and Rf = 40 kΩ. Find Vout.

Solution
Given: V1 = 2V, V2 = 3V, R1 = 10 kΩ, R2 = 20 kΩ, Rf = 40 kΩ
Applying the general formula:
Vout = −Rf × (V1R1 + V2R2)
Vout = −40k × (210k + 320k)
= −40k × (0.0002 + 0.00015)
= −40k × 0.00035
= −40000 × 0.00035
Vout = −14 V

Note: Check if this is within the supply voltage limits! If supply is ±15V, output will saturate at about ±13V. So in practice, Vout ≈ −13V (saturated).

Example 6.5 — Design Problem

Design a non-inverting amplifier with a gain of 11 using IC 741. R1 = 1 kΩ. Find Rf. If Vin = 1V (peak), find Vout and check if the output saturates with ±15V supply.

Solution

Finding Rf:
Av = 1 + RfR1
11 = 1 + Rf1k
Rf = (11 − 1) × 1k = 10 × 1k = 10 kΩ
Finding Vout:
Vout = 11 × 1 = 11 V (peak)
Saturation check:
For IC 741 with ±15V supply, maximum output ≈ ±13V (typically about 2V less than supply).
Vout = 11V < 13V → No saturation. Output is within limits. ✅
Rf = 10 kΩ, Vout = +11V (not saturated)

Important Definitions

1. Operational Amplifier (Op-Amp)

A high-gain, direct-coupled, differential amplifier that amplifies the difference between two input voltages. It can perform mathematical operations like addition, subtraction, integration, and differentiation. Available as an IC (e.g., IC 741).

2. Open-Loop Gain (AOL)

The voltage gain of the Op-Amp without any external feedback. For an ideal Op-Amp, AOL = ∞. For IC 741, AOL ≈ 200,000 (2 × 10⁵).

3. Closed-Loop Gain (ACL)

The voltage gain when negative feedback is applied from output to inverting input. It is determined by external resistors and is much smaller than the open-loop gain.

4. Virtual Ground

A point in an Op-Amp circuit that is maintained at ground potential (0V) without being physically connected to ground. It appears at the inverting input of an inverting amplifier because V⁺ = V⁻ = 0V.

5. Negative Feedback

When a portion of the output is fed back to the inverting input (−) of the Op-Amp. This stabilizes gain, increases bandwidth, reduces distortion, and lowers output impedance.

6. Input Offset Voltage (Vio)

The small DC voltage that must be applied between the two input terminals to make the output exactly zero. Ideally zero; for IC 741, it is about 1–5 mV.

7. Offset Null Adjustment

The process of cancelling the input offset voltage using a potentiometer connected between pins 1 and 5 of IC 741, with the wiper connected to −VEE (pin 4).

8. CMRR (Common Mode Rejection Ratio)

The ratio of differential gain to common-mode gain. Measures the Op-Amp's ability to reject noise that appears equally at both inputs. Ideally infinite; for IC 741, about 90 dB.

9. Slew Rate

The maximum rate of change of output voltage with time, measured in V/μs. It limits how fast the Op-Amp output can respond to rapidly changing inputs. For IC 741, slew rate ≈ 0.5 V/μs.

10. Inverting Amplifier

An Op-Amp circuit where input is applied to the inverting input through Rin, feedback resistor Rf connects output to inverting input, and non-inverting input is grounded. Gain = −Rf/Rin. Output is inverted (180° phase shift).

11. Non-Inverting Amplifier

An Op-Amp circuit where input is applied directly to the non-inverting input, Rf connects output to inverting input, and R1 connects inverting input to ground. Gain = 1 + Rf/R1. Output is in same phase (0° phase shift).

12. Voltage Follower (Buffer)

A special non-inverting amplifier where the output is directly connected to the inverting input (100% feedback). Gain = 1. Used for impedance matching — high input impedance, low output impedance.

13. Summing Amplifier (Adder)

An inverting amplifier circuit with multiple input resistors. The output is the weighted sum of all input voltages, inverted. Vout = −Rf(V1/R1 + V2/R2 + ...).

14. Integrator

An inverting amplifier circuit where the feedback resistor is replaced by a capacitor. Output is proportional to the time integral of the input. Vout = −(1/RC) ∫ Vin dt.

15. Differentiator

An inverting amplifier circuit where the input resistor is replaced by a capacitor. Output is proportional to the rate of change (derivative) of the input. Vout = −RC (dVin/dt).

Long Answer Questions (5–7 Marks Style)

Q 6.L1

Explain the features of an ideal Op-Amp. Compare ideal and practical Op-Amp.

An ideal Op-Amp is a theoretical perfect amplifier with the following characteristics:

  1. Infinite open-loop gain — even tiny input difference gives large output
  2. Infinite input impedance — no input current flows
  3. Zero output impedance — no voltage drop at output
  4. Infinite bandwidth — all frequencies amplified equally
  5. Infinite CMRR — common-mode signals completely rejected
  6. Infinite slew rate — output changes instantly
  7. Zero offset voltage — output is zero when both inputs are equal

Practical Op-Amp (IC 741) has finite values:

  • Open-loop gain ≈ 2 × 10⁵
  • Input impedance ≈ 2 MΩ
  • Output impedance ≈ 75 Ω
  • Bandwidth ≈ 1 MHz
  • CMRR ≈ 90 dB
  • Slew rate ≈ 0.5 V/μs
  • Offset voltage ≈ 1–5 mV

Conclusion: Ideal Op-Amp simplifies analysis, while practical Op-Amps like IC 741 approximate these ideal properties and are used in real circuits.

Q 6.L3

Draw the circuit of an inverting amplifier and derive its voltage gain.

Circuit: Input Vin is applied through Rin to the inverting input. Feedback resistor Rf is connected between output and inverting input. Non-inverting input is grounded.

Derivation:

  1. Since non-inverting input is at 0V, by virtual ground concept V⁻ = 0V
  2. No current enters the Op-Amp input, so current through Rin equals current through Rf
  3. Iin = Vin/Rin
  4. If = −Vout/Rf
  5. Equating: Vin/Rin = −Vout/Rf
  6. Therefore: Vout = −(Rf/Rin) Vin

Voltage Gain:

Av = Vout/Vin = −Rf/Rin

The negative sign indicates 180° phase inversion.

Q 6.L4

Draw the circuit of a non-inverting amplifier and derive its voltage gain.

Circuit: Input Vin is applied directly to the non-inverting input. Rf connects output to inverting input, and R1 connects inverting input to ground.

Derivation:

  1. By golden rule, V⁺ = V⁻
  2. Since Vin is applied to V⁺, therefore V⁻ = Vin
  3. R1 and Rf form a voltage divider between Vout and ground
  4. So V⁻ = Vout × [R1/(R1 + Rf)]
  5. Equating: Vin = Vout × [R1/(R1 + Rf)]
  6. Therefore: Vout/Vin = 1 + Rf/R1

Voltage Gain:

Av = 1 + Rf/R1

The output is in the same phase as input.

Q 6.L5

Explain the concept of virtual ground and offset null adjustment.

Virtual Ground:

  • In an inverting amplifier, the non-inverting input is connected to ground (0V)
  • By Op-Amp golden rule, V⁺ = V⁻
  • Therefore the inverting input is also at 0V
  • But it is not physically connected to ground — so it is called virtual ground
  • This concept simplifies analysis of inverting amplifier, adder, integrator and differentiator

Offset Null Adjustment:

  • Practical Op-Amps have a small input offset voltage due to transistor mismatch
  • Because of this, output may not be zero even when both inputs are zero
  • IC 741 provides pins 1 and 5 for offset null adjustment
  • A 10 kΩ potentiometer is connected between pins 1 and 5, with the wiper to pin 4 (−VEE)
  • Adjust the potentiometer until output becomes zero

Conclusion: Virtual ground helps in analysis, while offset null improves accuracy of practical Op-Amp circuits.

Q 6.L6

Explain the Op-Amp summing amplifier with circuit and equation.

A summing amplifier is an Op-Amp circuit that adds two or more input voltages and gives their algebraic sum at the output.

Circuit:

  • Multiple inputs V1, V2, V3 are applied through resistors R1, R2, R3 to the inverting input
  • Feedback resistor Rf is connected from output to inverting input
  • Non-inverting input is grounded

Using KCL at virtual ground:

Vout = −Rf(V1/R1 + V2/R2 + V3/R3)

Special case: If all resistors are equal,

Vout = −(V1 + V2 + V3)

Applications: audio mixing, DAC, averaging circuit, waveform summation.

Q 6.L7

Explain the Op-Amp integrator circuit with working and applications.

An integrator is an Op-Amp circuit in which the feedback resistor of the inverting amplifier is replaced by a capacitor.

Circuit:

  • Input applied through resistor R to inverting input
  • Capacitor C used in feedback path
  • Non-inverting input grounded

Working:

  • The current through input resistor charges/discharges the feedback capacitor
  • The output is proportional to the time integral of input voltage

Output equation:

Vout = −(1/RC) ∫ Vin dt

Waveform relation:

  • Square wave input → Triangular wave output
  • DC input → Ramp output

Applications: waveform generation, analog computers, PID controller, low-pass filter, ramp generator.

Q 6.L8

Explain the Op-Amp differentiator circuit with working and applications.

A differentiator is an Op-Amp circuit in which the input resistor of the inverting amplifier is replaced by a capacitor.

Circuit:

  • Input applied through capacitor C to inverting input
  • Feedback resistor R connected from output to inverting input
  • Non-inverting input grounded

Working:

  • The capacitor passes only changing signals
  • The output is proportional to the rate of change of input voltage

Output equation:

Vout = −RC (dVin/dt)

Waveform relation:

  • Triangular input → Square output
  • Square input → Spikes at transitions

Applications: edge detection, waveform shaping, high-pass filter, PID controller, rate measurement.
