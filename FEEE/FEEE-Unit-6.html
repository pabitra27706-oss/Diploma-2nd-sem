Features of an Ideal Operational Amplifier

Ideal Op-Amp Characteristics
  • Infinite open-loop gain (AOL = ∞)
  • Infinite input impedance (Zin = ∞) → no input current
  • Zero output impedance (Zout = 0) → can drive any load
  • Infinite bandwidth → amplifies all frequencies equally
  • Common-mode rejection ratio (CMRR) = ∞
  • Slew rate = ∞ (instantaneous output change)
  • Zero offset voltage (Vout = 0 when Vin = 0)
Virtual Ground Concept

In negative feedback configuration, the differential input voltage (V+ - V-) is ideally zero. Since the non-inverting terminal is often grounded, the inverting terminal is at virtual ground (0V but no current flow).

V1 ≈ V2 (virtual short)

741C Op-Amp: Pin Configuration & Offset Null

1 Offset Null 8 NC
2 Inverting (-) 7 VCC+
3 Non-inverting (+) 6 Output
4 VEE- 5 Offset Null
Pin Functions
  • Pin 1 & 5 (Offset Null): Used to nullify output offset voltage by connecting a potentiometer (typically 10kΩ).
  • Pin 2 (Inverting Input): Signal applied here gets inverted at output.
  • Pin 3 (Non-inverting Input): Signal applied here appears same polarity at output.
  • Pin 4 (VEE): Negative supply (usually -15V).
  • Pin 6 (Output): Amplified output.
  • Pin 7 (VCC): Positive supply (+15V).
  • Pin 8 (NC): No connection.
Offset Null Adjustment

A 10kΩ potentiometer is connected between pins 1 and 5, with wiper connected to VEE (pin 4). This balances the internal circuitry to make output zero when inputs are zero.

Inverting & Non-Inverting Modes · Gain Calculation

Inverting Amplifier

Input applied to inverting terminal via R1. Non-inverting grounded.

Av = -Rf / R1

Negative sign indicates phase inversion.

Non-Inverting Amplifier

Input applied to non-inverting terminal. Feedback from output to inverting via Rf and R1 to ground.

Av = 1 + (Rf / R1)

Output in phase with input.

Example 1: Inverting Amplifier

R1 = 1kΩ, Rf = 10kΩ. Find gain and output for Vin = 0.5V.

Gain Av = -Rf/R1 = -10k/1k = -10
Vout = Av × Vin = -10 × 0.5 = -5V
Example 2: Non-Inverting Amplifier

R1 = 2kΩ, Rf = 8kΩ. Find gain and output for Vin = 1V.

Gain Av = 1 + (Rf/R1) = 1 + (8k/2k) = 1 + 4 = 5
Vout = Av × Vin = 5 × 1 = 5V

Op-Amp Applications: Amplifier · Adder · Integrator · Differentiator

Adder (Summing Amplifier)

Inverting configuration with multiple inputs.

Vout = -Rf (V1/R1 + V2/R2 + ...)

Integrator

Replaces feedback resistor with capacitor.

Vout = - (1/R1C) ∫ Vin dt

Differentiator

Input resistor replaced with capacitor.

Vout = -RfC dVin/dt

Adder Circuit Example

Three inputs: V1=1V, V2=2V, V3=3V. All input resistors = 10kΩ, Rf = 10kΩ. Find output.

Vout = -Rf (V1/R1 + V2/R2 + V3/R3)
= -10k (1/10k + 2/10k + 3/10k) = -10k × (6/10k) = -6V
Vout = -6V

Integrator Circuit Example

R = 10kΩ, C = 0.1μF. Input is a step voltage of 2V. Find output after 1ms.

Vout = - (1/RC) ∫ Vin dt = - (1/(10k × 0.1×10⁻⁶)) × (2V × 1ms)
RC = 10⁴ × 10⁻⁷ = 10⁻³ = 1ms
1/RC = 1000
Vout = -1000 × (2 × 0.001) = -1000 × 0.002 = -2V

Differentiator Circuit Example

C = 0.1μF, Rf = 10kΩ. Input is a ramp voltage increasing at 1000 V/s. Find output.

Vout = -RfC dVin/dt
= -10k × 0.1×10⁻⁶ × 1000
= -10⁴ × 10⁻⁷ × 10³ = -10⁴ × 10⁻⁴ = -1V
Amplifier Circuit

Both inverting and non-inverting configurations are used as amplifier circuits. The gain is set by resistor ratio as shown in section 3.
