Boolean Algebra: Rules and Laws

Basic Laws
  • Identity: A + 0 = A, A · 1 = A
  • Null: A + 1 = 1, A · 0 = 0
  • Idempotent: A + A = A, A · A = A
  • Complement: A + A' = 1, A · A' = 0
  • Involution: (A')' = A
  • Commutative: A + B = B + A, A·B = B·A
  • Associative: (A+B)+C = A+(B+C), (A·B)·C = A·(B·C)
  • Distributive: A·(B+C) = A·B + A·C, A + (B·C) = (A+B)·(A+C)
  • Absorption: A + A·B = A, A·(A+B) = A

Universal Logic Gates

NAND and NOR are universal gates. Any Boolean function can be implemented using only NAND or only NOR gates.

De-Morgan's Theorem · SOP · POS

De-Morgan's Theorem

(A·B)' = A' + B'

(A+B)' = A' · B'

Min term (SOP)

Sum of Products: OR of AND terms. Each AND term contains all variables (in true or complemented form).

Example: F = A'B + AB'

Max term (POS)

Product of Sums: AND of OR terms. Each OR term contains all variables.

Example: F = (A+B)·(A'+B')

Boolean Simplification Example

Simplify: F = A'B'C + A'BC + AB'C + ABC

F = A'C(B' + B) + AC(B' + B) = A'C + AC = C(A' + A) = C

Sequential Circuits: Flip Flops as Storage Elements

Flip Flop Basics

Flip flop is a bistable multivibrator that stores 1 bit of data (0 or 1). It has two stable states.

Types: SR, JK, D, T flip flops.

Flip flops are edge-triggered (positive or negative) and form the basis of counters and registers.

Counters: Ripple · Synchronous Up-Down

Ripple Counter (Asynchronous)

Flip flops are connected in series. Clock applied to first FF only. Output of one FF clocks the next.

Simple but slow (propagation delay accumulates).

Synchronous Counter

All flip flops receive same clock pulse simultaneously. Faster, can be designed for up/down counting.

Up-Down Counter

A counter that can count both forward and backward depending on control signal. Uses additional logic (e.g., XOR gates) to select counting direction.

Digital IC Logic Gates: TTL

TTL (Transistor-Transistor Logic)

TTL is a family of digital logic ICs using bipolar transistors. Standard TTL series: 7400.

  • Typical supply voltage: 5V
  • Logic 0: 0-0.8V, Logic 1: 2-5V
  • Examples: 7400 (quad NAND), 7402 (quad NOR), 7404 (hex inverter), 7408 (quad AND), 7432 (quad OR)
  • Propagation delay ~10ns, power consumption ~10mW per gate
