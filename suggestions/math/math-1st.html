📐 Key Formulas
Determinant 2×2
|a b; c d| = ad − bc
Cofactor
Cᵢⱼ = (−1)ⁱ⁺ʲ · Mᵢⱼ
det(kA) for n×n
det(kA) = kⁿ · det(A)
Inverse (adjoint method)
A⁻¹ = (1/det(A)) · adj(A)
adj(A) = transpose of cofactor matrix
Orthogonal Matrix
AAᵀ = I ⇒ A⁻¹ = Aᵀ
Cramer's Rule
x = det(Aₓ)/det(A), y = det(Aᵧ)/det(A),
z = det(A_z)/det(A)
A(adj A)
A · adj(A) = det(A) · I
Singular Matrix
Matrix is singular ⇔ det(A) = 0

📐 Key Formulas
Distance
d = √[(x₂−x₁)² + (y₂−y₁)²]
Area of Triangle / Collinear
Area = ½|x₁(y₂−y₃)+x₂(y₃−y₁)+x₃(y₁−y₂)|
Collinear ⇔ Area = 0
Slope
m = (y₂−y₁)/(x₂−x₁)
Line: Point‑Slope
y − y₁ = m(x − x₁)
Line: Intercept
x/a + y/b = 1
Angle Between Lines
tan θ = |(m₁−m₂)/(1+m₁m₂)|
∥ : m₁=m₂, ⊥ : m₁m₂=−1
Distance Point→Line
|Ax₁+By₁+C| / √(A²+B²)
Circle (Centre‑Radius)
(x−h)²+(y−k)²=r²
General: x²+y²+2gx+2fy+c=0
Centre (−g,−f), r=√(g²+f²−c)
Parabola y²=4ax
Focus (a,0), Directrix x=−a,
Latus Rectum = 4a
Ellipse x²/a²+y²/b²=1
c²=a²−b², e=c/a, foci (±c,0),
Latus Rectum = 2b²/a
Polar ⇔ Cartesian
x = r cos θ, y = r sin θ
r = √(x²+y²), θ = tan⁻¹(y/x)

20212022202320242025 ★★★★★
📐 Key Formulas
Separable ODE
dy/dx = f(x)·g(y)
⇒ ∫dy/g(y) = ∫f(x)dx
Homogeneous ODE
dy/dx = f(y/x)
Put y = vx ⇒ dy/dx = v + x·dv/dx
Exact ODE
Mdx+Ndy=0 is exact if ∂M/∂y = ∂N/∂x
Linear ODE + IF
dy/dx + Py = Q
IF = e^{∫P dx}
y·IF = ∫Q·IF dx + c
Bernoulli Equation
dy/dx + Py = Qyⁿ
Substitute z = y¹⁻ⁿ to make linear
2nd Order CF (Roots m₁,m₂)
Real≠ : c₁e^{m₁x}+c₂e^{m₂x}
Real= : (c₁+c₂x)e^{mx}
Complex α±iβ : e^{αx}(c₁cos βx+c₂sin βx)
PI for eᵃˣ
PI = eᵃˣ / F(D), D→a (if F(a)≠0)
If F(a)=0, multiply by x
PI for sin/cos ax
PI = sin ax / F(D²), D²→−a²
PI for xⁿ
Expand 1/F(D) as series in D

💡 Exam Tips & Strategy

  • 2024–25 Pattern: MCQs require a written explanation for the chosen answer — 1 mark for the correct option + 1 mark for correct reasoning. Never skip the explanation.
  • Time Management: In the 1‑hour paper (2024), allocate ~20 min for 10 MCQs and ~35 min for 5 long questions. Practise speed.
  • Easiest 35+ Marks Path: Focus on Units 1, 2, 5, and 6. They are formula‑driven and require less practice than Integration (Unit 3) or Differential Equations (Unit 4).
  • Cramer's Rule & Inverse: These appear almost every year. Memorise the steps — they are easy 5–8 marks.
  • Euler's Theorem Proofs: The identity questions in Unit 5 follow a standard pattern. Practise 3–4 examples and you can solve any variant.
  • Median/Mode/Mean: Always write the formula first, then substitute values. Many marks are awarded for correct formula even if the arithmetic slips.
  • Probability: Draw a sample space or tree diagram for coin/dice/urn problems. For Bayes' theorem, identify P(A), P(B|A), P(B|Ā) clearly before plugging into the formula.
  • Show All Steps: In long‑answer questions, stepwise working fetches partial marks. Never skip directly to the final answer.
  • Determinant Properties: For "prove without expanding" questions, use row/column operations — look for common factors, row subtractions, or cyclic symmetry.
  • Revise Past Papers: Over 70% of questions are direct repeats or minor variations from 2021–2025. Practise all starred questions above.
