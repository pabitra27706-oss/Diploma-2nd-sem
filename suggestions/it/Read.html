📐 Quick Formula & Facts Card – Easy Units

🔢 Binary ⇄ Decimal
(1101)₂ = 1×8 + 1×4 + 0×2 + 1×1 = 13
Decimal to binary: divide by 2, read remainders upward.
🌈 Gray code (binary to gray)
MSB stays same; next bits = XOR(bitᵢ, bitᵢ₋₁)
Example: 1011 → 1110
➖ 2’s Complement subtraction
A − B = A + (2’s complement of B)
2’s complement = invert bits + 1
🧩 De Morgan’s Theorems
(A+B)’ = A’·B’   (A·B)’ = A’+B’
⚙️ Universal gate (NOR as AND)
A·B = (A NOR A) NOR (B NOR B)
🌡️ Celsius → Fahrenheit
F = (C × 9/5) + 32
📊 Flowchart decision
Rhombus → condition (Yes/No)
Parallelogram → Input/Output
🛡️ Malware definitions
Virus: self-replicates, attaches to files.
Worm: network spread, no host file needed.
⚠️ 2024–2025 PATTERN ALERT: In Group-A, each MCQ carries 2 marks → 1 mark for correct option + 1 mark for valid explanation. No marks if explanation is missing or wrong. So understand the “why” behind conversions, De Morgan, OS scheduling, Gray code, etc.
