Unit Overview

What This Unit Covers

In ordinary differentiation (Unit 4), we dealt with functions of one variable like y = f(x). But in real engineering problems, quantities often depend on more than one variable.

Partial Differentiation is the technique of differentiating functions of two or more independent variables.

🌡️ Real-World Example:

Consider the temperature T inside a room. It depends on:

  • x — horizontal position
  • y — vertical position (height)
  • t — time

So T = f(x, y, t) is a function of THREE variables. To study how temperature changes when only ONE variable changes (keeping others constant), we use partial derivatives.

5.1 Definition & Meaning of Partial Derivative 🔥

Definition — Function of Two Variables

A function of two independent variables x and y is written as:

\[ z = f(x, y) \]

where z is the dependent variable and x, y are independent variables.

Examples:

  • \(z = x^2 + y^2\)
  • \(z = x^3 + 3x^2y - 2y^3\)
  • \(z = \sin(x + y)\)
  • \(z = e^{x+y}\)
  • \(u = x^2 + y^2 + z^2\) (function of three variables)
Definition — Partial Derivative 🔥

Let z = f(x, y) be a function of two independent variables x and y.

Partial Derivative with respect to x:

The partial derivative of z with respect to x is the derivative of z treating y as a constant.

\[ \frac{\partial z}{\partial x} = \lim_{\Delta x \to 0} \frac{f(x + \Delta x, y) - f(x, y)}{\Delta x} \]

Partial Derivative with respect to y:

The partial derivative of z with respect to y is the derivative of z treating x as a constant.

\[ \frac{\partial z}{\partial y} = \lim_{\Delta y \to 0} \frac{f(x, y + \Delta y) - f(x, y)}{\Delta y} \]
Notations for Partial Derivatives

There are several equivalent notations used:

Notation 1 Notation 2 Notation 3 Notation 4 Meaning
\(\frac{\partial z}{\partial x}\) \(\frac{\partial f}{\partial x}\) \(f_x\) \(z_x\) Partial derivative w.r.t. x
\(\frac{\partial z}{\partial y}\) \(\frac{\partial f}{\partial y}\) \(f_y\) \(z_y\) Partial derivative w.r.t. y

Read as: "partial z by partial x" or "del z by del x" or "z sub x"

Simple Rule to Remember

When finding ∂z/∂x:

  • ✅ Differentiate with respect to x (as usual)
  • ✅ Treat y as a constant (like the number 5 or π)

When finding ∂z/∂y:

  • ✅ Differentiate with respect to y (as usual)
  • ✅ Treat x as a constant

Geometric Interpretation

If z = f(x, y) represents a surface in 3D space:

  • \(\frac{\partial z}{\partial x}\) = Slope of the surface in the x-direction (keeping y constant)
  • \(\frac{\partial z}{\partial y}\) = Slope of the surface in the y-direction (keeping x constant)

Analogy: Imagine walking on a hilly terrain. ∂z/∂x tells you how steep the hill is if you walk east, while ∂z/∂y tells you how steep it is if you walk north.

5.2 Evaluation of Partial Derivatives 🔥🔥

Steps to Find Partial Derivatives

Identify which variable you're differentiating with respect to.

Treat ALL OTHER variables as constants.

Apply standard differentiation rules (power rule, chain rule, product rule, etc.).

Simplify the result if needed.

Example 5.1 — Basic Partial Derivatives ⭐

If \(z = x^3 + 3x^2y + y^3\), find \(\frac{\partial z}{\partial x}\) and \(\frac{\partial z}{\partial y}\).

Solution

Finding ∂z/∂x: (Treat y as constant)

\(\frac{\partial z}{\partial x} = \frac{\partial}{\partial x}(x^3) + \frac{\partial}{\partial x}(3x^2y) + \frac{\partial}{\partial x}(y^3)\)

\(= 3x^2 + 3y \cdot 2x + 0\) (y³ is constant w.r.t. x)

\[ \boxed{\frac{\partial z}{\partial x} = 3x^2 + 6xy} \]

Finding ∂z/∂y: (Treat x as constant)

\(\frac{\partial z}{\partial y} = \frac{\partial}{\partial y}(x^3) + \frac{\partial}{\partial y}(3x^2y) + \frac{\partial}{\partial y}(y^3)\)

\(= 0 + 3x^2 \cdot 1 + 3y^2\) (x³ is constant w.r.t. y)

\[ \boxed{\frac{\partial z}{\partial y} = 3x^2 + 3y^2} \]
Example 5.2 — With Product Terms ⭐⭐

If \(z = x^2y^3 + 2x^3y^2 - 5xy\), find \(\frac{\partial z}{\partial x}\) and \(\frac{\partial z}{\partial y}\).

Solution

Finding ∂z/∂x: (y is constant)

\(\frac{\partial z}{\partial x} = y^3 \cdot 2x + 2y^2 \cdot 3x^2 - 5y \cdot 1\)

\[ \boxed{\frac{\partial z}{\partial x} = 2xy^3 + 6x^2y^2 - 5y} \]

Finding ∂z/∂y: (x is constant)

\(\frac{\partial z}{\partial y} = x^2 \cdot 3y^2 + 2x^3 \cdot 2y - 5x \cdot 1\)

\[ \boxed{\frac{\partial z}{\partial y} = 3x^2y^2 + 4x^3y - 5x} \]
Example 5.3 — Exponential Function ⭐⭐

If \(z = e^{x^2 + y^2}\), find \(\frac{\partial z}{\partial x}\) and \(\frac{\partial z}{\partial y}\).

Solution

Using chain rule: \(\frac{d}{dx}(e^u) = e^u \cdot \frac{du}{dx}\)

Finding ∂z/∂x:

Let \(u = x^2 + y^2\), then \(\frac{\partial u}{\partial x} = 2x\) (y² is constant)

\(\frac{\partial z}{\partial x} = e^{x^2 + y^2} \cdot 2x\)

\[ \boxed{\frac{\partial z}{\partial x} = 2xe^{x^2 + y^2}} \]

Finding ∂z/∂y:

\(\frac{\partial u}{\partial y} = 2y\) (x² is constant)

\(\frac{\partial z}{\partial y} = e^{x^2 + y^2} \cdot 2y\)

\[ \boxed{\frac{\partial z}{\partial y} = 2ye^{x^2 + y^2}} \]
Example 5.4 — Trigonometric Function ⭐⭐

If \(z = \sin(x + 2y)\), find \(\frac{\partial z}{\partial x}\) and \(\frac{\partial z}{\partial y}\).

Solution

Using chain rule: \(\frac{d}{dx}(\sin u) = \cos u \cdot \frac{du}{dx}\)

Finding ∂z/∂x:

Let \(u = x + 2y\), then \(\frac{\partial u}{\partial x} = 1\)

\[ \boxed{\frac{\partial z}{\partial x} = \cos(x + 2y)} \]

Finding ∂z/∂y:

\(\frac{\partial u}{\partial y} = 2\)

\[ \boxed{\frac{\partial z}{\partial y} = 2\cos(x + 2y)} \]
Example 5.5 — Logarithmic Function ⭐⭐

If \(z = \ln(x^2 + y^2)\), find \(\frac{\partial z}{\partial x}\) and \(\frac{\partial z}{\partial y}\).

Solution

Using chain rule: \(\frac{d}{dx}(\ln u) = \frac{1}{u} \cdot \frac{du}{dx}\)

Finding ∂z/∂x:

\(\frac{\partial z}{\partial x} = \frac{1}{x^2 + y^2} \cdot 2x\)

\[ \boxed{\frac{\partial z}{\partial x} = \frac{2x}{x^2 + y^2}} \]

Finding ∂z/∂y:

\(\frac{\partial z}{\partial y} = \frac{1}{x^2 + y^2} \cdot 2y\)

\[ \boxed{\frac{\partial z}{\partial y} = \frac{2y}{x^2 + y^2}} \]
Example 5.6 — Quotient / Fraction ⭐⭐

If \(u = \frac{x + y}{x - y}\), find \(\frac{\partial u}{\partial x}\) and \(\frac{\partial u}{\partial y}\).

Solution

Using quotient rule: \(\frac{d}{dx}\left(\frac{f}{g}\right) = \frac{g \cdot f' - f \cdot g'}{g^2}\)

Finding ∂u/∂x: (y is constant)

Let f = x + y, g = x - y

\(\frac{\partial f}{\partial x} = 1\), \(\frac{\partial g}{\partial x} = 1\)

\(\frac{\partial u}{\partial x} = \frac{(x-y)(1) - (x+y)(1)}{(x-y)^2} = \frac{x - y - x - y}{(x-y)^2} = \frac{-2y}{(x-y)^2}\)

\[ \boxed{\frac{\partial u}{\partial x} = \frac{-2y}{(x-y)^2}} \]

Finding ∂u/∂y: (x is constant)

\(\frac{\partial f}{\partial y} = 1\), \(\frac{\partial g}{\partial y} = -1\)

\(\frac{\partial u}{\partial y} = \frac{(x-y)(1) - (x+y)(-1)}{(x-y)^2} = \frac{x - y + x + y}{(x-y)^2} = \frac{2x}{(x-y)^2}\)

\[ \boxed{\frac{\partial u}{\partial y} = \frac{2x}{(x-y)^2}} \]
Example 5.7 — Mixed Exponential ⭐⭐

If \(z = e^{xy}\), find \(\frac{\partial z}{\partial x}\) and \(\frac{\partial z}{\partial y}\).

Solution

Finding ∂z/∂x: (y is constant)

\(\frac{\partial z}{\partial x} = e^{xy} \cdot \frac{\partial}{\partial x}(xy) = e^{xy} \cdot y\)

\[ \boxed{\frac{\partial z}{\partial x} = ye^{xy}} \]

Finding ∂z/∂y: (x is constant)

\(\frac{\partial z}{\partial y} = e^{xy} \cdot \frac{\partial}{\partial y}(xy) = e^{xy} \cdot x\)

\[ \boxed{\frac{\partial z}{\partial y} = xe^{xy}} \]
Common Mistakes to Avoid
  1. Forgetting to treat other variables as constants: When finding ∂z/∂x, the term y³ becomes 0 (constant!), not 3y².
  2. Wrong chain rule application: For \(e^{x+y}\), ∂/∂x gives \(e^{x+y} \cdot 1\), not \(e^{x+y} \cdot (1+0)\).
  3. Confusing ∂ with d: Use ∂ (curly d) for partial derivatives, d for ordinary derivatives.

Second Order Partial Derivatives ⭐⭐

Definition — Second Order Partial Derivatives

If z = f(x, y), the first order partial derivatives ∂z/∂x and ∂z/∂y are themselves functions of x and y. We can differentiate them again to get second order partial derivatives:

\[ \frac{\partial^2 z}{\partial x^2} = \frac{\partial}{\partial x}\left(\frac{\partial z}{\partial x}\right) = f_{xx} \]
\[ \frac{\partial^2 z}{\partial y^2} = \frac{\partial}{\partial y}\left(\frac{\partial z}{\partial y}\right) = f_{yy} \]
\[ \frac{\partial^2 z}{\partial x \partial y} = \frac{\partial}{\partial x}\left(\frac{\partial z}{\partial y}\right) = f_{yx} \]
\[ \frac{\partial^2 z}{\partial y \partial x} = \frac{\partial}{\partial y}\left(\frac{\partial z}{\partial x}\right) = f_{xy} \]
Mixed Partial Derivatives

\(f_{xy}\) and \(f_{yx}\) are called mixed partial derivatives.

Important Theorem: For most functions we encounter, the order of differentiation doesn't matter:

\[ \frac{\partial^2 z}{\partial x \partial y} = \frac{\partial^2 z}{\partial y \partial x} \quad \text{i.e., } f_{xy} = f_{yx} \]

This is known as Clairaut's Theorem (or Schwarz's Theorem).

Example 5.8 — Second Order Derivatives ⭐⭐

If \(z = x^3 + 3x^2y + y^3\), find all second order partial derivatives.

Solution

Step 1: First order derivatives

\(\frac{\partial z}{\partial x} = 3x^2 + 6xy\)

\(\frac{\partial z}{\partial y} = 3x^2 + 3y^2\)

Step 2: Second order derivatives

\(\frac{\partial^2 z}{\partial x^2} = \frac{\partial}{\partial x}(3x^2 + 6xy) = 6x + 6y\)

\[ \boxed{\frac{\partial^2 z}{\partial x^2} = 6x + 6y} \]

\(\frac{\partial^2 z}{\partial y^2} = \frac{\partial}{\partial y}(3x^2 + 3y^2) = 6y\)

\[ \boxed{\frac{\partial^2 z}{\partial y^2} = 6y} \]

\(\frac{\partial^2 z}{\partial x \partial y} = \frac{\partial}{\partial x}(3x^2 + 3y^2) = 6x\)

\[ \boxed{\frac{\partial^2 z}{\partial x \partial y} = 6x} \]

\(\frac{\partial^2 z}{\partial y \partial x} = \frac{\partial}{\partial y}(3x^2 + 6xy) = 6x\)

\[ \boxed{\frac{\partial^2 z}{\partial y \partial x} = 6x} \]

✓ Note: \(f_{xy} = f_{yx} = 6x\) (Clairaut's theorem verified)

Example 5.9 — Verify Mixed Derivatives ⭐

If \(z = x^2y + xy^2\), verify that \(\frac{\partial^2 z}{\partial x \partial y} = \frac{\partial^2 z}{\partial y \partial x}\).

Solution

Method 1: ∂²z/∂x∂y

First find ∂z/∂y: \(\frac{\partial z}{\partial y} = x^2 + 2xy\)

Then differentiate w.r.t. x: \(\frac{\partial^2 z}{\partial x \partial y} = 2x + 2y\)

Method 2: ∂²z/∂y∂x

First find ∂z/∂x: \(\frac{\partial z}{\partial x} = 2xy + y^2\)

Then differentiate w.r.t. y: \(\frac{\partial^2 z}{\partial y \partial x} = 2x + 2y\)

\[ \frac{\partial^2 z}{\partial x \partial y} = \frac{\partial^2 z}{\partial y \partial x} = \boxed{2x + 2y} \quad \text{✓ Verified} \]

5.3 Homogeneous Functions 🔥🔥

Definition — Homogeneous Function 🔥

A function f(x, y) is called homogeneous of degree n if:

\[ f(\lambda x, \lambda y) = \lambda^n f(x, y) \]

for all values of λ (lambda), where n is called the degree of homogeneity.

In simple words: If we multiply BOTH x and y by the same factor λ, the function value gets multiplied by λⁿ.

Understanding Degree of Homogeneity

The degree n is found by:

  • In each term, add the powers of x and y
  • If ALL terms have the SAME total power, that power is the degree n
  • If different terms have different total powers → NOT homogeneous
Example 5.10 — Identifying Homogeneous Functions

Check if the following are homogeneous and find their degree:

(a) \(f(x,y) = x^2 + xy + y^2\)

(b) \(f(x,y) = x^3 + 2x^2y + y^3\)

(c) \(f(x,y) = x^2 + y + 1\)

Solution

(a) \(f(x,y) = x^2 + xy + y^2\)

Term x²: power of x = 2, power of y = 0 → Total = 2

Term xy: power of x = 1, power of y = 1 → Total = 2

Term y²: power of x = 0, power of y = 2 → Total = 2

All terms have degree 2 → Homogeneous of degree 2 ✓

Verification: \(f(\lambda x, \lambda y) = (\lambda x)^2 + (\lambda x)(\lambda y) + (\lambda y)^2 = \lambda^2 x^2 + \lambda^2 xy + \lambda^2 y^2 = \lambda^2(x^2 + xy + y^2) = \lambda^2 f(x,y)\) ✓

(b) \(f(x,y) = x^3 + 2x^2y + y^3\)

Term x³: 3 + 0 = 3

Term 2x²y: 2 + 1 = 3

Term y³: 0 + 3 = 3

All terms have degree 3 → Homogeneous of degree 3 ✓

(c) \(f(x,y) = x^2 + y + 1\)

Term x²: 2 + 0 = 2

Term y: 0 + 1 = 1

Term 1: 0 + 0 = 0 (constant)

Terms have different degrees (2, 1, 0) → NOT Homogeneous ✗

Example 5.11 — More Examples of Homogeneous Functions ⭐⭐

Find the degree of homogeneity:

(a) \(f = \frac{x^3 + y^3}{x + y}\)

(b) \(f = \sqrt{x^2 + y^2}\)

(c) \(f = \tan^{-1}\left(\frac{y}{x}\right)\)

Solution

(a) \(f = \frac{x^3 + y^3}{x + y}\)

Replace x with λx and y with λy:

\(f(\lambda x, \lambda y) = \frac{(\lambda x)^3 + (\lambda y)^3}{\lambda x + \lambda y} = \frac{\lambda^3(x^3 + y^3)}{\lambda(x + y)} = \lambda^2 \cdot \frac{x^3 + y^3}{x + y} = \lambda^2 f(x,y)\)

\[ \boxed{\text{Degree} = 2} \]

(b) \(f = \sqrt{x^2 + y^2} = (x^2 + y^2)^{1/2}\)

\(f(\lambda x, \lambda y) = \sqrt{(\lambda x)^2 + (\lambda y)^2} = \sqrt{\lambda^2(x^2 + y^2)} = \lambda\sqrt{x^2 + y^2} = \lambda^1 f(x,y)\)

\[ \boxed{\text{Degree} = 1} \]

(c) \(f = \tan^{-1}\left(\frac{y}{x}\right)\)

\(f(\lambda x, \lambda y) = \tan^{-1}\left(\frac{\lambda y}{\lambda x}\right) = \tan^{-1}\left(\frac{y}{x}\right) = \lambda^0 f(x,y)\)

\[ \boxed{\text{Degree} = 0} \]
Summary — Common Homogeneous Functions
FunctionDegreeReason
\(x^2 + xy + y^2\)2All terms: power 2
\(x^3 + y^3\)3All terms: power 3
\(\frac{x+y}{x-y}\)0Numerator & denominator same degree
\(\sqrt{x^2+y^2}\)1\((x^2+y^2)^{1/2}\) → 2 × ½ = 1
\(\frac{x^2+y^2}{xy}\)0Degree 2 / Degree 2 = 0

Quick Rules for Finding Degree
  • Polynomial: Add powers of x and y in each term; all should be equal
  • Fraction: Degree = (degree of numerator) − (degree of denominator)
  • √(expression): Degree = ½ × (degree of expression inside)
  • (expression)^k: Degree = k × (degree of expression)
  • Functions of (y/x) or (x/y): Degree = 0 (like sin(y/x), ln(x/y), tan⁻¹(y/x))
Example 5.12 — Finding Degree ⭐

Find the degree of \(u = \frac{x^4 + y^4}{x^2 - y^2}\)

Solution

Degree of numerator (x⁴ + y⁴) = 4

Degree of denominator (x² − y²) = 2

Degree of function = 4 − 2 = 2

\[ \boxed{\text{Degree} = 2} \]

5.4 Euler's Theorem on Homogeneous Functions 🔥🔥🔥

Euler's Theorem (First Order) — MUST MEMORIZE!

If \(z = f(x, y)\) is a homogeneous function of degree n in x and y, then:

\[ x\frac{\partial z}{\partial x} + y\frac{\partial z}{\partial y} = nz \]

Or equivalently:

\[ x \cdot z_x + y \cdot z_y = n \cdot z \]

Statement: "The sum of x times partial derivative w.r.t. x and y times partial derivative w.r.t. y equals n times the function itself."

Why is Euler's Theorem Important?
  • 🔥 Most common exam question from this unit!
  • Two types of questions: (a) Verify Euler's theorem (b) Use it to find expressions
  • You must identify that the function is homogeneous AND find its degree
Steps to Verify Euler's Theorem

Check if f(x, y) is homogeneous. Find degree n.

Find \(\frac{\partial z}{\partial x}\) and \(\frac{\partial z}{\partial y}\).

Calculate LHS: \(x\frac{\partial z}{\partial x} + y\frac{\partial z}{\partial y}\)

Calculate RHS: \(nz\)

Show LHS = RHS. Hence verified.

Example 5.13 — Verify Euler's Theorem ⭐⭐⭐

Verify Euler's theorem for \(z = x^3 + 3x^2y + y^3\).

Solution

Each term: x³ (degree 3), 3x²y (degree 2+1=3), y³ (degree 3)

All terms have degree 3, so z is homogeneous with n = 3.

\(\frac{\partial z}{\partial x} = 3x^2 + 6xy\)

\(\frac{\partial z}{\partial y} = 3x^2 + 3y^2\)

\(x\frac{\partial z}{\partial x} + y\frac{\partial z}{\partial y}\)

\(= x(3x^2 + 6xy) + y(3x^2 + 3y^2)\)

\(= 3x^3 + 6x^2y + 3x^2y + 3y^3\)

\(= 3x^3 + 9x^2y + 3y^3\)

\(= 3(x^3 + 3x^2y + y^3)\)

\(nz = 3 \times (x^3 + 3x^2y + y^3) = 3(x^3 + 3x^2y + y^3)\)

LHS = RHS = \(3(x^3 + 3x^2y + y^3)\)

\[ \boxed{\text{Hence, Euler's theorem is verified.}} \]
Example 5.14 — Verify Euler's Theorem for Fraction ⭐⭐⭐

Verify Euler's theorem for \(u = \frac{x^2 + y^2}{x + y}\).

Solution

Degree of numerator = 2, Degree of denominator = 1

Degree of u = 2 − 1 = 1

(Or verify: \(u(\lambda x, \lambda y) = \frac{\lambda^2(x^2+y^2)}{\lambda(x+y)} = \lambda u(x,y)\))

Using quotient rule:

\(\frac{\partial u}{\partial x} = \frac{(x+y)(2x) - (x^2+y^2)(1)}{(x+y)^2} = \frac{2x^2 + 2xy - x^2 - y^2}{(x+y)^2} = \frac{x^2 + 2xy - y^2}{(x+y)^2}\)

\(\frac{\partial u}{\partial y} = \frac{(x+y)(2y) - (x^2+y^2)(1)}{(x+y)^2} = \frac{2xy + 2y^2 - x^2 - y^2}{(x+y)^2} = \frac{-x^2 + 2xy + y^2}{(x+y)^2}\)

\(x\frac{\partial u}{\partial x} + y\frac{\partial u}{\partial y}\)

\(= x \cdot \frac{x^2 + 2xy - y^2}{(x+y)^2} + y \cdot \frac{-x^2 + 2xy + y^2}{(x+y)^2}\)

\(= \frac{x(x^2 + 2xy - y^2) + y(-x^2 + 2xy + y^2)}{(x+y)^2}\)

\(= \frac{x^3 + 2x^2y - xy^2 - x^2y + 2xy^2 + y^3}{(x+y)^2}\)

\(= \frac{x^3 + x^2y + xy^2 + y^3}{(x+y)^2}\)

\(= \frac{x^2(x+y) + y^2(x+y)}{(x+y)^2} = \frac{(x^2+y^2)(x+y)}{(x+y)^2} = \frac{x^2+y^2}{x+y}\)

\(nu = 1 \times \frac{x^2+y^2}{x+y} = \frac{x^2+y^2}{x+y}\)

LHS = RHS = \(\frac{x^2+y^2}{x+y}\)

\[ \boxed{\text{Euler's theorem verified.}} \]
Example 5.15 — Direct Application of Euler's Theorem ⭐⭐

If \(z = x^3 + y^3 - 3x^2y\), find the value of \(x\frac{\partial z}{\partial x} + y\frac{\partial z}{\partial y}\).

Solution

First check: Is z homogeneous?

Terms: x³ (degree 3), y³ (degree 3), −3x²y (degree 2+1=3)

Yes, z is homogeneous of degree n = 3.

By Euler's theorem:

\[ x\frac{\partial z}{\partial x} + y\frac{\partial z}{\partial y} = nz = \boxed{3(x^3 + y^3 - 3x^2y)} \]

Note: No need to actually find the partial derivatives when using Euler's theorem directly!

Example 5.16 — Euler's Theorem for √ Function ⭐⭐

Verify Euler's theorem for \(u = \sqrt{x^2 + y^2}\).

Solution

Degree: \(u = (x^2 + y^2)^{1/2}\), degree = \(\frac{1}{2} \times 2 = 1\)

So n = 1.

Partial derivatives:

\(\frac{\partial u}{\partial x} = \frac{1}{2}(x^2+y^2)^{-1/2} \cdot 2x = \frac{x}{\sqrt{x^2+y^2}}\)

\(\frac{\partial u}{\partial y} = \frac{1}{2}(x^2+y^2)^{-1/2} \cdot 2y = \frac{y}{\sqrt{x^2+y^2}}\)

LHS:

\(x\frac{\partial u}{\partial x} + y\frac{\partial u}{\partial y} = x \cdot \frac{x}{\sqrt{x^2+y^2}} + y \cdot \frac{y}{\sqrt{x^2+y^2}}\)

\(= \frac{x^2 + y^2}{\sqrt{x^2+y^2}} = \sqrt{x^2+y^2}\)

RHS: \(nu = 1 \times \sqrt{x^2+y^2} = \sqrt{x^2+y^2}\)

\[ \text{LHS = RHS} \Rightarrow \boxed{\text{Verified}} \]
Example 5.17 — Euler's Theorem with tan⁻¹ ⭐⭐⭐

If \(u = \tan^{-1}\left(\frac{x^3 + y^3}{x + y}\right)\), find \(x\frac{\partial u}{\partial x} + y\frac{\partial u}{\partial y}\).

Solution

Let \(v = \frac{x^3 + y^3}{x + y}\), so \(u = \tan^{-1}v\)

Degree of v: = (degree of x³+y³) − (degree of x+y) = 3 − 1 = 2

Since u = tan⁻¹(v) and v is homogeneous of degree 2, u is NOT directly homogeneous.

But we can work with v:

By Euler's theorem for v: \(x\frac{\partial v}{\partial x} + y\frac{\partial v}{\partial y} = 2v\)

Now, \(u = \tan^{-1}v\), so:

\(\frac{\partial u}{\partial x} = \frac{1}{1+v^2} \cdot \frac{\partial v}{\partial x}\)

\(\frac{\partial u}{\partial y} = \frac{1}{1+v^2} \cdot \frac{\partial v}{\partial y}\)

\(x\frac{\partial u}{\partial x} + y\frac{\partial u}{\partial y} = \frac{1}{1+v^2}\left(x\frac{\partial v}{\partial x} + y\frac{\partial v}{\partial y}\right) = \frac{1}{1+v^2} \times 2v = \frac{2v}{1+v^2}\)

\[ x\frac{\partial u}{\partial x} + y\frac{\partial u}{\partial y} = \boxed{\frac{2v}{1+v^2} = \frac{2 \cdot \frac{x^3+y^3}{x+y}}{1+\left(\frac{x^3+y^3}{x+y}\right)^2}} \]

Or simply: \(\boxed{\sin 2u}\) (since \(\frac{2\tan\theta}{1+\tan^2\theta} = \sin 2\theta\))

Common Exam Mistakes
  1. Not checking homogeneity: Always verify the function is homogeneous before applying Euler's theorem!
  2. Wrong degree calculation: Be careful with fractions and powers
  3. Calculation errors in partial derivatives: Double-check your differentiation
  4. Forgetting to multiply by x and y: It's x·(∂z/∂x), not just (∂z/∂x)
Example 5.18 — Another Euler's Theorem Verification ⭐⭐

Verify Euler's theorem for \(z = x^2 + xy - y^2\).

Solution

Step 1: Check homogeneity

Terms: x² (degree 2), xy (degree 1+1=2), −y² (degree 2)

All terms have degree 2 → Homogeneous with n = 2

Step 2: Find partial derivatives

\(\frac{\partial z}{\partial x} = 2x + y\)

\(\frac{\partial z}{\partial y} = x - 2y\)

Step 3: Calculate LHS

\(x\frac{\partial z}{\partial x} + y\frac{\partial z}{\partial y} = x(2x + y) + y(x - 2y)\)

\(= 2x^2 + xy + xy - 2y^2 = 2x^2 + 2xy - 2y^2\)

\(= 2(x^2 + xy - y^2)\)

Step 4: Calculate RHS

\(nz = 2(x^2 + xy - y^2)\)

Step 5: LHS = RHS ✓

\[ \boxed{\text{Euler's theorem is verified.}} \]

Euler's Theorem (First Order)

If z = f(x, y) is homogeneous of degree n, then:

\[ \boxed{x\frac{\partial z}{\partial x} + y\frac{\partial z}{\partial y} = nz} \]

Exam Strategy

MCQs from Unit 5 typically test: Evaluating partial derivatives, finding degree of homogeneous functions, and applying Euler's theorem. Practice these patterns!

Q 1.

If \(z = x^3 + y^3\), then \(\frac{\partial z}{\partial x}\) is:

  • a. \(3x^2 + 3y^2\)
  • b. \(3x^2\)
  • c. \(x^3\)
  • d. \(3x^2 + y^3\)
Answer: b. \(3x^2\)

When differentiating w.r.t. x, y³ is treated as constant (derivative = 0). So ∂z/∂x = 3x² + 0 = 3x².

Q 2.

The degree of homogeneity of \(f(x,y) = x^2 + xy + y^2\) is:

  • a. 1
  • b. 2
  • c. 3
  • d. 0
Answer: b. 2

Each term has total degree 2: x² (2+0), xy (1+1), y² (0+2). All equal → degree = 2.

Q 3.

If \(z = e^{xy}\), then \(\frac{\partial z}{\partial x}\) is:

  • a. \(e^{xy}\)
  • b. \(xe^{xy}\)
  • c. \(ye^{xy}\)
  • d. \(xye^{xy}\)
Answer: c. \(ye^{xy}\)

Using chain rule: ∂/∂x(e^{xy}) = e^{xy} · ∂(xy)/∂x = e^{xy} · y = ye^{xy}

Q 4.

According to Euler's theorem, if z is homogeneous of degree n, then:

  • a. \(x\frac{\partial z}{\partial x} + y\frac{\partial z}{\partial y} = z\)
  • b. \(x\frac{\partial z}{\partial x} + y\frac{\partial z}{\partial y} = nz\)
  • c. \(\frac{\partial z}{\partial x} + \frac{\partial z}{\partial y} = nz\)
  • d. \(x\frac{\partial z}{\partial x} \cdot y\frac{\partial z}{\partial y} = nz\)
Answer: b. \(x\frac{\partial z}{\partial x} + y\frac{\partial z}{\partial y} = nz\)

This is the statement of Euler's theorem for homogeneous functions of degree n.

Q 5.

The degree of \(f = \frac{x^3 + y^3}{x + y}\) is:

  • a. 3
  • b. 1
  • c. 2
  • d. 4
Answer: c. 2

Degree = (degree of numerator) − (degree of denominator) = 3 − 1 = 2

Q 6.

If \(z = x^2y + xy^2\), then \(\frac{\partial^2 z}{\partial x \partial y}\) is:

  • a. \(2x + 2y\)
  • b. \(2xy\)
  • c. \(x + y\)
  • d. \(2x + y\)
Answer: a. \(2x + 2y\)

∂z/∂y = x² + 2xy, then ∂²z/∂x∂y = 2x + 2y

Q 7.

Which of the following is NOT homogeneous?

  • a. \(x^2 + y^2\)
  • b. \(x^2 + xy\)
  • c. \(x^2 + y + 1\)
  • d. \(\frac{x+y}{x-y}\)
Answer: c. \(x^2 + y + 1\)

Terms have different degrees: x² (deg 2), y (deg 1), 1 (deg 0). Not homogeneous.

Q 8.

If \(z = \sin(x + y)\), then \(\frac{\partial z}{\partial x}\) is:

  • a. \(\sin(x + y)\)
  • b. \(\cos(x + y)\)
  • c. \(-\cos(x + y)\)
  • d. \(\cos x + \cos y\)
Answer: b. \(\cos(x + y)\)

Chain rule: ∂/∂x[sin(x+y)] = cos(x+y) · 1 = cos(x+y)

Q 9.

The degree of \(\sqrt{x^2 + y^2}\) is:

  • a. 2
  • b. 1
  • c. 1/2
  • d. 0
Answer: b. 1

\(\sqrt{x^2+y^2} = (x^2+y^2)^{1/2}\), degree = ½ × 2 = 1

Q 10.

If \(u = \ln(x/y)\), then \(\frac{\partial u}{\partial x}\) is:

  • a. \(\frac{1}{x}\)
  • b. \(\frac{1}{y}\)
  • c. \(\frac{y}{x}\)
  • d. \(\frac{1}{xy}\)
Answer: a. \(\frac{1}{x}\)

\(\ln(x/y) = \ln x - \ln y\). ∂/∂x gives 1/x − 0 = 1/x

Q 11.

The degree of \(\tan^{-1}(y/x)\) is:

  • a. 1
  • b. −1
  • c. 0
  • d. 2
Answer: c. 0

Functions of y/x (or x/y) are homogeneous of degree 0.

Q 12.

If \(z = x^3 - 3x^2y\), then \(x\frac{\partial z}{\partial x} + y\frac{\partial z}{\partial y}\) equals:

  • a. \(2z\)
  • b. \(3z\)
  • c. \(z\)
  • d. \(4z\)
Answer: b. \(3z\)

z is homogeneous of degree 3 (x³ has deg 3, 3x²y has deg 2+1=3). By Euler's theorem: x·zₓ + y·zᵧ = nz = 3z

Q 13.

If \(z = x^2 + 2xy\), then \(\frac{\partial z}{\partial y}\) is:

  • a. \(2x + 2y\)
  • b. \(2x\)
  • c. \(2y\)
  • d. \(x^2 + 2x\)
Answer: b. \(2x\)

∂/∂y(x²) = 0 (x² is constant w.r.t. y), ∂/∂y(2xy) = 2x. Total = 2x

Q 14.

For \(f_{xy} = f_{yx}\) to hold, the function must be:

  • a. Homogeneous
  • b. Continuous with continuous partial derivatives
  • c. Linear
  • d. Always true for all functions
Answer: b. Continuous with continuous partial derivatives

This is Clairaut's theorem — mixed partials are equal if the function has continuous second partial derivatives.

Q 15.

If \(z = \ln(x^2 + y^2)\), then \(\frac{\partial z}{\partial x}\) is:

  • a. \(\frac{2x}{x^2 + y^2}\)
  • b. \(\frac{1}{x^2 + y^2}\)
  • c. \(\frac{x}{x^2 + y^2}\)
  • d. \(\frac{2}{x}\)
Answer: a. \(\frac{2x}{x^2 + y^2}\)

Using chain rule: ∂/∂x[ln(x²+y²)] = 1/(x²+y²) · 2x = 2x/(x²+y²)

Short & Long Answer Questions

Short Answer Questions (2–3 Marks)

Q1. Define partial derivative with an example.

Answer: A partial derivative of a function of two or more variables is the derivative with respect to one variable, treating all other variables as constants.

Example: If z = x² + xy + y², then:

∂z/∂x = 2x + y (treating y as constant)

∂z/∂y = x + 2y (treating x as constant)

Q2. What is a homogeneous function? Give an example.

Answer: A function f(x, y) is called homogeneous of degree n if f(λx, λy) = λⁿf(x, y) for all values of λ.

Example: f(x, y) = x³ + y³ is homogeneous of degree 3.

Verification: f(λx, λy) = (λx)³ + (λy)³ = λ³(x³ + y³) = λ³f(x, y) ✓

Q3. State Euler's theorem on homogeneous functions.

Answer: If z = f(x, y) is a homogeneous function of degree n in x and y, then:

\[x\frac{\partial z}{\partial x} + y\frac{\partial z}{\partial y} = nz\]

This means the sum of x times partial derivative w.r.t. x and y times partial derivative w.r.t. y equals n times the function itself.

Q4. Find ∂z/∂x and ∂z/∂y if z = x²y + xy².

Answer:

∂z/∂x = 2xy + y² (differentiating w.r.t. x, y constant)

∂z/∂y = x² + 2xy (differentiating w.r.t. y, x constant)

Q5. Find the degree of homogeneity of \(u = \frac{x^2 - y^2}{x + y}\).

Answer:

Degree of numerator (x² − y²) = 2

Degree of denominator (x + y) = 1

Degree of u = 2 − 1 = 1

Q6. What is the difference between ∂ and d?

Answer:

  • d (ordinary derivative): Used for functions of one variable, e.g., dy/dx for y = f(x)
  • ∂ (partial derivative): Used for functions of two or more variables, e.g., ∂z/∂x for z = f(x, y)

∂ indicates that other variables are held constant during differentiation.

Q7. Find ∂z/∂x if z = e^(x+y).

Answer:

Using chain rule:

∂z/∂x = e^(x+y) · ∂(x+y)/∂x = e^(x+y) · 1 = e^(x+y)

Q8. Is f(x, y) = x² + 3x + y² homogeneous? Justify.

Answer: No, f(x, y) = x² + 3x + y² is NOT homogeneous.

Reason:

  • Term x²: degree = 2
  • Term 3x: degree = 1
  • Term y²: degree = 2

Since all terms don't have the same degree, the function is not homogeneous.

Long Answer Questions (5–7 Marks)

Q9. Verify Euler's theorem for z = x³ + 3x²y + 3xy² + y³. [5 marks]

Solution:

Step 1: Check homogeneity

All terms: x³ (deg 3), 3x²y (deg 3), 3xy² (deg 3), y³ (deg 3)

z is homogeneous of degree n = 3

Step 2: Find partial derivatives

∂z/∂x = 3x² + 6xy + 3y²

∂z/∂y = 3x² + 6xy + 3y²

Step 3: Calculate LHS

x·∂z/∂x + y·∂z/∂y

= x(3x² + 6xy + 3y²) + y(3x² + 6xy + 3y²)

= 3x³ + 6x²y + 3xy² + 3x²y + 6xy² + 3y³

= 3x³ + 9x²y + 9xy² + 3y³

= 3(x³ + 3x²y + 3xy² + y³)

Step 4: Calculate RHS

nz = 3(x³ + 3x²y + 3xy² + y³)

Step 5: LHS = RHS

Hence, Euler's theorem is verified. ✓

Q10. If u = log(x³ + y³ - x²y - xy²), find x·∂u/∂x + y·∂u/∂y. [6 marks]

Solution:

Let v = x³ + y³ - x²y - xy², so u = log(v)

Check degree of v:

x³ (deg 3), y³ (deg 3), x²y (deg 3), xy² (deg 3)

v is homogeneous of degree 3

By Euler's theorem for v:

x·∂v/∂x + y·∂v/∂y = 3v

Now, u = log(v):

∂u/∂x = (1/v)·∂v/∂x

∂u/∂y = (1/v)·∂v/∂y

Therefore:

x·∂u/∂x + y·∂u/∂y = x·(1/v)·∂v/∂x + y·(1/v)·∂v/∂y

= (1/v)(x·∂v/∂x + y·∂v/∂y)

= (1/v)(3v)

= 3

Q11. If z = x⁴ + x³y + xy³ + y⁴, verify that ∂²z/∂x∂y = ∂²z/∂y∂x. [5 marks]

Solution:

Method 1: Find ∂²z/∂x∂y

First: ∂z/∂y = x³ + 3xy² + 4y³

Then: ∂²z/∂x∂y = ∂/∂x(x³ + 3xy² + 4y³) = 3x² + 3y²

Method 2: Find ∂²z/∂y∂x

First: ∂z/∂x = 4x³ + 3x²y + y³

Then: ∂²z/∂y∂x = ∂/∂y(4x³ + 3x²y + y³) = 3x² + 3y²

Conclusion:

∂²z/∂x∂y = ∂²z/∂y∂x = 3x² + 3y²

Hence verified. ✓

Q12. Find all first and second order partial derivatives of z = x³y + xy³. [6 marks]

Solution:

First Order Partial Derivatives:

∂z/∂x = 3x²y + y³

∂z/∂y = x³ + 3xy²

Second Order Partial Derivatives:

∂²z/∂x² = ∂/∂x(3x²y + y³) = 6xy

∂²z/∂y² = ∂/∂y(x³ + 3xy²) = 6xy

∂²z/∂x∂y = ∂/∂x(x³ + 3xy²) = 3x² + 3y²

∂²z/∂y∂x = ∂/∂y(3x²y + y³) = 3x² + 3y²

Note: ∂²z/∂x∂y = ∂²z/∂y∂x = 3x² + 3y² (Clairaut's theorem verified)

Q13. If u = sin⁻¹[(x² + y²)/(x + y)], find x·∂u/∂x + y·∂u/∂y. [6 marks]

Solution:

Let v = (x² + y²)/(x + y), so u = sin⁻¹(v)

Degree of v:

Degree of numerator = 2, Degree of denominator = 1

Degree of v = 2 - 1 = 1

By Euler's theorem for v:

x·∂v/∂x + y·∂v/∂y = 1·v = v

For u = sin⁻¹(v):

∂u/∂x = 1/√(1-v²) · ∂v/∂x

∂u/∂y = 1/√(1-v²) · ∂v/∂y

Therefore:

x·∂u/∂x + y·∂u/∂y = 1/√(1-v²) · (x·∂v/∂x + y·∂v/∂y)

= 1/√(1-v²) · v

= v/√(1-v²)

= tan u (since if u = sin⁻¹v, then v = sin u, and v/√(1-v²) = tan u)

Check: x² (deg 2), 2xy (deg 2), y² (deg 2)

z is homogeneous of degree n = 2

By Euler's theorem:

x·∂z/∂x + y·∂z/∂y = nz = 2(x² + 2xy + y²) = 2(x + y)²

Q18. If u = (x + y)/(x - y), find ∂u/∂x at x = 2, y = 1. [4 marks]

Solution:

Using quotient rule:

∂u/∂x = [(x-y)(1) - (x+y)(1)] / (x-y)²

= (x - y - x - y) / (x-y)²

= -2y / (x-y)²

At x = 2, y = 1:

∂u/∂x = -2(1) / (2-1)² = -2/1 = -2

Quick Revision — Unit Summary

Complete Unit Summary

📌 5.1 Partial Derivative Basics

  • Definition: Derivative of function of multiple variables w.r.t. one variable, keeping others constant
  • Notation: ∂z/∂x, fₓ, zₓ (curly d, not straight d)
  • Rule: To find ∂z/∂x — treat y as constant, differentiate w.r.t. x normally

📌 5.2 Evaluating Partial Derivatives

  • Use standard differentiation rules (power, chain, product, quotient)
  • Remember: constant × variable differentiation
  • Second order: ∂²z/∂x², ∂²z/∂y², ∂²z/∂x∂y
  • Clairaut's Theorem: ∂²z/∂x∂y = ∂²z/∂y∂x

📌 5.3 Homogeneous Functions

  • Definition: f(λx, λy) = λⁿf(x, y)
  • Polynomial: All terms must have same total degree
  • Fraction: Degree = deg(numerator) − deg(denominator)
  • Power: Degree = k × deg(expression) for [f]^k
  • Functions of y/x: Degree = 0

📌 5.4 Euler's Theorem (MOST IMPORTANT!)

\[ \boxed{x\frac{\partial z}{\partial x} + y\frac{\partial z}{\partial y} = nz} \]
  • Applies only to homogeneous functions
  • n = degree of homogeneity
  • Can use to find x·zₓ + y·zᵧ without actually computing derivatives!
Don't Forget in Exam!
  1. Use ∂ (curly d) for partial derivatives, not d
  2. When differentiating w.r.t. x, y is treated as constant (and vice versa)
  3. Always check homogeneity before applying Euler's theorem
  4. For fractions, degree = (numerator degree) − (denominator degree)
  5. For √(x²+y²), degree = ½ × 2 = 1
  6. tan⁻¹(y/x), ln(x/y), sin(y/x) → degree = 0
