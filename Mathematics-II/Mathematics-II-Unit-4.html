Ordinary Differential Equation: Order & Degree

Definition

An ordinary differential equation (ODE) is an equation involving derivatives of a dependent variable with respect to one independent variable.

Order

The order of the highest derivative appearing in the equation.

Example: \( \frac{d^2y}{dx^2} + 3\frac{dy}{dx} + y = 0 \) has order 2.

Degree

The power of the highest order derivative after the equation is made free of radicals and fractions.

Example: \( \left(\frac{d^2y}{dx^2}\right)^3 + 5y = 0 \) has degree 3.

Example

Find order and degree of \( \left(\frac{d^3y}{dx^3}\right)^2 + \sin\left(\frac{dy}{dx}\right) = 0 \).

Order = 3 (highest derivative is \( \frac{d^3y}{dx^3} \))
Degree = Not defined because of the sine term (not a polynomial in derivatives).

Separation of Variables

If equation can be written as \( \frac{dy}{dx} = f(x)g(y) \), then separate: \( \frac{dy}{g(y)} = f(x)dx \) and integrate.

Example: Separation

Solve \( \frac{dy}{dx} = 2xy \).

Separate: \( \frac{dy}{y} = 2x dx \)
Integrate: \( \ln|y| = x^2 + C \) ⇒ \( y = Ae^{x^2} \)
Homogeneous Type

Equation of form \( \frac{dy}{dx} = \frac{f(x,y)}{g(x,y)} \) where f and g are homogeneous of same degree. Substitute \( y = vx \).

Example: Homogeneous

Solve \( \frac{dy}{dx} = \frac{x+y}{x} \).

Let \( y = vx \), \( \frac{dy}{dx} = v + x\frac{dv}{dx} \)
Equation becomes \( v + x\frac{dv}{dx} = 1 + v \) ⇒ \( x\frac{dv}{dx} = 1 \) ⇒ \( dv = \frac{dx}{x} \)
Integrate: \( v = \ln|x| + C \) ⇒ \( y = x(\ln|x| + C) \)
Exact Type

Equation \( M dx + N dy = 0 \) is exact if \( \frac{\partial M}{\partial y} = \frac{\partial N}{\partial x} \).

Solution: \( \int M dx + \int (N - \frac{\partial}{\partial y}\int M dx) dy = C \).

Example: Exact

Solve \( (2xy + 1) dx + (x^2 + 3) dy = 0 \).

\( M = 2xy+1, N = x^2+3 \)
\( \frac{\partial M}{\partial y} = 2x, \frac{\partial N}{\partial x} = 2x \) ⇒ Exact
\( \int M dx = \int (2xy+1)dx = x^2y + x + f(y) \)
\( \frac{\partial}{\partial y}(x^2y + x + f(y)) = x^2 + f'(y) = N = x^2+3 \) ⇒ \( f'(y)=3 \) ⇒ \( f(y)=3y \)
Solution: \( x^2y + x + 3y = C \)
Linear Type

Form: \( \frac{dy}{dx} + P(x)y = Q(x) \). Integrating factor \( IF = e^{\int P dx} \).

Solution: \( y(IF) = \int Q(IF) dx + C \).

Example: Linear

Solve \( \frac{dy}{dx} + 2xy = x \).

\( P = 2x, IF = e^{\int 2x dx} = e^{x^2} \)
\( y e^{x^2} = \int x e^{x^2} dx = \frac{1}{2}e^{x^2} + C \)
\( y = \frac{1}{2} + Ce^{-x^2} \)

Second Order Linear ODE with Constant Coefficients

General Form

\( a\frac{d^2y}{dx^2} + b\frac{dy}{dx} + cy = f(x) \) with constant \( a,b,c \).

Solution: \( y = C.F. + P.I. \)

Complementary Function (C.F.)

Solve auxiliary equation \( am^2 + bm + c = 0 \):

  • Roots real distinct \( m_1, m_2 \): \( C.F. = C_1 e^{m_1 x} + C_2 e^{m_2 x} \)
  • Roots real equal \( m \): \( C.F. = (C_1 + C_2 x)e^{m x} \)
  • Roots complex \( \alpha \pm i\beta \): \( C.F. = e^{\alpha x}(C_1 \cos\beta x + C_2 \sin\beta x) \)
Particular Integral (P.I.)

Case 1: Polynomial function

If \( f(x) \) is polynomial, assume P.I. polynomial of same degree.

Case 2: \( e^{ax} \)

If \( f(x) = e^{ax} \), try \( P.I. = \frac{e^{ax}}{a^2a + ba + c} \) provided denominator ≠ 0.

Case 3: \( \sin ax \) or \( \cos ax \)

If \( f(x) = \sin ax \) (or \( \cos ax \)), and \( F(-a^2) \neq 0 \), where \( F(D) = aD^2 + bD + c \), then:

\( P.I. = \frac{\sin ax}{F(-a^2)} \) or \( \frac{\cos ax}{F(-a^2)} \).

Case 4: \( e^{ax} V \) where V is polynomial

Use method: \( P.I. = \frac{1}{F(D)} e^{ax} V = e^{ax} \frac{1}{F(D+a)} V \).

Example: CF + PI (e^{ax})

Solve \( y'' - 3y' + 2y = e^{3x} \).

Auxiliary: \( m^2 - 3m + 2 = 0 \) ⇒ \( m = 1, 2 \)
CF = \( C_1 e^{x} + C_2 e^{2x} \)
PI: \( \frac{e^{3x}}{(3)^2 - 3(3) + 2} = \frac{e^{3x}}{9 - 9 + 2} = \frac{e^{3x}}{2} \)
General solution: \( y = C_1 e^{x} + C_2 e^{2x} + \frac{1}{2}e^{3x} \)
Example: sin ax

Solve \( y'' + 4y = \sin 2x \).

Auxiliary: \( m^2 + 4 = 0 \) ⇒ \( m = \pm 2i \)
CF = \( C_1 \cos 2x + C_2 \sin 2x \)
For PI: \( F(D) = D^2 + 4 \), \( F(-a^2) = F(-4) = (-4) + 4 = 0 \) so need modification.
Use \( P.I. = -\frac{x}{2a} \cos ax \) rule: \( P.I. = -\frac{x}{4} \cos 2x \)
General: \( y = C_1 \cos 2x + C_2 \sin 2x - \frac{x}{4}\cos 2x \)
Example: e^{ax}V (polynomial)

Solve \( y'' - 2y' + y = x e^{x} \).

Auxiliary: \( m^2 - 2m + 1 = (m-1)^2 = 0 \) ⇒ \( m = 1,1 \)
CF = \( (C_1 + C_2 x)e^{x} \)
PI: Use \( \frac{1}{F(D)} e^{x} x = e^{x} \frac{1}{F(D+1)} x \)
\( F(D) = (D-1)^2 \), so \( F(D+1) = D^2 \)
\( \frac{1}{D^2} x = \int\int x dx dx = \int \frac{x^2}{2} dx = \frac{x^3}{6} \)
PI = \( e^{x} \cdot \frac{x^3}{6} \)
General: \( y = (C_1 + C_2 x)e^{x} + \frac{x^3}{6}e^{x} \)
