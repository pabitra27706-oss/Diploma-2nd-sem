Unit Overview

What This Unit Covers

A Differential Equation (DE) is an equation that contains one or more derivatives of an unknown function. Solving a DE means finding the unknown function that satisfies the equation.

Why are DEs important? They model real-world phenomena where things change:

  • ⚡ Electrical circuits: Current/voltage changes over time (RC, RL, RLC circuits)
  • 🔥 Heat transfer: Temperature changes with time/position
  • 🏗️ Mechanics: Vibrations, motion, forces
  • 📈 Population growth: Growth/decay models
  • 💧 Fluid flow: How liquids move through pipes

Real-world analogy: If you know how fast something is changing (derivative), a differential equation helps you figure out what that something actually is!

4.1 Definition of Ordinary Differential Equation

Definition — Differential Equation

A differential equation is an equation that involves:

  • An unknown function (usually y)
  • One or more of its derivatives (\(\frac{dy}{dx}, \frac{d^2y}{dx^2}\), etc.)
  • Possibly the independent variable (usually x) and constants
Ordinary vs Partial Differential Equation

Ordinary Differential Equation (ODE): The unknown function depends on only one independent variable. Contains ordinary derivatives like \(\frac{dy}{dx}\).

Partial Differential Equation (PDE): The unknown function depends on two or more independent variables. Contains partial derivatives like \(\frac{\partial u}{\partial x}\).

This syllabus covers only ODEs.

Solution of a Differential Equation

A solution of a differential equation is a function y = f(x) which, when substituted into the equation, makes it an identity (true statement).

  • General Solution: Contains arbitrary constants (as many as the order of the DE)
  • Particular Solution: Obtained from general solution by giving specific values to the constants (using initial/boundary conditions)
Example 4.1 — Verifying a Solution

Verify that \(y = e^{2x}\) is a solution of \(\frac{dy}{dx} = 2y\).

Solution

Given: \(y = e^{2x}\)

LHS: \(\frac{dy}{dx} = 2e^{2x}\)

RHS: \(2y = 2e^{2x}\)

Since LHS = RHS, \(y = e^{2x}\) is indeed a solution. ✓

4.1 Order and Degree of a Differential Equation 🔥

Definition — Order

The order of a differential equation is the highest order derivative present in the equation.

  • If highest derivative is \(\frac{dy}{dx}\) or \(y'\) → Order = 1
  • If highest derivative is \(\frac{d^2y}{dx^2}\) or \(y''\) → Order = 2
  • If highest derivative is \(\frac{d^3y}{dx^3}\) or \(y'''\) → Order = 3
Definition — Degree

The degree of a differential equation is the power (exponent) of the highest order derivative, after the equation has been made free from radicals and fractions involving derivatives.

Important conditions for degree to be defined:

  • The equation must be a polynomial in derivatives
  • No fractional powers of derivatives
  • No derivatives inside sin, cos, log, exp, etc.
When Degree is NOT Defined

Degree is not defined when the equation contains:

  • \(\sin\left(\frac{dy}{dx}\right)\) — derivative inside trig function
  • \(e^{dy/dx}\) — derivative inside exponential
  • \(\ln\left(\frac{dy}{dx}\right)\) — derivative inside logarithm
Example 4.2 — Finding Order and Degree

Find the order and degree of:

(i) \(\frac{d^2y}{dx^2} + 3\frac{dy}{dx} + 2y = 0\)

(ii) \(\left(\frac{dy}{dx}\right)^3 + y^2 = x\)

(iii) \(\frac{d^3y}{dx^3} + \left(\frac{d^2y}{dx^2}\right)^4 + y = 0\)

Solution

(i) Highest derivative: \(\frac{d^2y}{dx^2}\) → Order = 2

Power of highest derivative: 1 → Degree = 1

(ii) Highest derivative: \(\frac{dy}{dx}\) → Order = 1

Power of \(\frac{dy}{dx}\): 3 → Degree = 3

(iii) Highest derivative: \(\frac{d^3y}{dx^3}\) → Order = 3

Power of \(\frac{d^3y}{dx^3}\): 1 → Degree = 1

(Note: The power 4 is on \(\frac{d^2y}{dx^2}\), which is NOT the highest order derivative.)

Example 4.3 — Equation with Radical

Find order and degree of: \(\sqrt{1 + \left(\frac{dy}{dx}\right)^2} = \frac{d^2y}{dx^2}\)

Solution

First, remove the radical by squaring both sides:

\(1 + \left(\frac{dy}{dx}\right)^2 = \left(\frac{d^2y}{dx^2}\right)^2\)

Now: Highest derivative: \(\frac{d^2y}{dx^2}\) → Order = 2

Power of highest derivative: 2 → Degree = 2

Example 4.4 — Degree Not Defined

Find order and degree of: \(\sin\left(\frac{dy}{dx}\right) = x + y\)

Solution

Highest derivative: \(\frac{dy}{dx}\) → Order = 1

The derivative is inside a sine function, so it's not a polynomial in the derivative.

Degree = Not Defined

4.2.1 Separation of Variables 🔥🔥

What is Separation of Variables?

This is the simplest method for solving first-order, first-degree differential equations. It works when we can write the equation in a form where all x terms (including dx) are on one side and all y terms (including dy) are on the other side.

Standard Form

If the equation can be written as:

\[ f(y)\,dy = g(x)\,dx \]

Then we integrate both sides:

\[ \int f(y)\,dy = \int g(x)\,dx + C \]
Steps for Separation of Variables

Write the equation in the form \(\frac{dy}{dx} = h(x, y)\).

Separate variables: Move all y terms and dy to one side, all x terms and dx to the other.

Integrate both sides.

Add the constant of integration C (only one C needed, add to one side).

Simplify if possible (express y in terms of x if required).

Example 4.5 — Basic Separation ⭐

Solve: \(\frac{dy}{dx} = xy\)

Solution

Separate variables:

\(\frac{dy}{y} = x\,dx\)

Integrate both sides:

\(\int \frac{dy}{y} = \int x\,dx\)

\(\ln|y| = \frac{x^2}{2} + C\)

Solve for y:

\(|y| = e^{x^2/2 + C} = e^C \cdot e^{x^2/2}\)

\[ y = Ae^{x^2/2} \quad \text{where } A = \pm e^C \]
Example 4.6 — Separation with Product ⭐

Solve: \(\frac{dy}{dx} = \frac{1+y^2}{1+x^2}\)

Solution

Separate variables:

\(\frac{dy}{1+y^2} = \frac{dx}{1+x^2}\)

Integrate both sides:

\(\tan^{-1}y = \tan^{-1}x + C\)

\[ \tan^{-1}y - \tan^{-1}x = C \]

Or equivalently: \(y = \tan(\tan^{-1}x + C)\)

Example 4.7 — Separation of Variables ⭐⭐

Solve: \((1+x)y\,dx + (1-y)x\,dy = 0\)

Solution

Rearrange:

\((1+x)y\,dx = -(1-y)x\,dy\)

Separate:

\(\frac{1+x}{x}\,dx = \frac{-(1-y)}{y}\,dy = \frac{y-1}{y}\,dy\)

\(\left(\frac{1}{x} + 1\right)dx = \left(1 - \frac{1}{y}\right)dy\)

Integrate:

\(\ln|x| + x = y - \ln|y| + C\)

\[ \ln|x| + \ln|y| + x - y = C \quad \text{or} \quad \ln|xy| + x - y = C \]
Example 4.8 — With Initial Condition

Solve: \(\frac{dy}{dx} = e^{x-y}\), given \(y(0) = 0\).

Solution

\(\frac{dy}{dx} = e^x \cdot e^{-y} = \frac{e^x}{e^y}\)

Separate: \(e^y\,dy = e^x\,dx\)

Integrate: \(e^y = e^x + C\)

Apply initial condition: When x = 0, y = 0:

\(e^0 = e^0 + C \Rightarrow 1 = 1 + C \Rightarrow C = 0\)

\[ e^y = e^x \quad \Rightarrow \quad \boxed{y = x} \]
How to Recognize Separable Equations

A DE is separable if \(\frac{dy}{dx}\) can be written as a product of a function of x alone and a function of y alone:

\(\frac{dy}{dx} = f(x) \cdot g(y)\)

Then: \(\frac{dy}{g(y)} = f(x)\,dx\)

4.2.2 Homogeneous Type Differential Equations 🔥🔥

Definition — Homogeneous Function

A function \(f(x, y)\) is called homogeneous of degree n if:

\[ f(\lambda x, \lambda y) = \lambda^n f(x, y) \]

for any constant λ and for all x, y.

Examples:

  • \(x^2 + xy + y^2\) is homogeneous of degree 2
  • \(x^3 + y^3\) is homogeneous of degree 3
  • \(\frac{x+y}{x-y}\) is homogeneous of degree 0
  • \(x + y + 1\) is NOT homogeneous (the constant 1 breaks it)
Homogeneous Differential Equation

A first-order DE \(\frac{dy}{dx} = f(x, y)\) is called homogeneous if \(f(x, y)\) can be expressed as a function of \(\frac{y}{x}\) only, i.e., \(f(x, y) = \phi\left(\frac{y}{x}\right)\).

Equivalently, if the equation can be written as \(\frac{dy}{dx} = \frac{M(x,y)}{N(x,y)}\) where M and N are homogeneous functions of the same degree.

Method — Substitution

To solve a homogeneous DE, substitute:

\[ y = vx \quad \text{where v is a function of x} \]

Then: \(\frac{dy}{dx} = v + x\frac{dv}{dx}\)

This substitution converts the homogeneous DE into a separable DE in v and x.

Steps for Homogeneous Type

Check that the equation is homogeneous (M and N have same degree, or RHS is function of y/x).

Substitute \(y = vx\), so \(\frac{dy}{dx} = v + x\frac{dv}{dx}\).

Replace y/x by v wherever it appears. The equation becomes separable in v and x.

Separate variables and integrate.

Replace v back with y/x to get the solution in terms of x and y.

Example 4.9 — Homogeneous DE ⭐⭐

Solve: \(\frac{dy}{dx} = \frac{x+y}{x}\)

Solution

The equation can be written as: \(\frac{dy}{dx} = 1 + \frac{y}{x}\)

This is a function of y/x only → Homogeneous.

Let \(y = vx\), so \(\frac{dy}{dx} = v + x\frac{dv}{dx}\)

Substituting:

\(v + x\frac{dv}{dx} = 1 + v\)

\(x\frac{dv}{dx} = 1\)

Separate: \(dv = \frac{dx}{x}\)

Integrate: \(v = \ln|x| + C\)

Replace \(v = \frac{y}{x}\):

\[ \frac{y}{x} = \ln|x| + C \quad \Rightarrow \quad \boxed{y = x\ln|x| + Cx} \]
Example 4.10 — Another Homogeneous DE ⭐⭐

Solve: \((x^2 + y^2)dx - 2xy\,dy = 0\)

Solution

Rewrite: \(\frac{dy}{dx} = \frac{x^2 + y^2}{2xy}\)

Check: Numerator degree = 2, Denominator degree = 2 → Homogeneous.

Let \(y = vx\), \(\frac{dy}{dx} = v + x\frac{dv}{dx}\):

\(v + x\frac{dv}{dx} = \frac{x^2 + v^2x^2}{2x \cdot vx} = \frac{x^2(1+v^2)}{2vx^2} = \frac{1+v^2}{2v}\)

\(x\frac{dv}{dx} = \frac{1+v^2}{2v} - v = \frac{1+v^2 - 2v^2}{2v} = \frac{1-v^2}{2v}\)

Separate: \(\frac{2v\,dv}{1-v^2} = \frac{dx}{x}\)

Integrate LHS: Let \(t = 1-v^2\), \(dt = -2v\,dv\), so \(\int \frac{2v\,dv}{1-v^2} = -\ln|1-v^2|\)

\(-\ln|1-v^2| = \ln|x| + C_1\)

\(\ln|1-v^2| = -\ln|x| + C_2 = \ln\frac{1}{|x|} + C_2\)

\(1 - v^2 = \frac{A}{x}\) where \(A = e^{C_2}\)

Replace \(v = \frac{y}{x}\): \(1 - \frac{y^2}{x^2} = \frac{A}{x}\)

\[ x^2 - y^2 = Ax \quad \text{or} \quad \boxed{x^2 - y^2 = Cx} \]
Example 4.11 — Homogeneous Type

Solve: \(x\,dy - y\,dx = \sqrt{x^2 + y^2}\,dx\)

Solution

Rewrite: \(x\,dy = (y + \sqrt{x^2+y^2})\,dx\)

\(\frac{dy}{dx} = \frac{y + \sqrt{x^2+y^2}}{x} = \frac{y}{x} + \frac{\sqrt{x^2+y^2}}{x} = \frac{y}{x} + \sqrt{1 + \frac{y^2}{x^2}}\)

This is a function of y/x → Homogeneous.

Let \(y = vx\): \(v + x\frac{dv}{dx} = v + \sqrt{1+v^2}\)

\(x\frac{dv}{dx} = \sqrt{1+v^2}\)

Separate: \(\frac{dv}{\sqrt{1+v^2}} = \frac{dx}{x}\)

Integrate: \(\ln|v + \sqrt{1+v^2}| = \ln|x| + C_1\)

\(v + \sqrt{1+v^2} = Ax\) where \(A = e^{C_1}\)

Replace \(v = y/x\):

\[ \frac{y}{x} + \sqrt{1+\frac{y^2}{x^2}} = Ax \] \[ \boxed{y + \sqrt{x^2+y^2} = Ax^2} \]
How to Recognize Homogeneous Equations
  • All terms have the same total degree in x and y
  • No standalone constants (like +1, +5)
  • Can be rewritten as a function of (y/x) or (x/y) only
  • If unsure, try substituting y = vx and see if x cancels out

4.2.3 Exact Type Differential Equations ⭐⭐

Definition — Exact Differential Equation

A first-order DE of the form:

\[ M(x,y)\,dx + N(x,y)\,dy = 0 \]

is called exact if there exists a function \(F(x,y)\) such that:

  • \(\frac{\partial F}{\partial x} = M\)
  • \(\frac{\partial F}{\partial y} = N\)

In this case, the equation becomes \(dF = 0\), so the solution is \(F(x,y) = C\).

Condition for Exactness 🔥

The equation \(M\,dx + N\,dy = 0\) is exact if and only if:

\[ \frac{\partial M}{\partial y} = \frac{\partial N}{\partial x} \]

This is the test for exactness — always check this first!

Steps for Solving Exact Equations

Identify M and N from the equation \(M\,dx + N\,dy = 0\).

Check exactness: Verify \(\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}\).

Find F by integrating: \(F = \int M\,dx\) (treat y as constant) + an unknown function g(y).

Differentiate F with respect to y and set equal to N to find g(y).

The solution is \(F(x,y) = C\).

Alternative Method (Shorter!)

The general solution can also be written as:

\[ \int M\,dx \text{ (y constant)} + \int (\text{terms of N not containing x})\,dy = C \]

This is often faster for exam purposes.

Example 4.12 — Exact DE ⭐⭐

Solve: \((2xy + 3)\,dx + (x^2 + 4y)\,dy = 0\)

Solution

\(M = 2xy + 3\), \(N = x^2 + 4y\)

\(\frac{\partial M}{\partial y} = 2x\)

\(\frac{\partial N}{\partial x} = 2x\)

Since \(\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}\), the equation is exact. ✓

Using the alternative method:

\(\int M\,dx = \int (2xy + 3)\,dx = x^2y + 3x\)

Terms in N not containing x: \(4y\)

\(\int 4y\,dy = 2y^2\)

\[ \boxed{x^2y + 3x + 2y^2 = C} \]
Example 4.13 — Exact DE (Detailed Method) ⭐

Solve: \((y\cos x + 2xe^y)dx + (\sin x + x^2e^y - 1)dy = 0\)

Solution

\(M = y\cos x + 2xe^y\), \(N = \sin x + x^2e^y - 1\)

\(\frac{\partial M}{\partial y} = \cos x + 2xe^y\)

\(\frac{\partial N}{\partial x} = \cos x + 2xe^y\)

Equal → Exact. ✓

\(\int M\,dx = \int (y\cos x + 2xe^y)\,dx = y\sin x + x^2e^y\)

Terms in N not containing x: \(-1\)

\(\int (-1)\,dy = -y\)

\[ \boxed{y\sin x + x^2e^y - y = C} \]
Example 4.14 — Checking Exactness (Not Exact)

Is the equation \((x + y)dx + x\,dy = 0\) exact?

Solution

\(M = x + y\), \(N = x\)

\(\frac{\partial M}{\partial y} = 1\)

\(\frac{\partial N}{\partial x} = 1\)

Since \(\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x} = 1\), the equation is exact. ✓

Solution: \(\int (x+y)\,dx + \int 0\,dy = \frac{x^2}{2} + xy = C\)

Common Mistakes in Exact Equations
  1. Wrong identification of M and N: M is the coefficient of dx, N is the coefficient of dy.
  2. Forgetting to check exactness: Always verify \(\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}\) first!
  3. Double counting terms: When using the alternative method, be careful not to include terms already in \(\int M\,dx\).

4.2.4 Linear Type Differential Equations 🔥🔥🔥

Definition — Linear First-Order DE

A first-order DE is called linear if it can be written in the standard form:

\[ \frac{dy}{dx} + P(x) \cdot y = Q(x) \]

where P(x) and Q(x) are functions of x only (or constants).

Key features:

  • The dependent variable y and its derivative appear to the first power only
  • No products like y·(dy/dx) or y²
  • The coefficients P and Q depend only on x
Solution Method — Integrating Factor 🔥

The solution uses an Integrating Factor (I.F.):

\[ \text{I.F.} = e^{\int P(x)\,dx} \]

The general solution is:

\[ y \cdot (\text{I.F.}) = \int Q(x) \cdot (\text{I.F.})\,dx + C \]

Or equivalently:

\[ y \cdot e^{\int P\,dx} = \int Q \cdot e^{\int P\,dx}\,dx + C \]
Steps for Linear First-Order DE

Write the equation in standard form: \(\frac{dy}{dx} + Py = Q\).

Identify P(x) and Q(x).

Calculate I.F. = \(e^{\int P\,dx}\).

Multiply both sides by I.F. and recognize that LHS = \(\frac{d}{dx}(y \cdot \text{I.F.})\).

Integrate: \(y \cdot \text{I.F.} = \int Q \cdot \text{I.F.}\,dx + C\).

Solve for y.

Example 4.15 — Linear DE (Basic) ⭐⭐

Solve: \(\frac{dy}{dx} + 2y = e^x\)

Solution

Already in standard form. \(P = 2\), \(Q = e^x\).

I.F. = \(e^{\int 2\,dx} = e^{2x}\)

Solution: \(y \cdot e^{2x} = \int e^x \cdot e^{2x}\,dx = \int e^{3x}\,dx = \frac{e^{3x}}{3} + C\)

Solve for y:

\[ y = \frac{e^{3x}}{3e^{2x}} + Ce^{-2x} = \boxed{\frac{e^x}{3} + Ce^{-2x}} \]
Example 4.16 — Linear DE with 1/x ⭐⭐

Solve: \(\frac{dy}{dx} + \frac{y}{x} = x^2\)

Solution

\(P = \frac{1}{x}\), \(Q = x^2\)

I.F. = \(e^{\int \frac{1}{x}dx} = e^{\ln x} = x\)

Solution: \(y \cdot x = \int x^2 \cdot x\,dx = \int x^3\,dx = \frac{x^4}{4} + C\)

\[ y = \frac{x^4}{4x} + \frac{C}{x} = \boxed{\frac{x^3}{4} + \frac{C}{x}} \]
Example 4.17 — Linear DE (Need to Rearrange) ⭐⭐

Solve: \(\frac{dy}{dx} - \frac{2y}{x} = x^2\cos x\)

Solution

\(P = -\frac{2}{x}\), \(Q = x^2\cos x\)

I.F. = \(e^{\int -\frac{2}{x}dx} = e^{-2\ln x} = e^{\ln x^{-2}} = x^{-2} = \frac{1}{x^2}\)

Solution: \(\frac{y}{x^2} = \int x^2\cos x \cdot \frac{1}{x^2}\,dx = \int \cos x\,dx = \sin x + C\)

\[ y = x^2(\sin x + C) = \boxed{x^2\sin x + Cx^2} \]
Example 4.18 — Linear DE (Trigonometric)

Solve: \(\frac{dy}{dx} + y\tan x = \sec x\)

Solution

\(P = \tan x\), \(Q = \sec x\)

I.F. = \(e^{\int \tan x\,dx} = e^{\ln|\sec x|} = \sec x\)

Solution: \(y \cdot \sec x = \int \sec x \cdot \sec x\,dx = \int \sec^2 x\,dx = \tan x + C\)

\[ y = \frac{\tan x + C}{\sec x} = \frac{\tan x}{\sec x} + C\cos x = \boxed{\sin x + C\cos x} \]
Linear in x (Bernoulli's hint)

Sometimes the equation is linear in x (not y). Then we treat x as the dependent variable:

\[ \frac{dx}{dy} + P(y) \cdot x = Q(y) \]

The I.F. would be \(e^{\int P(y)\,dy}\), and the solution is \(x \cdot \text{I.F.} = \int Q \cdot \text{I.F.}\,dy + C\).

Common Mistakes in Linear DEs
  1. Wrong sign in P: Be careful with the sign when rearranging to standard form.
  2. Forgetting to integrate P: I.F. = \(e^{\int P\,dx}\), not \(e^P\)!
  3. Not solving for y at the end: After finding \(y \cdot \text{I.F.}\), you must divide by I.F. to get y.
  4. Forgetting the constant: Always include +C in the solution.

4.3 Second Order Linear DE with Constant Coefficients 🔥🔥🔥

Definition — Second Order Linear DE

A second order linear differential equation with constant coefficients has the form:

\[ a\frac{d^2y}{dx^2} + b\frac{dy}{dx} + cy = f(x) \]

where a, b, c are constants (a ≠ 0), and f(x) is a function of x.

Alternative notation:

\[ ay'' + by' + cy = f(x) \quad \text{or} \quad (aD^2 + bD + c)y = f(x) \]

where \(D = \frac{d}{dx}\) (the differential operator).

Understanding the D Operator

The operator D represents differentiation with respect to x:

  • \(Dy = \frac{dy}{dx}\) — first derivative
  • \(D^2y = \frac{d^2y}{dx^2}\) — second derivative
  • \(D^ny = \frac{d^ny}{dx^n}\) — nth derivative

Why use D notation? It makes the equation look like an algebraic equation, which is easier to solve!

Homogeneous vs Non-Homogeneous

Homogeneous: If f(x) = 0, the equation is:

\[ ay'' + by' + cy = 0 \]

Non-Homogeneous: If f(x) ≠ 0, the equation is:

\[ ay'' + by' + cy = f(x) \]
General Solution Structure 🔥

The complete (general) solution of a non-homogeneous second order linear DE is:

\[ y = \text{C.F.} + \text{P.I.} \]

where:

  • C.F. (Complementary Function): General solution of the homogeneous equation (with f(x) = 0)
  • P.I. (Particular Integral): Any one particular solution of the non-homogeneous equation

For homogeneous equations: y = C.F. only (since P.I. = 0).

Auxiliary Equation (Characteristic Equation) 🔥

To find C.F., we first form the auxiliary equation by replacing:

  • D² with m²
  • D with m
  • y with 1 (or simply remove it)

For the equation \((aD^2 + bD + c)y = 0\), the auxiliary equation is:

\[ am^2 + bm + c = 0 \]

The roots of this quadratic equation determine the form of C.F.

Solving the Auxiliary Equation

Use the quadratic formula:

\[ m = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \]

The discriminant \(\Delta = b^2 - 4ac\) determines the nature of roots:

  • Δ > 0 → Two distinct real roots
  • Δ = 0 → Two equal (repeated) real roots
  • Δ < 0 → Two complex conjugate roots

4.3.1 Complementary Function (C.F.) 🔥🔥🔥

Definition — Complementary Function

The Complementary Function (C.F.) is the general solution of the homogeneous equation:

\[ ay'' + by' + cy = 0 \quad \text{or} \quad (aD^2 + bD + c)y = 0 \]

It contains two arbitrary constants (because it's a second order DE).

The Three Cases of C.F.

The form of C.F. depends on the nature of roots of the auxiliary equation \(am^2 + bm + c = 0\):

  1. Case I: Two distinct real roots (m₁ ≠ m₂)
  2. Case II: Two equal real roots (m₁ = m₂ = m)
  3. Case III: Complex conjugate roots (m = α ± iβ)

⚠️ You MUST memorize all three cases!

Case I: Two Distinct Real Roots (m₁ ≠ m₂)

When: Discriminant b² − 4ac > 0

Roots: m₁ and m₂ are different real numbers

\[ \text{C.F.} = C_1 e^{m_1 x} + C_2 e^{m_2 x} \]

where C₁ and C₂ are arbitrary constants.

Example 4.19 — Case I (Distinct Real Roots) ⭐⭐

Find the C.F. of: \(\frac{d^2y}{dx^2} - 5\frac{dy}{dx} + 6y = 0\)

Solution

Write in operator form: \((D^2 - 5D + 6)y = 0\)

Form auxiliary equation: \(m^2 - 5m + 6 = 0\)

Solve: \((m-2)(m-3) = 0\)

\(m_1 = 2\), \(m_2 = 3\) — Two distinct real roots

Apply Case I formula:

\[ \boxed{\text{C.F.} = C_1 e^{2x} + C_2 e^{3x}} \]
Example 4.20 — Case I (Using Quadratic Formula)

Find the C.F. of: \(y'' + y' - 6y = 0\)

Solution

Auxiliary equation: \(m^2 + m - 6 = 0\)

Using quadratic formula: \(m = \frac{-1 \pm \sqrt{1 + 24}}{2} = \frac{-1 \pm 5}{2}\)

\(m_1 = \frac{-1+5}{2} = 2\), \(m_2 = \frac{-1-5}{2} = -3\)

\[ \boxed{\text{C.F.} = C_1 e^{2x} + C_2 e^{-3x}} \]

Case II: Two Equal Real Roots (m₁ = m₂ = m)

When: Discriminant b² − 4ac = 0

Roots: m₁ = m₂ = m (repeated root)

\[ \text{C.F.} = (C_1 + C_2 x) e^{mx} \]

⚠️ Note the extra 'x' factor! This is crucial for repeated roots.

Example 4.21 — Case II (Equal Roots) ⭐⭐

Find the C.F. of: \(\frac{d^2y}{dx^2} - 4\frac{dy}{dx} + 4y = 0\)

Solution

Auxiliary equation: \(m^2 - 4m + 4 = 0\)

\((m-2)^2 = 0\)

\(m = 2, 2\) — Equal (repeated) roots

Apply Case II formula:

\[ \boxed{\text{C.F.} = (C_1 + C_2 x) e^{2x}} \]
Example 4.22 — Case II

Solve: \(y'' + 6y' + 9y = 0\)

Solution

Auxiliary equation: \(m^2 + 6m + 9 = 0\)

\((m+3)^2 = 0\)

\(m = -3, -3\) — Equal roots

\[ y = \text{C.F.} = \boxed{(C_1 + C_2 x) e^{-3x}} \]

Case III: Complex Conjugate Roots (m = α ± iβ)

When: Discriminant b² − 4ac < 0

Roots: \(m = \alpha \pm i\beta\) (complex conjugates)

where \(\alpha = -\frac{b}{2a}\) (real part) and \(\beta = \frac{\sqrt{4ac - b^2}}{2a}\) (imaginary part)

\[ \text{C.F.} = e^{\alpha x} (C_1 \cos\beta x + C_2 \sin\beta x) \]

Special case: If α = 0 (purely imaginary roots ±iβ):

\[ \text{C.F.} = C_1 \cos\beta x + C_2 \sin\beta x \]
Understanding Complex Roots

What is i? The imaginary unit i = √(−1), so i² = −1.

Complex conjugates: If one root is α + iβ, the other is always α − iβ.

Example: If m² + 4 = 0, then m² = −4, so m = ±√(−4) = ±2i = 0 ± 2i

Here α = 0, β = 2.

Example 4.23 — Case III (Complex Roots) ⭐⭐

Find the C.F. of: \(\frac{d^2y}{dx^2} + 4y = 0\)

Solution

Auxiliary equation: \(m^2 + 4 = 0\)

\(m^2 = -4\)

\(m = \pm 2i = 0 \pm 2i\)

Here α = 0, β = 2

Apply Case III formula:

\[ \text{C.F.} = e^{0 \cdot x}(C_1\cos 2x + C_2\sin 2x) = \boxed{C_1\cos 2x + C_2\sin 2x} \]
Example 4.24 — Case III (Complex Roots with Real Part) ⭐⭐

Find the C.F. of: \(y'' - 2y' + 5y = 0\)

Solution

Auxiliary equation: \(m^2 - 2m + 5 = 0\)

Discriminant: \(\Delta = 4 - 20 = -16 < 0\) → Complex roots

\(m = \frac{2 \pm \sqrt{-16}}{2} = \frac{2 \pm 4i}{2} = 1 \pm 2i\)

Here α = 1, β = 2

\[ \boxed{\text{C.F.} = e^{x}(C_1\cos 2x + C_2\sin 2x)} \]
Example 4.25 — Case III

Solve: \((D^2 + 2D + 10)y = 0\)

Solution

Auxiliary equation: \(m^2 + 2m + 10 = 0\)

\(m = \frac{-2 \pm \sqrt{4 - 40}}{2} = \frac{-2 \pm \sqrt{-36}}{2} = \frac{-2 \pm 6i}{2} = -1 \pm 3i\)

Here α = −1, β = 3

\[ y = \boxed{e^{-x}(C_1\cos 3x + C_2\sin 3x)} \]
Summary — C.F. for All Three Cases

Quick Tips for Finding C.F.
  1. Always factor the auxiliary equation first if possible
  2. If you can't factor, use the quadratic formula
  3. Check the discriminant to identify the case
  4. For Case III, carefully identify α (real part) and β (imaginary part)
  5. Don't forget the 'x' factor in Case II!

4.3.2 Particular Integral (P.I.) 🔥🔥🔥

Definition — Particular Integral

The Particular Integral (P.I.) is any one specific solution of the non-homogeneous equation:

\[ (aD^2 + bD + c)y = f(x) \]

It represents the effect of the forcing function f(x) on the system.

Using operator notation:

\[ \text{P.I.} = \frac{1}{aD^2 + bD + c} \cdot f(x) = \frac{1}{\phi(D)} \cdot f(x) \]

where \(\phi(D) = aD^2 + bD + c\).

Types of P.I. Based on f(x)

The method for finding P.I. depends on the form of f(x):

  1. Type I: \(f(x) = e^{ax}\) (exponential)
  2. Type II: \(f(x) = \sin(ax)\) or \(\cos(ax)\) (trigonometric)
  3. Type III: \(f(x) = x^n\) (polynomial)
  4. Type IV: \(f(x) = e^{ax} \cdot V(x)\) (exponential times another function)

Type I: P.I. when f(x) = eax

\[ \text{P.I.} = \frac{1}{\phi(D)} e^{ax} = \frac{e^{ax}}{\phi(a)} \quad \text{if } \phi(a) \neq 0 \]

Method: Replace D with 'a' in φ(D). If φ(a) ≠ 0, P.I. = eax/φ(a).

Failure case: If φ(a) = 0, multiply by x and differentiate the denominator:

\[ \text{P.I.} = \frac{x \cdot e^{ax}}{\phi'(a)} \quad \text{if } \phi(a) = 0 \text{ but } \phi'(a) \neq 0 \]

If φ'(a) = 0 also, multiply by x² and use φ''(a), and so on.

Example 4.26 — P.I. for eax (Normal Case) ⭐⭐

Find the P.I. of: \((D^2 - 5D + 6)y = e^{4x}\)

Solution

\(\text{P.I.} = \frac{1}{D^2 - 5D + 6} e^{4x}\)

Replace D with 4:

\(\text{P.I.} = \frac{e^{4x}}{4^2 - 5(4) + 6} = \frac{e^{4x}}{16 - 20 + 6} = \frac{e^{4x}}{2}\)

\[ \boxed{\text{P.I.} = \frac{e^{4x}}{2}} \]
Example 4.27 — P.I. for eax (Failure Case) ⭐⭐⭐

Find the P.I. of: \((D^2 - 5D + 6)y = e^{2x}\)

Solution

\(\text{P.I.} = \frac{1}{D^2 - 5D + 6} e^{2x}\)

Check: Replace D with 2: \(2^2 - 5(2) + 6 = 4 - 10 + 6 = 0\) ❌ Failure!

Use failure case method:

\(\phi(D) = D^2 - 5D + 6 = (D-2)(D-3)\)

\(\phi'(D) = 2D - 5\)

\(\phi'(2) = 2(2) - 5 = -1\)

\(\text{P.I.} = \frac{x \cdot e^{2x}}{\phi'(2)} = \frac{xe^{2x}}{-1} = -xe^{2x}\)

\[ \boxed{\text{P.I.} = -xe^{2x}} \]
Example 4.28 — P.I. for eax (Double Failure)

Find the P.I. of: \((D^2 - 4D + 4)y = e^{2x}\)

Solution

\(\phi(D) = D^2 - 4D + 4 = (D-2)^2\)

Check: \(\phi(2) = (2-2)^2 = 0\) ❌

\(\phi'(D) = 2(D-2)\), \(\phi'(2) = 0\) ❌ Double failure!

\(\phi''(D) = 2\), \(\phi''(2) = 2\) ✓

\(\text{P.I.} = \frac{x^2 \cdot e^{2x}}{\phi''(2)} = \frac{x^2 e^{2x}}{2}\)

\[ \boxed{\text{P.I.} = \frac{x^2 e^{2x}}{2}} \]

Type II: P.I. when f(x) = sin(ax) or cos(ax)

\[ \text{P.I.} = \frac{1}{\phi(D^2)} \sin(ax) = \frac{\sin(ax)}{\phi(-a^2)} \quad \text{if } \phi(-a^2) \neq 0 \]

Key Rule: Replace D² with −a² (NOT D with a!)

Similarly for cos(ax): Replace D² with −a².

What if there's a D term (odd power)?

  • If φ(D) = D² + pD + q, we can't directly replace D² only.
  • Method: Write φ(D) = (D² + q) + pD, replace D² = −a², then rationalize.

Failure case: If φ(−a²) = 0, multiply numerator by x and differentiate denominator w.r.t. D².

Example 4.29 — P.I. for sin(ax) (Normal Case) ⭐⭐

Find the P.I. of: \((D^2 + 4)y = \sin 3x\)

Solution

\(\text{P.I.} = \frac{1}{D^2 + 4} \sin 3x\)

Replace D² with −(3)² = −9:

\(\text{P.I.} = \frac{\sin 3x}{-9 + 4} = \frac{\sin 3x}{-5}\)

\[ \boxed{\text{P.I.} = -\frac{\sin 3x}{5}} \]
Example 4.30 — P.I. for cos(ax) ⭐

Find the P.I. of: \((D^2 + 9)y = \cos 2x\)

Solution

\(\text{P.I.} = \frac{1}{D^2 + 9} \cos 2x\)

Replace D² with −(2)² = −4:

\(\text{P.I.} = \frac{\cos 2x}{-4 + 9} = \frac{\cos 2x}{5}\)

\[ \boxed{\text{P.I.} = \frac{\cos 2x}{5}} \]
Example 4.31 — P.I. for sin(ax) (Failure Case) ⭐⭐⭐

Find the P.I. of: \((D^2 + 4)y = \sin 2x\)

Solution

\(\text{P.I.} = \frac{1}{D^2 + 4} \sin 2x\)

Replace D² with −(2)² = −4: \(\frac{\sin 2x}{-4+4} = \frac{\sin 2x}{0}\) ❌ Failure!

Use failure case method:

\(\text{P.I.} = \frac{x}{2D} \sin 2x = \frac{x}{2} \int \sin 2x\,dx = \frac{x}{2} \cdot \left(-\frac{\cos 2x}{2}\right)\)

\[ \boxed{\text{P.I.} = -\frac{x\cos 2x}{4}} \]
Example 4.32 — P.I. with D term (Rationalization) ⭐⭐⭐

Find the P.I. of: \((D^2 + 2D + 5)y = \sin 2x\)

Solution

\(\text{P.I.} = \frac{1}{D^2 + 2D + 5} \sin 2x\)

Replace D² = −4:

\(\text{P.I.} = \frac{1}{-4 + 2D + 5} \sin 2x = \frac{1}{1 + 2D} \sin 2x\)

Rationalize by multiplying by (1 − 2D)/(1 − 2D):

\(\text{P.I.} = \frac{1 - 2D}{(1+2D)(1-2D)} \sin 2x = \frac{1-2D}{1 - 4D^2} \sin 2x\)

Replace D² = −4 again: \(\frac{1-2D}{1-4(-4)} = \frac{1-2D}{17}\)

\(\text{P.I.} = \frac{1}{17}(1-2D)\sin 2x = \frac{1}{17}[\sin 2x - 2D(\sin 2x)]\)

\(= \frac{1}{17}[\sin 2x - 2 \cdot 2\cos 2x] = \frac{1}{17}[\sin 2x - 4\cos 2x]\)

\[ \boxed{\text{P.I.} = \frac{\sin 2x - 4\cos 2x}{17}} \]

Type III: P.I. when f(x) = xn (Polynomial)

When f(x) is a polynomial of degree n, we use binomial expansion:

\[ \text{P.I.} = \frac{1}{\phi(D)} x^n = \frac{1}{c(1 + \psi(D))} x^n = \frac{1}{c}[1 + \psi(D)]^{-1} x^n \]

Binomial expansion: \([1 + \psi(D)]^{-1} = 1 - \psi(D) + [\psi(D)]^2 - [\psi(D)]^3 + ...\)

Expand up to Dn terms only (since Dn+1xn = 0).

Useful Binomial Expansions
  • \((1+u)^{-1} = 1 - u + u^2 - u^3 + ...\)
  • \((1-u)^{-1} = 1 + u + u^2 + u^3 + ...\)
  • \((1+u)^{-2} = 1 - 2u + 3u^2 - 4u^3 + ...\)
Example 4.33 — P.I. for x² ⭐⭐

Find the P.I. of: \((D^2 + 3D + 2)y = x^2\)

Solution

\(\text{P.I.} = \frac{1}{D^2 + 3D + 2} x^2 = \frac{1}{2(1 + \frac{D^2+3D}{2})} x^2\)

Let \(u = \frac{D^2+3D}{2}\)

\(\text{P.I.} = \frac{1}{2}(1+u)^{-1}x^2 = \frac{1}{2}(1 - u + u^2 - ...)x^2\)

Since we have x², we need terms up to D²:

\(= \frac{1}{2}\left[1 - \frac{D^2+3D}{2} + \frac{9D^2}{4}\right]x^2\) (keeping D² terms only)

\(= \frac{1}{2}\left[1 - \frac{3D}{2} - \frac{D^2}{2} + \frac{9D^2}{4}\right]x^2\)

\(= \frac{1}{2}\left[1 - \frac{3D}{2} + \frac{7D^2}{4}\right]x^2\)

\(= \frac{1}{2}\left[x^2 - \frac{3}{2}(2x) + \frac{7}{4}(2)\right] = \frac{1}{2}\left[x^2 - 3x + \frac{7}{2}\right]\)

\[ \boxed{\text{P.I.} = \frac{x^2}{2} - \frac{3x}{2} + \frac{7}{4}} \]
Example 4.34 — P.I. for x (Simple Case) ⭐

Find the P.I. of: \((D^2 + 1)y = x\)

Solution

\(\text{P.I.} = \frac{1}{D^2 + 1} x = \frac{1}{1+D^2} x = (1+D^2)^{-1}x\)

\(= (1 - D^2 + D^4 - ...)x\)

For x (degree 1), we need terms up to D only, but D²x = 0:

\(= (1 - 0)x = x\)

\[ \boxed{\text{P.I.} = x} \]

Type IV: P.I. when f(x) = eax · V(x)

When f(x) is an exponential multiplied by another function V(x):

\[ \text{P.I.} = \frac{1}{\phi(D)} e^{ax} V(x) = e^{ax} \cdot \frac{1}{\phi(D+a)} V(x) \]

Rule: Take eax outside and replace D with (D + a) in φ(D).

Then evaluate \(\frac{1}{\phi(D+a)} V(x)\) based on the type of V(x).

Example 4.35 — P.I. for eax·x ⭐⭐⭐

Find the P.I. of: \((D^2 - 4D + 4)y = xe^{2x}\)

Solution

\(\text{P.I.} = \frac{1}{D^2-4D+4} xe^{2x} = \frac{1}{(D-2)^2} xe^{2x}\)

Take e2x outside, replace D with (D+2):

\(= e^{2x} \cdot \frac{1}{(D+2-2)^2} x = e^{2x} \cdot \frac{1}{D^2} x\)

\(= e^{2x} \cdot \frac{x^3}{6}\) (since \(\frac{1}{D^2}x = \int\int x\,dx\,dx = \frac{x^3}{6}\))

\[ \boxed{\text{P.I.} = \frac{x^3 e^{2x}}{6}} \]

Complete Solution Examples (CF + PI)

Example 4.36 — Complete Solution ⭐⭐⭐

Solve completely: \(\frac{d^2y}{dx^2} - 5\frac{dy}{dx} + 6y = e^{x}\)

Solution

Auxiliary equation: \(m^2 - 5m + 6 = 0\)

\((m-2)(m-3) = 0 \Rightarrow m = 2, 3\)

\(\text{C.F.} = C_1e^{2x} + C_2e^{3x}\)

\(\text{P.I.} = \frac{1}{D^2-5D+6} e^x\)

Replace D = 1: \(\frac{e^x}{1-5+6} = \frac{e^x}{2}\)

\[ y = \text{C.F.} + \text{P.I.} = \boxed{C_1e^{2x} + C_2e^{3x} + \frac{e^x}{2}} \]
Example 4.37 — Complete Solution with cos(ax) ⭐⭐⭐

Solve: \((D^2 + 4)y = \cos 3x\)

Solution

\(m^2 + 4 = 0 \Rightarrow m = \pm 2i\) (Complex: α = 0, β = 2)

\(\text{C.F.} = C_1\cos 2x + C_2\sin 2x\)

\(\text{P.I.} = \frac{1}{D^2+4}\cos 3x = \frac{\cos 3x}{-9+4} = -\frac{\cos 3x}{5}\)

\[ y = \boxed{C_1\cos 2x + C_2\sin 2x - \frac{\cos 3x}{5}} \]
Example 4.38 — Complete Solution (Equal Roots + Failure) ⭐⭐⭐

Solve: \(\frac{d^2y}{dx^2} - 6\frac{dy}{dx} + 9y = e^{3x}\)

Solution

\(m^2 - 6m + 9 = 0 \Rightarrow (m-3)^2 = 0 \Rightarrow m = 3, 3\)

Equal roots → \(\text{C.F.} = (C_1 + C_2x)e^{3x}\)

\(\text{P.I.} = \frac{1}{(D-3)^2} e^{3x}\)

Replace D = 3: \(\frac{e^{3x}}{(3-3)^2} = \frac{e^{3x}}{0}\) ❌ Failure!

Double failure (since (D-3)²): Use \(\frac{x^2 e^{3x}}{2!} = \frac{x^2 e^{3x}}{2}\)

\[ y = \boxed{(C_1 + C_2x)e^{3x} + \frac{x^2e^{3x}}{2}} \]

Q 1.

The order and degree of \(\frac{d^2y}{dx^2} + 3\left(\frac{dy}{dx}\right)^2 + y = 0\) are:

  • a. Order 2, Degree 1
  • b. Order 2, Degree 2
  • c. Order 1, Degree 2
  • d. Order 1, Degree 1
Answer: a. Order 2, Degree 1

Highest derivative is d²y/dx² → Order = 2. Power of d²y/dx² is 1 → Degree = 1. (The power 2 on dy/dx doesn't matter since d²y/dx² is the highest order derivative.)

Q 2.

The integrating factor for \(\frac{dy}{dx} + \frac{y}{x} = x^2\) is:

  • a. \(e^x\)
  • b. \(x\)
  • c. \(\frac{1}{x}\)
  • d. \(\ln x\)
Answer: b. x

P = 1/x, so I.F. = \(e^{\int \frac{1}{x}dx} = e^{\ln x} = x\)

Q 3.

The C.F. of \((D^2 + 4)y = 0\) is:

  • a. \(C_1e^{2x} + C_2e^{-2x}\)
  • b. \((C_1 + C_2x)e^{2x}\)
  • c. \(C_1\cos 2x + C_2\sin 2x\)
  • d. \(e^{2x}(C_1\cos x + C_2\sin x)\)
Answer: c. \(C_1\cos 2x + C_2\sin 2x\)

Auxiliary equation: m² + 4 = 0 → m = ±2i (complex roots with α = 0, β = 2)

Q 4.

The P.I. of \((D^2 + 1)y = e^{2x}\) is:

  • a. \(\frac{e^{2x}}{3}\)
  • b. \(\frac{e^{2x}}{5}\)
  • c. \(e^{2x}\)
  • d. \(\frac{e^{2x}}{4}\)
Answer: b. \(\frac{e^{2x}}{5}\)

P.I. = \(\frac{e^{2x}}{2^2 + 1} = \frac{e^{2x}}{5}\)

Q 5.

Which equation is exact?

  • a. \((2xy + 1)dx + x^2dy = 0\)
  • b. \((x + y)dx + xdy = 0\)
  • c. \(ydx + xdy = 0\)
  • d. Both a and c
Answer: d. Both a and c

For (a): ∂M/∂y = 2x, ∂N/∂x = 2x ✓
For (b): ∂M/∂y = 1, ∂N/∂x = 1 ✓ (Actually b is also exact!)
For (c): ∂M/∂y = 1, ∂N/∂x = 1 ✓
All three are exact, but if only a and c given as option, select d.

  • d. Not defined
Answer: b. 2

Squaring: \(1 + (dy/dx)^2 = x^2\), so degree = 2

Q 7.

The equation \(y'' + 6y' + 9y = 0\) has:

  • a. Two distinct real roots
  • b. Two equal real roots

Answer: b. Two equal real roots

m² + 6m + 9 = (m + 3)² = 0 → m = −3, −3 (repeated roots)

Q 9.

The equation \(\frac{dy}{dx} = \frac{x + y}{x - y}\) is of type:

Short & Long Answer Questions

Short Answer Questions (2–3 Marks)

Q1. Define ordinary differential equation with an example.

Answer: An ordinary differential equation (ODE) is an equation containing an unknown function of one independent variable and its derivatives.

Example: \(\frac{dy}{dx} + y = e^x\)

Here, y is the unknown function of x, and dy/dx is its derivative.

Q2. What is the order and degree of \(\left(\frac{d^2y}{dx^2}\right)^4 + \left(\frac{dy}{dx}\right)^3 = x\)?

Answer:

Highest order derivative present: \(\frac{d^2y}{dx^2}\) → Order = 2

Power of highest order derivative: 4 → Degree = 4

Q3. Write the condition for exactness of the equation \(Mdx + Ndy = 0\).

Answer: The equation \(Mdx + Ndy = 0\) is exact if and only if:

\[\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}\]

This means the partial derivative of M with respect to y equals the partial derivative of N with respect to x.

Q4. Find the integrating factor for \(\frac{dy}{dx} + 2y = e^x\).

Answer:

Comparing with standard form \(\frac{dy}{dx} + Py = Q\): P = 2

I.F. = \(e^{\int P\,dx} = e^{\int 2\,dx} = e^{2x}\)

Integrating Factor = \(e^{2x}\)

Q5. State the three cases of complementary function based on roots of auxiliary equation.

Answer:

  1. Case I (Distinct real roots m₁, m₂): C.F. = \(C_1e^{m_1x} + C_2e^{m_2x}\)
  2. Case II (Equal real roots m, m): C.F. = \((C_1 + C_2x)e^{mx}\)
  3. Case III (Complex roots α ± iβ): C.F. = \(e^{\alpha x}(C_1\cos\beta x + C_2\sin\beta x)\)

Q6. What substitution is used for solving homogeneous differential equations?

Answer: For solving homogeneous differential equations, we use the substitution:

\[y = vx\]

where v is a function of x. This gives \(\frac{dy}{dx} = v + x\frac{dv}{dx}\).

This substitution converts the homogeneous DE into a separable equation in v and x.

Q7. Write the formula for P.I. when f(x) = eax.

Answer: For \((aD^2 + bD + c)y = e^{ax}\), the P.I. is:

\[\text{P.I.} = \frac{1}{\phi(D)}e^{ax} = \frac{e^{ax}}{\phi(a)}\]

where we replace D with 'a' in φ(D).

Failure case: If φ(a) = 0, then P.I. = \(\frac{xe^{ax}}{\phi'(a)}\)

Long Answer Questions (5–7 Marks)

Q9. Solve the differential equation \(\frac{dy}{dx} + \frac{y}{x} = x^2\) using the method of integrating factor. [5 marks]

Solution:

Given: \(\frac{dy}{dx} + \frac{y}{x} = x^2\)

This is a linear DE of the form \(\frac{dy}{dx} + Py = Q\) where P = 1/x, Q = x².

Step 1: Find I.F.

I.F. = \(e^{\int \frac{1}{x}dx} = e^{\ln x} = x\)

Step 2: Solution formula

\(y \times \text{I.F.} = \int Q \times \text{I.F.}\,dx + C\)

\(y \cdot x = \int x^2 \cdot x\,dx + C\)

\(xy = \int x^3\,dx + C\)

\(xy = \frac{x^4}{4} + C\)

Answer: \(\boxed{y = \frac{x^3}{4} + \frac{C}{x}}\)

Q10. Solve: \((x^2 + y^2)dx - 2xy\,dy = 0\) [6 marks]

Solution:

Rewriting: \(\frac{dy}{dx} = \frac{x^2 + y^2}{2xy}\)

This is homogeneous (numerator and denominator both have degree 2).

Step 1: Put y = vx, so \(\frac{dy}{dx} = v + x\frac{dv}{dx}\)

\(v + x\frac{dv}{dx} = \frac{x^2 + v^2x^2}{2x \cdot vx} = \frac{1 + v^2}{2v}\)

Step 2: Separate

\(x\frac{dv}{dx} = \frac{1 + v^2}{2v} - v = \frac{1 + v^2 - 2v^2}{2v} = \frac{1 - v^2}{2v}\)

\(\frac{2v\,dv}{1 - v^2} = \frac{dx}{x}\)

Step 3: Integrate

\(-\ln|1 - v^2| = \ln|x| + C_1\)

\(\ln\frac{1}{|1-v^2|} = \ln|x| + C_1\)

\(\frac{1}{1-v^2} = Ax\) (where A = ±e^C₁)

Step 4: Replace v = y/x

\(\frac{1}{1 - y^2/x^2} = Ax\)

\(\frac{x^2}{x^2 - y^2} = Ax\)

Answer: \(\boxed{x^2 - y^2 = Cx}\) (where C = 1/A)

Q11. Solve completely: \(\frac{d^2y}{dx^2} - 5\frac{dy}{dx} + 6y = e^{2x}\) [7 marks]

Solution:

Given: \((D^2 - 5D + 6)y = e^{2x}\)

Part A: Find C.F.

Auxiliary equation: \(m^2 - 5m + 6 = 0\)

\((m - 2)(m - 3) = 0\)

m = 2, 3 (distinct real roots)

C.F. = \(C_1e^{2x} + C_2e^{3x}\)

Part B: Find P.I.

P.I. = \(\frac{1}{D^2 - 5D + 6}e^{2x}\)

Replace D = 2: \(\frac{e^{2x}}{4 - 10 + 6} = \frac{e^{2x}}{0}\) ❌ Failure!

φ(D) = (D - 2)(D - 3), φ'(D) = 2D - 5

φ'(2) = 4 - 5 = -1

P.I. = \(\frac{xe^{2x}}{\phi'(2)} = \frac{xe^{2x}}{-1} = -xe^{2x}\)

Complete Solution:

\(\boxed{y = C_1e^{2x} + C_2e^{3x} - xe^{2x}}\)

Q12. Solve: \((D^2 + 4)y = \cos 3x\) [6 marks]

Solution:

Part A: Find C.F.

Auxiliary equation: \(m^2 + 4 = 0\)

\(m^2 = -4\), m = ±2i

Complex roots: α = 0, β = 2

C.F. = \(C_1\cos 2x + C_2\sin 2x\)

Part B: Find P.I.

P.I. = \(\frac{1}{D^2 + 4}\cos 3x\)

Replace D² = −9:

P.I. = \(\frac{\cos 3x}{-9 + 4} = \frac{\cos 3x}{-5} = -\frac{\cos 3x}{5}\)

Complete Solution:

\(\boxed{y = C_1\cos 2x + C_2\sin 2x - \frac{\cos 3x}{5}}\)

Q13. Verify that (2xy + 3)dx + (x² + 4y)dy = 0 is exact and solve it. [5 marks]

Solution:

M = 2xy + 3, N = x² + 4y

Check exactness:

\(\frac{\partial M}{\partial y} = 2x\)

\(\frac{\partial N}{\partial x} = 2x\)

Since \(\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}\), the equation is exact. ✓

Solution:

\(\int M\,dx\) (treating y as constant) = \(\int (2xy + 3)dx = x^2y + 3x\)

Terms in N not containing x = 4y

\(\int 4y\,dy = 2y^2\)

Answer: \(\boxed{x^2y + 3x + 2y^2 = C}\)

Numerical Problems

Q14. Solve: \(\frac{dy}{dx} = xy\) [4 marks]

Solution: This is separable.

\(\frac{dy}{y} = x\,dx\)

Integrating: \(\ln|y| = \frac{x^2}{2} + C\)

\(|y| = e^{x^2/2 + C} = e^C \cdot e^{x^2/2}\)

Answer: \(\boxed{y = Ae^{x^2/2}}\) where A = ±e^C

Q15. Solve: \(y'' - 4y' + 4y = 0\) [4 marks]

Solution:

Auxiliary equation: \(m^2 - 4m + 4 = 0\)

\((m - 2)^2 = 0\)

m = 2, 2 (equal roots)

Answer: \(\boxed{y = (C_1 + C_2x)e^{2x}}\)

Q16. Find P.I. of \((D^2 + 1)y = \sin 2x\) [4 marks]

Solution:

P.I. = \(\frac{1}{D^2 + 1}\sin 2x\)

Replace D² = −4:

P.I. = \(\frac{\sin 2x}{-4 + 1} = \frac{\sin 2x}{-3}\)

Answer: \(\boxed{\text{P.I.} = -\frac{\sin 2x}{3}}\)

Q17. Solve: \(\frac{dy}{dx} - \frac{2y}{x} = x^3\) [5 marks]

Solution:

Linear DE with P = −2/x, Q = x³

I.F. = \(e^{\int -\frac{2}{x}dx} = e^{-2\ln x} = x^{-2} = \frac{1}{x^2}\)

\(\frac{y}{x^2} = \int x^3 \cdot \frac{1}{x^2}dx = \int x\,dx = \frac{x^2}{2} + C\)

Answer: \(\boxed{y = \frac{x^4}{2} + Cx^2}\)

Q18. Find C.F. and P.I. of \((D^2 + 9)y = e^{2x}\) [6 marks]

Solution:

C.F.: m² + 9 = 0 → m = ±3i

C.F. = \(C_1\cos 3x + C_2\sin 3x\)

P.I.: = \(\frac{e^{2x}}{4 + 9} = \frac{e^{2x}}{13}\)

Complete Solution: \(\boxed{y = C_1\cos 3x + C_2\sin 3x + \frac{e^{2x}}{13}}\)

Quick Revision — Unit Summary

Complete Unit Summary

📌 Basic Concepts

  • Order = Highest derivative present (always defined)
  • Degree = Power of highest derivative (after removing radicals; may be undefined)
  • General Solution = Contains arbitrary constants
  • Particular Solution = Specific values of constants given

📌 First Order Methods (Remember the Types!)

  • Separation of Variables: Separate x's and y's, integrate both sides
  • Homogeneous: Same degree → Put y = vx, then separate
  • Exact: Check ∂M/∂y = ∂N/∂x, then integrate M w.r.t. x + remaining N terms w.r.t. y
  • Linear: dy/dx + Py = Q → I.F. = e^(∫P dx) → y·I.F. = ∫Q·I.F. dx + C

📌 Second Order C.F. (Three Cases - MUST MEMORIZE!)

  • Case I: Distinct roots m₁, m₂ → C.F. = C₁e^(m₁x) + C₂e^(m₂x)
  • Case II: Equal roots m, m → C.F. = (C₁ + C₂x)e^(mx)
  • Case III: Complex roots α ± iβ → C.F. = e^(αx)(C₁cos βx + C₂sin βx)

📌 P.I. Formulas

  • e^(ax): Replace D = a; if 0, multiply by x and differentiate denom
  • sin(ax)/cos(ax): Replace D² = −a²; if 0, multiply by x
  • x^n: Binomial expansion of 1/φ(D)
  • e^(ax)·V(x): Take e^(ax) out, replace D → D+a

📌 Complete Solution

\[ y = \text{C.F.} + \text{P.I.} \]
Don't Forget in Exam!
  1. Always check type of equation first before solving
  2. For linear type: P is coefficient of y (after making coefficient of dy/dx = 1)
  3. For C.F.: Solve auxiliary equation correctly using quadratic formula if needed
  4. For P.I. with sin/cos: Replace D² (not D) with −a²
  5. Always add constant C in first order solutions
  6. For second order: Two constants C₁ and C₂ in C.F.
  7. Final answer = C.F. + P.I. for non-homogeneous equations
