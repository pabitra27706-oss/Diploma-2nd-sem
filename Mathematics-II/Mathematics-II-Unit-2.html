Unit Overview

What This Unit Covers

This unit teaches you how to represent geometric shapes — points, lines, circles, parabolas, and ellipses — using mathematical equations. This is the foundation of Coordinate Geometry (also called Analytical Geometry).

Real-world analogy: Think of GPS coordinates. Every location on Earth has a latitude and longitude — that's coordinate geometry! Similarly, every point on a flat surface (2D plane) can be described by two numbers (x, y).

Engineers use coordinate geometry for:

  • 🏗️ Designing roads, bridges, and buildings (civil engineering)
  • ⚡ Plotting electrical signal curves (ECE/EE)
  • 💻 Computer graphics and game development (CSE)
  • ⚙️ CNC machine programming and CAD/CAM (mechanical)

2.1.1 Cartesian & Polar Coordinate Systems

🔹 Cartesian Coordinate System

Definition — Cartesian Coordinates

In the Cartesian coordinate system (named after René Descartes), the position of any point P in a 2D plane is described by an ordered pair (x, y), where:

  • x = horizontal distance from the origin (called the abscissa)
  • y = vertical distance from the origin (called the ordinate)

The two perpendicular lines (axes) are called the X-axis (horizontal) and Y-axis (vertical). Their intersection point is the Origin O(0, 0).

🔹 Polar Coordinate System

Definition — Polar Coordinates

In the Polar coordinate system, the position of any point P is described by an ordered pair (r, θ), where:

  • r = distance of the point from the origin (called the radius vector or radial distance). Always ≥ 0.
  • θ (theta) = angle made by the line OP with the positive X-axis, measured anti-clockwise (called the vectorial angle). Measured in degrees or radians.

The origin O is called the pole, and the positive X-axis is called the initial line.

🔹 Relation Between Cartesian & Polar Coordinates 🔥

Conversion Formulas

From Polar (r, θ) to Cartesian (x, y):

\[ x = r\cos\theta \] \[ y = r\sin\theta \]

From Cartesian (x, y) to Polar (r, θ):

\[ r = \sqrt{x^2 + y^2} \] \[ \theta = \tan^{-1}\left(\frac{y}{x}\right) \]

Also note: \( x^2 + y^2 = r^2 \)

Easy Way to Remember

Think of a right triangle with the point P at the top:

  • r = hypotenuse (distance from O to P)
  • x = base (adjacent side) = \(r\cos\theta\)
  • y = height (opposite side) = \(r\sin\theta\)

This is just trigonometry in a right triangle!

Example 2.1 — Polar to Cartesian

Convert the polar coordinates \((4, 60°)\) to Cartesian coordinates.

Solution

Here \(r = 4\), \(\theta = 60°\)

\( x = r\cos\theta = 4\cos 60° = 4 \times \frac{1}{2} = 2 \)

\( y = r\sin\theta = 4\sin 60° = 4 \times \frac{\sqrt{3}}{2} = 2\sqrt{3} \)

\[ \text{Cartesian coordinates} = \boxed{(2, \ 2\sqrt{3})} \]
Example 2.2 — Cartesian to Polar

Convert the Cartesian coordinates \((1, 1)\) to Polar coordinates.

Solution

\( r = \sqrt{x^2 + y^2} = \sqrt{1^2 + 1^2} = \sqrt{2} \)

\( \theta = \tan^{-1}\left(\frac{y}{x}\right) = \tan^{-1}\left(\frac{1}{1}\right) = \tan^{-1}(1) = 45° \)

\[ \text{Polar coordinates} = \boxed{(\sqrt{2}, \ 45°)} \]
Example 2.3 — Cartesian to Polar (Second Quadrant)

Convert \((-1, \sqrt{3})\) to polar coordinates.

Solution

\( r = \sqrt{(-1)^2 + (\sqrt{3})^2} = \sqrt{1 + 3} = \sqrt{4} = 2 \)

\( \tan\theta = \frac{\sqrt{3}}{-1} = -\sqrt{3} \)

Since \(x < 0\) and \(y > 0\), the point is in the 2nd quadrant.

\( \theta = 180° - 60° = 120° \) (because \(\tan 60° = \sqrt{3}\))

\[ \text{Polar coordinates} = \boxed{(2, \ 120°)} \]
Common Mistake — Finding θ

When using \(\theta = \tan^{-1}(y/x)\), always check which quadrant the point is in!

  • Quadrant I: \(\theta\) as calculated
  • Quadrant II: \(\theta = 180° - \alpha\)
  • Quadrant III: \(\theta = 180° + \alpha\)
  • Quadrant IV: \(\theta = 360° - \alpha\)

where \(\alpha = \tan^{-1}|y/x|\) (always positive acute angle).

2.1.2 Distance Formula & Section Formula

🔹 Distance Between Two Points 🔥

Distance Formula

The distance between two points \(A(x_1, y_1)\) and \(B(x_2, y_2)\) in a 2D plane is:

\[ d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2} \]

This formula is derived from the Pythagoras theorem.

Special Cases
  • Distance from Origin: If one point is origin O(0,0), then \(d = \sqrt{x^2 + y^2}\)
  • Points on X-axis: If \(y_1 = y_2 = 0\), then \(d = |x_2 - x_1|\)
  • Points on Y-axis: If \(x_1 = x_2 = 0\), then \(d = |y_2 - y_1|\)
Example 2.4 — Distance Between Two Points

Find the distance between A(3, 4) and B(−1, 7).

Solution

\( d = \sqrt{(-1-3)^2 + (7-4)^2} = \sqrt{(-4)^2 + (3)^2} = \sqrt{16 + 9} = \sqrt{25} \)

\[ d = \boxed{5 \text{ units}} \]
Example 2.5 — Distance from Origin

Find the distance of the point P(5, 12) from the origin.

Solution

\( d = \sqrt{5^2 + 12^2} = \sqrt{25 + 144} = \sqrt{169} \)

\[ d = \boxed{13 \text{ units}} \]

🔹 Section Formula (Division of a Line Segment) 🔥🔥

Internal Division

If a point P(x, y) divides the line segment joining \(A(x_1, y_1)\) and \(B(x_2, y_2)\) internally in the ratio \(m : n\), then:

\[ x = \frac{mx_2 + nx_1}{m + n}, \quad y = \frac{my_2 + ny_1}{m + n} \]

"Internally" means the point P lies between A and B on the line segment.

External Division

If a point P(x, y) divides the line segment joining \(A(x_1, y_1)\) and \(B(x_2, y_2)\) externally in the ratio \(m : n\), then:

\[ x = \frac{mx_2 - nx_1}{m - n}, \quad y = \frac{my_2 - ny_1}{m - n} \]

"Externally" means the point P lies outside the line segment AB (on its extension).

Memory Trick — Section Formula

Internal: "Cross multiply, add, divide by total" → \(\frac{mx_2 + nx_1}{m+n}\)

External: Same but with minus signs → \(\frac{mx_2 - nx_1}{m-n}\)

Remember: The ratio point closer to B means m relates to B's coordinates → \(m \times x_2\).

Mid-Point Formula (Special Case)

When \(m = n = 1\) (equal division), P is the midpoint:

\[ \text{Midpoint} = \left(\frac{x_1 + x_2}{2}, \ \frac{y_1 + y_2}{2}\right) \]

This is just the average of x-coordinates and average of y-coordinates!

Example 2.6 — Internal Division ⭐

Find the coordinates of the point that divides the line segment joining A(2, 3) and B(8, 9) internally in the ratio 2:1.

Solution

Here \(m = 2, n = 1, (x_1,y_1) = (2,3), (x_2,y_2) = (8,9)\)

\( x = \frac{2(8) + 1(2)}{2+1} = \frac{16+2}{3} = \frac{18}{3} = 6 \)

\( y = \frac{2(9) + 1(3)}{2+1} = \frac{18+3}{3} = \frac{21}{3} = 7 \)

\[ P = \boxed{(6, 7)} \]
Example 2.7 — External Division

Find the coordinates of the point that divides the line joining A(1, 2) and B(4, 5) externally in the ratio 3:2.

Solution

External division: \(m = 3, n = 2\)

\( x = \frac{3(4) - 2(1)}{3-2} = \frac{12-2}{1} = 10 \)

\( y = \frac{3(5) - 2(2)}{3-2} = \frac{15-4}{1} = 11 \)

\[ P = \boxed{(10, 11)} \]
Example 2.8 — Finding Ratio When Point is Given

In what ratio does the point (3, 5) divide the line joining (1, 3) and (5, 7)?

Solution

Let the ratio be \(m : n\). Using x-coordinate:

\( 3 = \frac{m(5) + n(1)}{m + n} \)

\( 3(m + n) = 5m + n \)

\( 3m + 3n = 5m + n \)

\( 2n = 2m \Rightarrow m = n \)

\[ \text{Ratio} = \boxed{1 : 1} \quad \text{(i.e., (3, 5) is the midpoint)} \]

Verify with y: \(\frac{1(7)+1(3)}{1+1} = \frac{10}{2} = 5\) ✓

Example 2.9 — Midpoint

Find the midpoint of the line segment joining P(−3, 5) and Q(7, −1).

Solution

\( \text{Midpoint} = \left(\frac{-3+7}{2}, \frac{5+(-1)}{2}\right) = \left(\frac{4}{2}, \frac{4}{2}\right) \)

\[ \text{Midpoint} = \boxed{(2, 2)} \]

2.1.3 Area of Triangle & Condition of Collinearity

🔹 Area of a Triangle Given Three Vertices 🔥

Area of Triangle

If the three vertices of a triangle are \(A(x_1, y_1)\), \(B(x_2, y_2)\), and \(C(x_3, y_3)\), then:

\[ \text{Area} = \frac{1}{2} |x_1(y_2 - y_3) + x_2(y_3 - y_1) + x_3(y_1 - y_2)| \]

The absolute value | | is taken because area is always positive.

Alternative — Using Determinant

The same formula can be written using a determinant:

\[ \text{Area} = \frac{1}{2} \left| \begin{vmatrix} x_1 & y_1 & 1 \\ x_2 & y_2 & 1 \\ x_3 & y_3 & 1 \end{vmatrix} \right| \]

This connects Unit 1 (Determinants) with Unit 2 (Coordinate Geometry)!

Example 2.10 — Area of Triangle ⭐

Find the area of the triangle whose vertices are A(1, 2), B(4, 6), and C(7, 2).

Solution

\( \text{Area} = \frac{1}{2} |x_1(y_2 - y_3) + x_2(y_3 - y_1) + x_3(y_1 - y_2)| \)

\( = \frac{1}{2} |1(6-2) + 4(2-2) + 7(2-6)| \)

\( = \frac{1}{2} |1(4) + 4(0) + 7(-4)| \)

\( = \frac{1}{2} |4 + 0 - 28| = \frac{1}{2} |-24| = \frac{1}{2} \times 24 \)

\[ \text{Area} = \boxed{12 \text{ sq. units}} \]
Example 2.11 — Area of Triangle (Another Example)

Find the area of the triangle with vertices (2, 3), (−1, 0), (2, −4).

Solution

\( = \frac{1}{2} |2(0-(-4)) + (-1)((-4)-3) + 2(3-0)| \)

\( = \frac{1}{2} |2(4) + (-1)(-7) + 2(3)| \)

\( = \frac{1}{2} |8 + 7 + 6| = \frac{1}{2} \times 21 \)

\[ \text{Area} = \boxed{10.5 \text{ sq. units}} \]

🔹 Condition of Collinearity ⭐

Definition — Collinear Points

Three points are said to be collinear if they all lie on the same straight line. If three points are collinear, they cannot form a triangle — so the area of the "triangle" is zero.

Condition for Collinearity

Three points \(A(x_1, y_1)\), \(B(x_2, y_2)\), \(C(x_3, y_3)\) are collinear if and only if:

\[ x_1(y_2 - y_3) + x_2(y_3 - y_1) + x_3(y_1 - y_2) = 0 \]

Equivalently: Area of triangle = 0.

Example 2.12 — Prove Collinearity

Show that the points (1, 1), (2, 3), and (4, 7) are collinear.

Solution

\( = x_1(y_2 - y_3) + x_2(y_3 - y_1) + x_3(y_1 - y_2) \)

\( = 1(3 - 7) + 2(7 - 1) + 4(1 - 3) \)

\( = 1(-4) + 2(6) + 4(-2) \)

\( = -4 + 12 - 8 = 0 \)

Since the result is 0, the points are collinear. ✓ Proved.

Example 2.13 — Finding Unknown for Collinearity

Find the value of k if the points (1, 2), (3, k), and (5, 8) are collinear.

Solution

For collinearity: \( x_1(y_2 - y_3) + x_2(y_3 - y_1) + x_3(y_1 - y_2) = 0 \)

\( 1(k - 8) + 3(8 - 2) + 5(2 - k) = 0 \)

\( k - 8 + 18 + 10 - 5k = 0 \)

\( -4k + 20 = 0 \)

\( 4k = 20 \)

\[ k = \boxed{5} \]

2.2.1 Gradient (Slope) of a Straight Line

Definition — Slope (Gradient)

The slope (or gradient) of a straight line is a measure of its steepness and direction. It tells us how much the line rises (or falls) for every unit of horizontal movement.

The slope is usually denoted by m.

Slope Formulas

Using two points: If a line passes through \(A(x_1, y_1)\) and \(B(x_2, y_2)\):

\[ m = \frac{y_2 - y_1}{x_2 - x_1} = \frac{\text{Rise}}{\text{Run}} = \frac{\text{Change in y}}{\text{Change in x}} \]

Using angle: If the line makes angle \(\theta\) with the positive X-axis:

\[ m = \tan\theta \]

Find the slope of the line passing through A(2, 3) and B(6, 11).

Solution

\( m = \frac{y_2 - y_1}{x_2 - x_1} = \frac{11 - 3}{6 - 2} = \frac{8}{4} \)

2.2.2 Equations of Straight Line in Standard Forms 🔥

What is the Equation of a Line?

An equation of a straight line is a mathematical relationship between x and y that is satisfied by every point on the line and by no point outside the line.

The general form of any straight line equation is: \(ax + by + c = 0\)

The syllabus covers 4 standard forms (no proof required). Let's learn each one:

📏 Form 1: Slope-Intercept Form ⭐

\[ y = mx + c \]

Where:

  • m = slope of the line
  • c = y-intercept (the y-coordinate where the line crosses the Y-axis)

When to use: When you know the slope and the y-intercept.

Example 2.16

Find the equation of a line with slope 3 and y-intercept 5.

Solution

Using \(y = mx + c\) with \(m = 3, c = 5\):

\[ \boxed{y = 3x + 5} \]

📏 Form 2: Slope-Point Form (Point-Slope Form) ⭐⭐

\[ y - y_1 = m(x - x_1) \]

Where:

  • m = slope of the line
  • (x₁, y₁) = a point on the line

When to use: When you know the slope and one point on the line. This is the most versatile form.

Example 2.17

Find the equation of a line passing through (2, 3) with slope 4.

Solution

\( y - 3 = 4(x - 2) \)

\( y - 3 = 4x - 8 \)

\[ \boxed{y = 4x - 5} \quad \text{or} \quad 4x - y - 5 = 0 \]

📏 Form 3: Two-Point Form ⭐

\[ \frac{y - y_1}{y_2 - y_1} = \frac{x - x_1}{x_2 - x_1} \]

Or equivalently:

\[ y - y_1 = \frac{y_2 - y_1}{x_2 - x_1}(x - x_1) \]

When to use: When you know two points on the line (but not the slope directly).

Example 2.18

Find the equation of the line through A(1, 2) and B(3, 8).

Solution

\( \frac{y - 2}{8 - 2} = \frac{x - 1}{3 - 1} \)

\( \frac{y - 2}{6} = \frac{x - 1}{2} \)

Cross multiply: \( 2(y - 2) = 6(x - 1) \)

\( 2y - 4 = 6x - 6 \)

\( 6x - 2y - 2 = 0 \) or dividing by 2:

\[ \boxed{3x - y - 1 = 0} \quad \text{or} \quad y = 3x - 1 \]

📏 Form 4: Intercept Form ⭐

\[ \frac{x}{a} + \frac{y}{b} = 1 \]

Where:

  • a = x-intercept (the x-coordinate where the line crosses the X-axis)
  • b = y-intercept (the y-coordinate where the line crosses the Y-axis)

When to use: When you know both intercepts.

Example 2.19

Find the equation of the line that makes intercepts 3 on the X-axis and 4 on the Y-axis.

Solution

Using intercept form with \(a = 3, b = 4\):

\( \frac{x}{3} + \frac{y}{4} = 1 \)

Multiply through by 12:

\[ \boxed{4x + 3y = 12} \quad \text{or} \quad 4x + 3y - 12 = 0 \]
Example 2.20 — Finding Equation Given Conditions

Find the equation of the line passing through (1, −2) and having slope −3.

Solution

Using point-slope form: \( y - (-2) = -3(x - 1) \)

\( y + 2 = -3x + 3 \)

\[ \boxed{3x + y - 1 = 0} \]
Example 2.21 — Convert to Slope-Intercept Form

Convert the equation \(2x + 3y = 12\) to slope-intercept form. Find the slope and y-intercept.

Solution

Solve for y: \( 3y = -2x + 12 \)

\( y = -\frac{2}{3}x + 4 \)

Comparing with \(y = mx + c\):

\[ \text{Slope } m = \boxed{-\frac{2}{3}}, \quad \text{y-intercept } c = \boxed{4} \]

2.2.3 Angle Between Two Lines — Parallel & Perpendicular 🔥

🔹 Angle Between Two Straight Lines

Angle Formula

If two lines have slopes \(m_1\) and \(m_2\), then the acute angle θ between them is given by:

\[ \tan\theta = \left| \frac{m_1 - m_2}{1 + m_1 m_2} \right| \]

The absolute value ensures we get the acute angle (0° < θ < 90°).

Example 2.22 — Angle Between Two Lines

Find the acute angle between the lines \(y = 2x + 3\) and \(y = 3x - 1\).

Solution

\(m_1 = 2\), \(m_2 = 3\)

\( \tan\theta = \left|\frac{2 - 3}{1 + (2)(3)}\right| = \left|\frac{-1}{7}\right| = \frac{1}{7} \)

\[ \theta = \tan^{-1}\left(\frac{1}{7}\right) = \boxed{8.13°} \approx 8°8' \]

🔹 Condition for Parallel Lines ⭐

Parallel Lines

Two lines are parallel if and only if they have the same slope.

\[ \text{Parallel} \iff m_1 = m_2 \]

When lines are parallel, \(\theta = 0°\), so \(\tan\theta = 0\), which gives \(m_1 - m_2 = 0\), i.e., \(m_1 = m_2\).

🔹 Condition for Perpendicular Lines ⭐

Perpendicular Lines

Two lines are perpendicular if and only if the product of their slopes is −1.

\[ \text{Perpendicular} \iff m_1 \times m_2 = -1 \]

When lines are perpendicular, \(\theta = 90°\), so \(\tan\theta = \infty\), which means the denominator \(1 + m_1 m_2 = 0\), giving \(m_1 m_2 = -1\).

Key Point to Remember

Parallel → Same slope: \(m_1 = m_2\)

Perpendicular → Product of slopes = −1: \(m_1 \cdot m_2 = -1\) (slopes are negative reciprocals)

Example: If one line has slope \(\frac{2}{3}\), a perpendicular line has slope \(-\frac{3}{2}\).

Example 2.23 — Check Parallel or Perpendicular

Check whether the lines \(3x + 4y = 7\) and \(6x + 8y = 15\) are parallel, perpendicular, or neither.

Solution

Convert to slope-intercept form:

Line 1: \(4y = -3x + 7 \Rightarrow y = -\frac{3}{4}x + \frac{7}{4}\), so \(m_1 = -\frac{3}{4}\)

Line 2: \(8y = -6x + 15 \Rightarrow y = -\frac{6}{8}x + \frac{15}{8} = -\frac{3}{4}x + \frac{15}{8}\), so \(m_2 = -\frac{3}{4}\)

Since \(m_1 = m_2 = -\frac{3}{4}\), the lines are parallel. ✓

Example 2.24 — Perpendicular Lines

Find the equation of the line perpendicular to \(2x - 3y + 5 = 0\) and passing through (1, 2).

Solution

Find slope of given line: \(3y = 2x + 5 \Rightarrow y = \frac{2}{3}x + \frac{5}{3}\), so \(m_1 = \frac{2}{3}\)

For perpendicular line: \(m_2 = -\frac{1}{m_1} = -\frac{3}{2}\)

Using point-slope form with point (1, 2) and slope \(-\frac{3}{2}\):

\( y - 2 = -\frac{3}{2}(x - 1) \)

\( 2(y - 2) = -3(x - 1) \)

\( 2y - 4 = -3x + 3 \)

\[ \boxed{3x + 2y - 7 = 0} \]
Example 2.25 — Equation of Parallel Line

Find the equation of the line parallel to \(y = 2x + 1\) and passing through (3, −4).

Solution

Given line slope: \(m = 2\). Parallel line also has slope = 2.

Using point-slope form: \( y - (-4) = 2(x - 3) \)

\( y + 4 = 2x - 6 \)

\[ \boxed{y = 2x - 10} \quad \text{or} \quad 2x - y - 10 = 0 \]

2.2.4 Perpendicular Distance 🔥

🔹 Perpendicular Distance from a Point to a Line

Distance from Point to Line

The perpendicular distance from a point \(P(x_1, y_1)\) to the line \(ax + by + c = 0\) is:

\[ d = \frac{|ax_1 + by_1 + c|}{\sqrt{a^2 + b^2}} \]

Important: The line must be in the form \(ax + by + c = 0\) (everything on one side).

Example 2.26 — Distance from Point to Line ⭐

Find the perpendicular distance from the point (3, 4) to the line \(3x + 4y - 5 = 0\).

Solution

Here \(a = 3, b = 4, c = -5, x_1 = 3, y_1 = 4\)

\( d = \frac{|3(3) + 4(4) + (-5)|}{\sqrt{3^2 + 4^2}} = \frac{|9 + 16 - 5|}{\sqrt{9 + 16}} = \frac{|20|}{\sqrt{25}} = \frac{20}{5} \)

\[ d = \boxed{4 \text{ units}} \]
Example 2.27 — Distance from Origin to a Line

Find the perpendicular distance from the origin to the line \(5x + 12y = 26\).

Solution

Rewrite: \(5x + 12y - 26 = 0\). Point is origin (0, 0).

\( d = \frac{|5(0) + 12(0) - 26|}{\sqrt{5^2 + 12^2}} = \frac{|-26|}{\sqrt{25 + 144}} = \frac{26}{\sqrt{169}} = \frac{26}{13} \)

\[ d = \boxed{2 \text{ units}} \]

🔹 Distance Between Two Parallel Lines ⭐

Distance Between Parallel Lines

If two parallel lines are \(ax + by + c_1 = 0\) and \(ax + by + c_2 = 0\) (same coefficients of x and y), then the distance between them is:

\[ d = \frac{|c_1 - c_2|}{\sqrt{a^2 + b^2}} \]

Note: Both lines must have the same coefficients of x and y. If not, multiply one equation to make them equal.

Example 2.28 — Distance Between Parallel Lines ⭐

Find the distance between the parallel lines \(3x + 4y - 7 = 0\) and \(3x + 4y + 8 = 0\).

Solution

Here \(a = 3, b = 4, c_1 = -7, c_2 = 8\)

\( d = \frac{|-7 - 8|}{\sqrt{3^2 + 4^2}} = \frac{|-15|}{\sqrt{25}} = \frac{15}{5} \)

\[ d = \boxed{3 \text{ units}} \]
Example 2.29 — Distance Between Parallel Lines (Need Adjustment)

Find the distance between \(2x + 3y = 6\) and \(4x + 6y = 18\).

Solution

Divide second equation by 2: \(2x + 3y = 9\)

Now both lines have same coefficients: \(2x + 3y - 6 = 0\) and \(2x + 3y - 9 = 0\)

\( d = \frac{|-6 - (-9)|}{\sqrt{4 + 9}} = \frac{|3|}{\sqrt{13}} = \frac{3}{\sqrt{13}} = \frac{3\sqrt{13}}{13} \)

\[ d = \boxed{\frac{3}{\sqrt{13}} \approx 0.832 \text{ units}} \]
Common Mistake

When using the parallel lines formula, make sure both equations have exactly the same coefficients of x and y. If the second line is \(6x + 8y + 10 = 0\) and the first is \(3x + 4y + 5 = 0\), you must divide the second by 2 first!

2.3 Circle — Equations & Problems 🔥

Definition — Circle

A circle is the set (locus) of all points in a plane that are at a fixed distance (called the radius) from a fixed point (called the centre).

Real-world analogy: Think of a compass drawing a circle — the needle stays at the centre, and the pencil traces all points at a fixed distance (radius) from it.

🔹 2.3.1 Equation of Circle — Centre-Radius Form ⭐⭐

Centre-Radius Form

If the centre of a circle is \(C(h, k)\) and the radius is \(r\), then the equation is:

\[ (x - h)^2 + (y - k)^2 = r^2 \]

Special Case — Centre at Origin: If centre is \(O(0, 0)\):

\[ x^2 + y^2 = r^2 \]
How to Derive This

Any point \(P(x, y)\) on the circle is at distance \(r\) from centre \(C(h, k)\):

\( \sqrt{(x-h)^2 + (y-k)^2} = r \)

Squaring both sides gives the equation. It's just the distance formula set equal to the radius!

Example 2.30 — Centre-Radius Form

Find the equation of the circle with centre (3, −2) and radius 5.

Solution

\( (x - 3)^2 + (y - (-2))^2 = 5^2 \)

\( (x - 3)^2 + (y + 2)^2 = 25 \)

Expanding: \( x^2 - 6x + 9 + y^2 + 4y + 4 = 25 \)

\[ \boxed{x^2 + y^2 - 6x + 4y - 12 = 0} \]
Example 2.31 — Circle with Centre at Origin

Find the equation of the circle with centre at origin and radius 7.

Solution
\[ \boxed{x^2 + y^2 = 49} \]
Example 2.32 — Circle Passing Through a Point

Find the equation of the circle with centre (1, 2) passing through the point (4, 6).

Solution

Find radius = distance from centre to the point on the circle:

\( r = \sqrt{(4-1)^2 + (6-2)^2} = \sqrt{9 + 16} = \sqrt{25} = 5 \)

Write the equation: \( (x-1)^2 + (y-2)^2 = 25 \)

Expanding: \( x^2 - 2x + 1 + y^2 - 4y + 4 = 25 \)

\[ \boxed{x^2 + y^2 - 2x - 4y - 20 = 0} \]

🔹 Equation of Circle — Diameter Form ⭐

Diameter Form

If \(A(x_1, y_1)\) and \(B(x_2, y_2)\) are the end points of a diameter of a circle, then the equation of the circle is:

\[ (x - x_1)(x - x_2) + (y - y_1)(y - y_2) = 0 \]
Why This Works

The angle subtended by a diameter at any point on the circle is always 90°. This formula uses the property that if P(x, y) is on the circle and AB is a diameter, then PA ⊥ PB (slopes multiply to −1), which leads to the above equation.

Example 2.33 — Diameter Form ⭐

Find the equation of the circle on the line segment joining (1, 2) and (3, 4) as diameter.

Solution

Using diameter form with \((x_1,y_1)=(1,2)\) and \((x_2,y_2)=(3,4)\):

\( (x - 1)(x - 3) + (y - 2)(y - 4) = 0 \)

\( x^2 - 4x + 3 + y^2 - 6y + 8 = 0 \)

\[ \boxed{x^2 + y^2 - 4x - 6y + 11 = 0} \]

Verify: Centre = midpoint of diameter = \(\left(\frac{1+3}{2}, \frac{2+4}{2}\right) = (2, 3)\)

Radius = half the diameter length = \(\frac{1}{2}\sqrt{(3-1)^2+(4-2)^2} = \frac{1}{2}\sqrt{8} = \sqrt{2}\)

🔹 2.3.2 General Equation of a Circle 🔥🔥

General Equation

The general equation of a circle is:

\[ x^2 + y^2 + 2gx + 2fy + c = 0 \]

where:

  • Centre = \((-g, -f)\)
  • Radius = \(\sqrt{g^2 + f^2 - c}\)

Condition for this to represent a real circle: \(g^2 + f^2 - c > 0\)

How to Identify a Circle Equation

An equation of the form \(ax^2 + ay^2 + bx + cy + d = 0\) represents a circle if:

  1. Coefficients of \(x^2\) and \(y^2\) are equal (both = a)
  2. There is no xy term

If the coefficient of \(x^2\) and \(y^2\) is not 1, divide the entire equation by that coefficient first.

How to Find Centre & Radius from General Equation

Make sure the coefficients of \(x^2\) and \(y^2\) are both 1 (divide if needed).

Compare with \(x^2 + y^2 + 2gx + 2fy + c = 0\) to find g, f, and c.

Remember: coefficient of x = 2g, coefficient of y = 2f.

Centre = \((-g, -f)\)

Radius = \(\sqrt{g^2 + f^2 - c}\)

Example 2.34 — Finding Centre & Radius ⭐⭐

Find the centre and radius of the circle: \(x^2 + y^2 - 4x + 6y - 12 = 0\)

Solution

Comparing with \(x^2 + y^2 + 2gx + 2fy + c = 0\):

\( 2g = -4 \Rightarrow g = -2 \)

\( 2f = 6 \Rightarrow f = 3 \)

\( c = -12 \)

Centre = \((-g, -f) = (2, -3)\)

Radius = \(\sqrt{g^2 + f^2 - c} = \sqrt{(-2)^2 + 3^2 - (-12)} = \sqrt{4 + 9 + 12} = \sqrt{25}\)

\[ \text{Centre} = \boxed{(2, -3)}, \quad \text{Radius} = \boxed{5} \]
Example 2.35 — Another Centre-Radius Problem

Find the centre and radius of: \(x^2 + y^2 + 8x - 10y + 5 = 0\)

Solution

\( 2g = 8 \Rightarrow g = 4, \quad 2f = -10 \Rightarrow f = -5, \quad c = 5 \)

Centre = \((-4, 5)\)

Radius = \(\sqrt{16 + 25 - 5} = \sqrt{36} = 6\)

\[ \text{Centre} = \boxed{(-4, 5)}, \quad \text{Radius} = \boxed{6} \]
Example 2.36 — Circle with Coefficient ≠ 1

Find the centre and radius of: \(3x^2 + 3y^2 - 12x + 18y + 9 = 0\)

Solution

Divide throughout by 3: \(x^2 + y^2 - 4x + 6y + 3 = 0\)

\( 2g = -4 \Rightarrow g = -2, \quad 2f = 6 \Rightarrow f = 3, \quad c = 3 \)

Centre = \((2, -3)\)

Radius = \(\sqrt{4 + 9 - 3} = \sqrt{10}\)

\[ \text{Centre} = \boxed{(2, -3)}, \quad \text{Radius} = \boxed{\sqrt{10} \approx 3.16} \]
Common Mistakes in Circle Problems
  1. Forgetting the negative signs: Centre is \((-g, -f)\), NOT \((g, f)\)! If \(2g = -4\), then \(g = -2\), so centre x = \(-(-2) = 2\).
  2. Wrong c value: The constant term c in the general equation is the standalone constant after \(x^2 + y^2 + 2gx + 2fy\).
  3. Not dividing first: If coefficients of \(x^2\) and \(y^2\) are not 1, you MUST divide first.
  4. Negative under square root: If \(g^2 + f^2 - c < 0\), it's NOT a real circle (imaginary circle).

2.4.1 Conic Sections — Definitions & Terms 🔥

Definition — Conic Section

A conic section (or simply "conic") is the curve obtained when a right circular cone is cut by a plane at different angles.

Alternatively, a conic is the locus (set of all positions) of a point that moves such that the ratio of its distance from a fixed point (focus) to its distance from a fixed line (directrix) is constant. This constant ratio is called the eccentricity (e).

🔹 Important Terms of Conic Sections ⭐⭐

📌 Focus (F)

The fixed point from which distances are measured. A conic may have one focus (parabola) or two foci (ellipse, hyperbola).

📌 Directrix

The fixed straight line from which distances are measured. It is perpendicular to the axis of the conic.

📌 Eccentricity (e)

The constant ratio of the distance of any point on the conic from the focus to its distance from the directrix.

\( e = \frac{\text{Distance from Focus}}{\text{Distance from Directrix}} \)

📌 Axis

The straight line passing through the focus and perpendicular to the directrix. It is the line of symmetry of the conic.

📌 Vertex (V)

The point where the conic crosses its axis. It is the point on the conic nearest to the directrix. A parabola has 1 vertex; an ellipse has 2 vertices.

📌 Latus Rectum (LR)

A chord of the conic passing through the focus and perpendicular to the axis. Its length is important for determining the "width" of the conic at the focus.

Eccentricity Formula

For any point P on the conic, if S is the focus and PM is the perpendicular distance from P to the directrix:

\[ e = \frac{PS}{PM} \]

This ratio is constant for all points on the conic.

Example 2.37 — Identifying Conic Type

A conic has eccentricity \(e = \frac{1}{2}\). What type of conic is it?

Solution

Since \(0 < e = \frac{1}{2} < 1\), the conic is an Ellipse. ✓

Example 2.38 — Eccentricity Identification

Identify the conic for: (i) e = 1 (ii) e = 0 (iii) e = 2 (iv) e = 0.6

Solution

(i) e = 1 → Parabola

(ii) e = 0 → Circle

(iii) e = 2 > 1 → Hyperbola

(iv) e = 0.6 (between 0 and 1) → Ellipse

Definition — Parabola

A parabola is the locus of a point that moves such that its distance from a fixed point (focus) is equal to its distance from a fixed straight line (directrix). Its eccentricity e = 1.

Standard Equation of Parabola

  • Axis: Y = 0 (the X-axis)
  • Latus Rectum (length): \(4a\)

🔹 ELLIPSE ⭐⭐⭐

Definition — Ellipse

An ellipse is the locus of a point that moves such that the ratio of its distance from a fixed point (focus) to its distance from a fixed line (directrix) is a constant less than 1. Its eccentricity \(0 < e < 1\).

Alternative definition: An ellipse is the locus of a point such that the sum of its distances from two fixed points (foci) is constant.

Standard Equation of Ellipse

Case 1: Major axis along X-axis (horizontal ellipse), \(a > b\):

\[ \frac{x^2}{a^2} + \frac{y^2}{b^2} = 1 \quad (a > b > 0) \]

Case 2: Major axis along Y-axis (vertical ellipse), \(b > a\):

\[ \frac{x^2}{a^2} + \frac{y^2}{b^2} = 1 \quad (b > a > 0) \]

(Same equation form, but when \(b > a\), the major axis is vertical.)

Example 2.43 — Find Elements of an Ellipse ⭐⭐

Find the foci, vertices, eccentricity, and length of latus rectum of: \(\frac{x^2}{25} + \frac{y^2}{9} = 1\)

Solution

Here \(a^2 = 25 \Rightarrow a = 5\) and \(b^2 = 9 \Rightarrow b = 3\)

Since \(a > b\), the major axis is along the X-axis.

\( c = \sqrt{a^2 - b^2} = \sqrt{25 - 9} = \sqrt{16} = 4 \)

\[ \text{Vertices} = \boxed{(\pm 5, 0)}, \quad \text{Foci} = \boxed{(\pm 4, 0)} \] \[ \text{Eccentricity } e = \frac{c}{a} = \frac{4}{5} = \boxed{0.8} \] \[ \text{LR} = \frac{2b^2}{a} = \frac{2(9)}{5} = \boxed{\frac{18}{5} = 3.6 \text{ units}} \]
Example 2.44 — Ellipse with Major Axis Along Y-axis

Find the foci, eccentricity, and LR of: \(\frac{x^2}{9} + \frac{y^2}{16} = 1\)

Solution

Here denominator under \(y^2\) (16) > denominator under \(x^2\) (9).

So major axis is along Y-axis. We have \(a^2 = 16, b^2 = 9\) (a corresponds to the larger denominator).

\( a = 4, b = 3, c = \sqrt{16-9} = \sqrt{7} \)

Foci are along Y-axis: \((0, \pm\sqrt{7})\)

\[ \text{Foci} = \boxed{(0, \pm\sqrt{7})}, \quad e = \frac{\sqrt{7}}{4} = \boxed{0.661} \] \[ \text{LR} = \frac{2b^2}{a} = \frac{2(9)}{4} = \boxed{\frac{9}{2} = 4.5 \text{ units}} \]
Example 2.45 — Find Equation of Ellipse

Find the equation of the ellipse whose foci are \((\pm 3, 0)\) and semi-minor axis is 4.

Solution

Foci on X-axis → major axis along X-axis.

\(c = 3, b = 4\)

\(a^2 = b^2 + c^2 = 16 + 9 = 25\), so \(a = 5\)

\[ \frac{x^2}{25} + \frac{y^2}{16} = 1 \Rightarrow \boxed{\frac{x^2}{25} + \frac{y^2}{16} = 1} \]
Example 2.46 — Find Equation from Eccentricity

Find the equation of the ellipse with major axis along X-axis, semi-major axis = 6, and eccentricity = ½.

Solution

\(a = 6, e = \frac{1}{2}\)

\(c = ae = 6 \times \frac{1}{2} = 3\)

\(b^2 = a^2 - c^2 = 36 - 9 = 27\)

\[ \boxed{\frac{x^2}{36} + \frac{y^2}{27} = 1} \]
Example 2.47 — Does a Point Lie on the Ellipse?

Check whether the point (3, 2) lies on, inside, or outside the ellipse \(\frac{x^2}{25} + \frac{y^2}{9} = 1\).

Solution

Substitute (3, 2) in the LHS:

\( \frac{3^2}{25} + \frac{2^2}{9} = \frac{9}{25} + \frac{4}{9} = \frac{81 + 100}{225} = \frac{181}{225} \approx 0.804 \)

Since 0.804 < 1, the point lies inside the ellipse.

(If = 1 → on the ellipse; if > 1 → outside; if < 1 → inside)

Common Mistakes in Conic Problems
  1. Confusing a and b: 'a' is always the semi-major axis (larger). If the larger denominator is under \(y^2\), the major axis is vertical.
  2. Wrong formula for c: For ellipse: \(c^2 = a^2 - b^2\) (NOT \(a^2 + b^2\)!). For hyperbola: \(c^2 = a^2 + b^2\) (but hyperbola is NOT in syllabus).
  3. Forgetting to identify axis direction: If foci are on Y-axis, the equation has the larger denominator under \(y^2\).
  4. Confusing LR formulas: Parabola LR = 4a. Ellipse LR = 2b²/a. These are different!

Important Definitions

1. Cartesian Coordinate System

A system in which the position of a point in a 2D plane is specified by an ordered pair (x, y) where x is the horizontal distance and y is the vertical distance from the origin, measured along two perpendicular axes.

2. Polar Coordinate System

A system in which the position of a point is specified by an ordered pair (r, θ) where r is the distance from the pole (origin) and θ is the angle measured anti-clockwise from the initial line (positive X-axis).

3. Collinear Points

Three or more points are said to be collinear if they all lie on the same straight line. For three points, the condition is that the area of the triangle formed by them equals zero.

4. Slope (Gradient)

The slope of a straight line is the tangent of the angle (θ) that the line makes with the positive direction of the X-axis. It is given by \(m = \tan\theta\) or \(m = (y_2-y_1)/(x_2-x_1)\).

5. X-intercept and Y-intercept

The x-intercept (a) is the x-coordinate of the point where a line crosses the X-axis (where y = 0). The y-intercept (b or c) is the y-coordinate where the line crosses the Y-axis (where x = 0).

6. Perpendicular Distance

The perpendicular distance from a point to a line is the shortest distance from the point to the line, measured along a line perpendicular to the given line. Formula: \(d = |ax_1+by_1+c|/\sqrt{a^2+b^2}\).

7. Circle

A circle is the locus of a point that moves in a plane such that its distance from a fixed point (centre) remains constant. This constant distance is called the radius.

8. Conic Section

A conic section is a curve obtained when a right circular cone is cut by a plane. It is also defined as the locus of a point whose distance from a fixed point (focus) bears a constant ratio (eccentricity) to its distance from a fixed straight line (directrix).

9. Eccentricity

The eccentricity (e) of a conic is the constant ratio of the distance of any point on the conic from the focus to its perpendicular distance from the directrix. e = 0 (circle), 0 < e < 1 (ellipse), e = 1 (parabola), e > 1 (hyperbola).

10. Parabola

A parabola is a conic section with eccentricity e = 1. It is the locus of a point equidistant from a fixed point (focus) and a fixed line (directrix). Standard form: \(y^2 = 4ax\).

11. Ellipse

An ellipse is a conic section with eccentricity 0 < e < 1. It is the locus of a point such that the sum of its distances from two fixed points (foci) is constant. Standard form: \(x^2/a^2 + y^2/b^2 = 1\).

12. Latus Rectum

The latus rectum of a conic is the chord through the focus perpendicular to the axis of the conic. Length: For parabola \(y^2=4ax\), LR = 4a. For ellipse, LR = \(2b^2/a\).

13. Vertex

The vertex of a conic is the point where the conic intersects its axis of symmetry. A parabola has one vertex, an ellipse has two vertices. For parabola \(y^2 = 4ax\), vertex = (0, 0).

📝 Short Answer Questions (2 Marks)

SA 1. State the distance formula. Find the distance between (−2, 3) and (4, −5).

\(d = \sqrt{(x_2-x_1)^2+(y_2-y_1)^2}\)

\(d = \sqrt{(4-(-2))^2+(-5-3)^2} = \sqrt{36+64} = \sqrt{100} = 10\) units.

SA 2. Write the four standard forms of straight line equation.

  1. Slope-Intercept: \(y = mx + c\)
  2. Point-Slope: \(y - y_1 = m(x - x_1)\)
  3. Two-Point: \(\frac{y-y_1}{y_2-y_1} = \frac{x-x_1}{x_2-x_1}\)
  4. Intercept: \(\frac{x}{a} + \frac{y}{b} = 1\)

SA 3. Write the conditions for two lines to be (i) parallel (ii) perpendicular.

(i) Two lines are parallel if \(m_1 = m_2\) (slopes are equal).

(ii) Two lines are perpendicular if \(m_1 \cdot m_2 = -1\) (product of slopes = −1).

SA 4. What is eccentricity? Identify the conic for e = 0, e = 1, e = ½, e = 3.

Eccentricity (e) is the constant ratio of the distance of a point on the conic from the focus to its distance from the directrix.

e = 0 → Circle, e = 1 → Parabola, e = ½ (0 < e < 1) → Ellipse, e = 3 (e > 1) → Hyperbola.

SA 5. Write the general equation of a circle and the formulas for its centre and radius.

General equation: \(x^2 + y^2 + 2gx + 2fy + c = 0\)

Centre = \((-g, -f)\), Radius = \(\sqrt{g^2 + f^2 - c}\)

SA 6. Convert the polar coordinates (6, 30°) to Cartesian.

\(x = 6\cos 30° = 6 \times \frac{\sqrt{3}}{2} = 3\sqrt{3}\)

\(y = 6\sin 30° = 6 \times \frac{1}{2} = 3\)

Cartesian = \((3\sqrt{3}, 3)\)

SA 7. Define latus rectum. State its length for the parabola \(y^2 = 4ax\).

The latus rectum is the chord through the focus of a conic, perpendicular to the axis.

For parabola \(y^2 = 4ax\), length of latus rectum = \(4a\).

SA 8. Find the slope and y-intercept of the line \(3x - 2y + 8 = 0\).

\(2y = 3x + 8 \Rightarrow y = \frac{3}{2}x + 4\)

Slope = \(\frac{3}{2}\), y-intercept = 4.

SA 9. Write the section formula for internal division. What is the midpoint formula?

Internal division in ratio m:n: \(P = \left(\frac{mx_2+nx_1}{m+n}, \frac{my_2+ny_1}{m+n}\right)\)

Midpoint (when m = n = 1): \(P = \left(\frac{x_1+x_2}{2}, \frac{y_1+y_2}{2}\right)\)

SA 10. What is the relation between \(a\), \(b\), and \(c\) in an ellipse?

In an ellipse \(\frac{x^2}{a^2}+\frac{y^2}{b^2}=1\) (where \(a > b\)):

\(c^2 = a^2 - b^2\), where c is the distance from centre to each focus.

Also, eccentricity \(e = c/a\).

Quick Revision — Unit 2 Summary

📌 Coordinate System
  • Cartesian: (x, y) — horizontal, vertical distances
  • Polar: (r, θ) — distance, angle from initial line
  • Conversion: \(x = r\cos\theta\), \(y = r\sin\theta\), \(r = \sqrt{x^2+y^2}\), \(\theta = \tan^{-1}(y/x)\)
  • Check the quadrant when finding θ!
📌 Distance & Division
  • Distance = \(\sqrt{(x_2-x_1)^2+(y_2-y_1)^2}\)
  • Internal m:n → \(\frac{mx_2+nx_1}{m+n}\) (plus, plus)
  • External m:n → \(\frac{mx_2-nx_1}{m-n}\) (minus, minus)
  • Midpoint = average of coordinates
  • Area = \(\frac{1}{2}|x_1(y_2-y_3)+x_2(y_3-y_1)+x_3(y_1-y_2)|\)
  • Collinear → Area = 0
📌 Straight Line
  • Slope \(m = (y_2-y_1)/(x_2-x_1) = \tan\theta\)
  • 4 forms: Slope-intercept, Point-slope, Two-point, Intercept
  • Parallel → \(m_1 = m_2\)
  • Perpendicular → \(m_1 \cdot m_2 = -1\)
  • Point to line: \(|ax_1+by_1+c|/\sqrt{a^2+b^2}\)
  • Between parallel lines: \(|c_1-c_2|/\sqrt{a^2+b^2}\)
📌 Circle
  • Centre-radius: \((x-h)^2+(y-k)^2=r^2\)
  • Diameter form: \((x-x_1)(x-x_2)+(y-y_1)(y-y_2)=0\)
  • General: \(x^2+y^2+2gx+2fy+c=0\)
  • Centre = \((-g, -f)\), Radius = \(\sqrt{g^2+f^2-c}\)
  • Must have equal coefficients of \(x^2\) and \(y^2\), no \(xy\) term
📌 Conics — Parabola & Ellipse
  • Parabola: e = 1, open curve, 1 focus, 1 directrix
  • \(y^2=4ax\): Focus (a,0), Dir: x=−a, LR = 4a
  • Check if \(y^2\) or \(x^2\) → determines axis direction
  • Positive → opens right/up, Negative → opens left/down
  • Ellipse: 0 < e < 1, closed curve, 2 foci
  • \(\frac{x^2}{a^2}+\frac{y^2}{b^2}=1\); larger denominator = major axis
  • \(c^2=a^2-b^2\), \(e=c/a\), LR = \(2b^2/a\)
